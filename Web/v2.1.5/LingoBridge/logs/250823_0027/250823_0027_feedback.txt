class hw4_2 {
    public static void main(String args[]) {
        // 印出 1~9 的奇數行，每行顯示 2~8 的偶數乘法
        for (int i = 1; i <= 9; i += 2) {
            for (int j = 2; j <= 8; j += 2) {
                System.out.print(i + "*" + j + "=" + i * j);
                if (j != 8) {
                    System.out.print("\t"); // 使用 tab 分隔，但最後一行不加
                }
            }
            System.out.println(); // 換行
        }
    }
}

### **5. 改進建議與反饋**
#### **給學生的建議**
1. **加入註解**：
   - 即使程式簡短，註解能幫助他人（或未來的自己）快速理解邏輯。
   - 例如：
     ```java
     // 印出奇數行 (1,3,5,7,9)
     for (int i = 1; i <= 9; i += 2) { ... }
     ```

2. **格式一致性**：
   - 在方法內部使用空行分隔不同的邏輯區塊（如兩個 `for` 迴圈）。
   - 參考 [Google Java Style Guide](https://google.github.io/styleguide/javaguide.html) 的縮排規範。

3. **避免硬編碼**：
   - 可定義常數（如 `final int MAX_ODD = 9;`）取代魔術數字（`9` 和 `8`），提升可維護性。
   - 例如：
     ```java
     final int MAX_ODD = 9;
     final int MAX_EVEN = 8;
     for (int i = 1; i <= MAX_ODD; i += 2) { ... }
     ```

4. **輸出格式的靈活性**：
   - 題目要求使用 `\t`，但實務上可考慮 `String.format()` 控制對齊：
     ```java
     System.out.printf("%d*%d=%-4d", i, j, i*j); // 左對齊，寬度4
     ```

5. **擴充性思考**：
