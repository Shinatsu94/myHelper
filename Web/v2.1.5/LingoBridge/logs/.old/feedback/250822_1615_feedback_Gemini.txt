**3. 改進的程式碼**

```java
class hw4_2 {
    public static void main(String args[]) {
        // 外層迴圈控制奇數 (1, 3, 5, 7, 9)
        for (int i = 1; i <= 9; i += 2) {
            // 內層迴圈控制偶數 (2, 4, 6, 8)
            for (int j = 2; j <= 8; j += 2) {
                System.out.print(i + "*" + j + "=" + i * j);
                // 判斷是否為該行的最後一個元素，如果是，則不輸出 tab
                if (j != 8) {
                    System.out.print("\t");
                }
            }
            // 每完成一行的輸出後，換行
            System.out.println();
        }
    }
}
```
