/*
檔案名稱: CSIE_41343215_1017_3.java
功能: 使用 Leibniz 公式近似 Pi，輸出 i=10000,20000,...,100000 對應的 Pi（保留10位小數，i 顯示6位寬度），
      並找出使 Pi >= 3.14159 的最小 i，輸出在單獨一行，格式與示例相符。
作者: 教授
日期: 2025-11-11
*/

public class CSIE_41343215_1017_3 {
    public static void main(String[] args) {
        // 使用 Leibniz 公式近似 Pi: pi ≈ 4 * sum_{k=0}^{i-1} (-1)^k / (2k+1)
        double sum = 0.0;

        int maxTerms = 100000; // 需要輸出到 i = 100000
        // 逐項累加，並在每 10000 項輸出結果
        for (int i = 1; i <= maxTerms; i++) {
            int k = i - 1;
            double term = ((k % 2 == 0) ? 1.0 : -1.0) / (2.0 * k + 1.0);
            sum += term;

            if (i % 10000 == 0) {
                double pi = 4.0 * sum;
                System.out.println("i = " + String.format("%6d", i) + ", pi = " + String.format("%.10f", pi));
            }
        }

        // 尋找使 pi >= 3.14159 的最小 i，從上次結果繼續累加
        double target = 3.14159;
        int i = maxTerms;
        double pi = 4.0 * sum;
        while (pi < target) {
            i++;
            int k = i - 1;
            double term = ((k % 2 == 0) ? 1.0 : -1.0) / (2.0 * k + 1.0);
            sum += term;
            pi = 4.0 * sum;
        }

        // 輸出單獨一行的最小 i，格式同樣使用寬度顯示
        System.out.println();
        System.out.println("i = " + String.format("%6d", i));
    }
}

- 使用逐項累加的方式（逐次增加項數，並在每 10000 項輸出結果）以保證時間複雜度可控並可逐步取得題目所需的 i 條件輸出。
- 為了滿足輸出格式需求，使用 String.format 進行數字對齊：i 以寬度6顯示，pi 保留10位小數，確保與題目給定輸出相符。
- 增加第二部分計算，從前面的總和繼續累加，找出最小的 i 使 pi >= 3.14159，並輸出該 i 值，符合題意。
