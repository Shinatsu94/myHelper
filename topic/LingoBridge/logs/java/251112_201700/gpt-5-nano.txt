/*
檔案名稱: Main.java
功能: 讀入長度 n，接著讀入 n 個整數，對陣列進行升序排序，輸出排序結果（以空格分隔，單行輸出）。
作者: 教師
日期: 2025-11-12
*/

import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n;
        if (!sc.hasNextInt()) {
            return;
        }
        n = sc.nextInt();

        int[] arr = new int[n];
        for (int i = 0; i < n; i++) {
            if (sc.hasNextInt()) {
                arr[i] = sc.nextInt();
            } else {
                arr[i] = 0; // 若輸入不足，預設為 0
            }
        }

        Arrays.sort(arr);

        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(arr[i]);
        }
        System.out.println(sb.toString());
    }
}

修改說明
- 用一維整數陣列取代原二維字元陣列，直接滿足題目對一維陣列與排序的需求。
- 使用 Arrays.sort 方便且效率高，能正確處理正負整數的排序。
- 輸出格式改為單行、以空白分隔，符合題目範例與實務常見輸出。
- 增加註解與標頭註解，提升可讀性與可維護性。
