```java
/* 
檔案名稱: CSIE_41343119_1031_1.java
功能: 讀取長度與元素，將陣列內容升序排序，並輸出排序後的結果
作者: 金宗翰
日期: 2025-10-31
說明: 使用 Arrays.sort 進行排序，輸出時避免尾端空格，並對 n=0 做防護
*/

import java.util.Scanner;
import java.util.Arrays;

public class CSIE_41343119_1031_1 {
    public static void main(String[] args) {
        Scanner scn = new Scanner(System.in);

        int n = 0;
        if (scn.hasNextInt()) {
            n = scn.nextInt();
        }

        int[] A = new int[Math.max(0, n)];

        for (int i = 0; i < n && scn.hasNextInt(); i++) {
            A[i] = scn.nextInt();
        }

        // 若陣列長度 > 0，進行排序
        if (A.length > 0) {
            Arrays.sort(A);
        }

        // 輸出，避免尾端空格
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < A.length; i++) {
            if (i > 0) sb.append(' ');
            sb.append(A[i]);
        }
        System.out.println(sb.toString());

        scn.close();
    }
}
```

修改部分說明
- 導入 Arrays.sort 以提升排序穩定性與效率，並使程式碼更清晰易懂。
- 加入對 n=0 的防護，避免在取 A[n-1] 時產生異常。
- 使用 StringBuilder 組裝輸出，避免尾端多餘的空格，輸出格式更加整齊。
- 在檔案開頭加入標頭註解，符合作業規範，方便閱讀與追蹤。
