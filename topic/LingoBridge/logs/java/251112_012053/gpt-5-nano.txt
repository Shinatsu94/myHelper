修改後完整程式碼（符合題意：讀入兩個整數，交換後印出交換結果，且不使用額外變數進行交換；輸出為交換後的兩個值，分別輸出於新的一行）：

/*
 檔案名稱: SwapTwoNumbers.java
 功能: 讀入兩個整數 A, B，交換兩數的值，並印出交換後的結果。
 作者: 教授
 日期: 2025-11-11
 說明: 不使用額外變數進行交換，採用位元異或交換法。
*/
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int A = scanner.nextInt();
        int B = scanner.nextInt();

        // 不使用額外變數，利用 XOR 交換法
        A = A ^ B;
        B = A ^ B;
        A = A ^ B;

        System.out.println(A);
        System.out.println(B);

        scanner.close();
    }
}

修改原因與重點說明：
- 針對題目需求修正：原學生作答讀入三個數字並輸出三行，且未按題意直接輸出兩個變數交換後的結果。新版本僅讀入兩個數字，並輸出交換後的兩個值，符合題目給定的輸入輸出範例。
- 不使用額外變數進行交換：按照題目限制，避免使用第三個變數作為暫存，改以位元異或（XOR）交換法實作，確保不新增變數。
- 程式風格與清晰度：加入檔案頭註解，清楚描述檔案名稱、功能、作者與日期；移除不必要的匯入與變數，讓程式更整潔。
- 輸出順序與格式：直接輸出交換後的兩個值，分別併成兩行，符合範例輸出。
