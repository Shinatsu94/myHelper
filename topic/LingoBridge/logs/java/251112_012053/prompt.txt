【題目】
讀入使用者輸入的二個數字存在變數 A, B 並印出其值，將這二個變數的值交換並印出。
注意：程式除了建立 Scanner  物件讀取使用者輸入之外，不可使用其他變數進行數字交換。

【INPUT-1】
3
5

【OUTPUT-1】

3

【INPUT-2】
234

【OUTPUT-2】

234

            min_a = num2;
        }
        if (num3 < min_a) {
            min_a = num3;
        }
        
        // (a) 找出中間值 (不使用 if-else)
        int sum_a = num1 + num2 + num3;

        int min_b = (num1 < num2) ? ((num1 < num3) ? num1 : num3) : ((num2 < num3) ? num2 : num3);
