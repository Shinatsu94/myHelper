/*
 * File: PiLeibniz.java
 * 功能: 使用萊布尼茲公式計算 pi，輸出 i=10000 到 100000，步進 10000 時的 pi 值。
 * 作者: 教學用範例
 * 日期: 2025-11-11
 * 說明: 輸出格式設定為 i=6 位寬度對齊、pi 為小數點後 10 位。
 */

public class PiLeibniz {
    public static void main(String[] args) {
        for (int i = 10000; i <= 100000; i += 10000) {
            double sum = 0.0;
            for (int k = 1; k <= i; k++) {
                double term = 1.0 / (2.0 * k - 1.0);
                if (k % 2 == 0) term = -term;
                sum += term;
            }
            double pi = 4.0 * sum;
            System.out.printf("i = %6d, pi = %.10f%n", i, pi);
        }
    }
}

修改說明要點
- 取代原程式的無關邏輯，改為計算萊布尼茲公式，確保正確性。
- 使用兩層迴圈：外層控制 i 的值，內層計算到第 i 項的和。變數命名改為更具語意的 sum、pi、i、k。
- 輸出格式使用 System.out.printf，確保 i 的寬度為 6，pi 保留小數點後 10 位，符合題目輸出範本。
- 移除不必要的輸入 (Scanner)，讓程式自動輸出固定序列，符合題目的輸出要求。
