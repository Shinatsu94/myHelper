```java
/**
 * File: CSIE_41343109_0926_3.java
 * 功能: 印出「虎科大」與學生姓名林武杰的每個字及對應的內碼（Unicode 內碼）。
 * 作者: 吳育丞

 *
 * 說明:
 * 不依賴輸入，直接輸出六行格式：字 空格 內碼。
 * 內碼使用 Unicode 編碼點（char 轉成 int）。
 */
public class CSIE_41343109_0926_3 {
    public static void main(String[] args) {
        String[] chars = {"虎", "科", "大", "林", "武", "杰"};
        for (String s : chars) {
            char ch = s.charAt(0);
            int code = (int) ch; // 使用 Unicode 內碼作為「對應的內碼」

        }
    }
}
```
