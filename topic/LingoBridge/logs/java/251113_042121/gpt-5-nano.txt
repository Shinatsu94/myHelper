修改後完整程式碼（CharacterCodePrinter.java）
/*
檔案名稱: CharacterCodePrinter.java
功能: 印出「虎科大」與學生姓名每個字的 Unicode 內碼
作者: [你的姓名]
日期: 2025-11-12

說明:
- 先輸出 "虎科大" 三個字及其對應的內碼
- 再輸出學生中文姓名的每個字及其對應的內碼
- 每行輸出：字元 + 空格 + 對應的內碼（十進位）
*/
public class CharacterCodePrinter {
    public static void main(String[] args) {
        // 先印虎科大
        printCharsWithCode("虎科大");
        // 再印學生中文姓名（此處以林武杰為例，若姓名不同可修改字串）
        printCharsWithCode("林武杰");
    }

    // 將字串中的每一個字元，輸出「字元 空格 內碼」
    private static void printCharsWithCode(String s) {
        for (int i = 0; i < s.length(); i++) {
            char ch = s.charAt(i);
            System.out.println(ch + " " + (int) ch);
        }
    }
}
