修改後完整程式碼
/*
檔案名稱: SortArray.java
功能說明: 讀取任意長度整數陣列，將其由小到大排序，輸出排序結果（以空格分隔，末尾換行）。
作者: 教師
日期: 2025-11-12
*/
import java.util.Arrays;
import java.util.Scanner;

public class SortArray {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();      // 讀取陣列長度
        int[] arr = new int[n];      // 建立陣列
        for (int i = 0; i < n; i++) {
            arr[i] = sc.nextInt();   // 讀取每個整數
        }

        Arrays.sort(arr);            // 由小到大排序

        // 輸出排序結果，元素之間以單一空格分隔，結束後換行
        for (int i = 0; i < n; i++) {
            if (i > 0) System.out.print(" ");
            System.out.print(arr[i]);
        }
        System.out.println();

        sc.close();
    }
}

修改說明
- 將排序邏輯改為使用 Java 內建的 Arrays.sort，能正確處理任意長度與包含負數的情況，確保結果正確且穩定。
- 輸出格式調整為僅輸出排序後的元素，以空格分隔，符合題意，避免多餘的符號（如 -1）。
- 加入程式標頭註解，說明檔案名稱、功能、作者與日期，並在程式內加入適當註解，提高可讀性與可維護性。
- 使用更具可讀性的變數名稱（arr、n），並關閉資源（sc.close()）。
