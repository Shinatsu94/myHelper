```java
/*
File: SortArray.java
功能: 讀入整數 n，接著讀入 n 個整數，將它們由小到大排序後輸出
作者: 資訊工程系教授
日期: 2025-11-12
說明: 使用動態長度陣列與 Arrays.sort 進行排序，輸出以空格分隔。
*/

import java.util.Arrays;
import java.util.Scanner;

public class SortArray {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 讀入長度 n
        if (!sc.hasNextInt()) {
            System.err.println("輸入格式錯誤：未提供長度 n。");
            return;
        }
        int n = sc.nextInt();
        if (n < 0) {
            System.err.println("長度 n 不可為負數。");
            return;
        }

        // 讀入 n 個整數
        int[] arr = new int[n];
        for (int i = 0; i < n; i++) {
            if (!sc.hasNextInt()) {
                System.err.println("輸入不足，缺少第 " + (i + 1) + " 個整數。");
                return;
            }
            arr[i] = sc.nextInt();
        }

        // 排序
        Arrays.sort(arr);

        // 輸出結果，元素以空格分隔
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(arr[i]);
        }
        System.out.println(sb.toString());
    }
}
```

- 使用 Arrays.sort 進行升序排序：
  - 簡潔且高效，時間複雜度 O(n log n)。
