修改後完整程式碼
/*
 * File: CSIE_41343252_1003_3.java
 * 功能: 讀入兩個整數 A, B，先印出 A, B 的值，接著交換兩數值，再印出交換後的值。
 *       交換過程不使用額外變數。
 * 作者: 教授
 * 日期: 2025-11-11
 */
import java.util.Scanner;

public class CSIE_41343252_1003_3 {
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);

        // 讀取兩個整數
        int a = in.nextInt();
        int b = in.nextInt();

        // 先印出尚未交換前的值
        System.out.println(a);
        System.out.println(b);

        // 不使用額外變數，使用 XOR 交換
        a = a ^ b;
        b = a ^ b;
        a = a ^ b;

        // 印出交換後的值
        System.out.println(a);
        System.out.println(b);

        in.close();
    }
}

修改原因說明
- 以符合題目要求的輸入與輸出順序：先印出兩數的原始值，之後再印出交換後的值。
- 為避免使用額外變數，採用位元運算的 XOR 交換法，確保不新增變數，符合「不可使用其他變數進行數字交換」的限制。
- 透過適當的程式頭與註解提升可讀性與維護性，並在程式結尾釋放資源。
