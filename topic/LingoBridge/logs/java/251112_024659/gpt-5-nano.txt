1) 使用 XOR 交換以避免整數溢位風險
- 原本使用 A = A + B; B = A - B; A = A - B 的交換方式可能在極端數值情況下造成整數溢位。改用 XOR 交換，只用 A、B 兩個變數即可完成交換，且不會產生溢位風險。
