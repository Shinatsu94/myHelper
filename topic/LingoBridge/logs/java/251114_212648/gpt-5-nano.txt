修改後完整程式碼

import java.util.Scanner;
import java.util.Arrays;

/*
檔案名稱: CSIE_41343128_1031_2.java
功能說明: 讀取使用者輸入的整數個數 n，接著讀入 n 個整數，將其升序排序，最後輸出排序結果。
作者: 你的姓名
日期: 2025-11-14
*/

public class CSIE_41343128_1031_2 {
    public static void main(String[] args) {
        Scanner scn = new Scanner(System.in);

        int n;
        if (!scn.hasNextInt()) {
            scn.close();
            return;
        }
        n = scn.nextInt();

        int[] list = new int[n];
        for (int i = 0; i < n; i++) {
            list[i] = scn.nextInt();
        }

        // 使用內建排序，確保正確性與效率
        Arrays.sort(list);

        // 輸出，元素之間以空格分隔，無多餘空格
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(list[i]);
        }
        System.out.print(sb.toString());

        scn.close();
    }
}

說明修改部分原因
- 使用 Arrays.sort 能保證正確且高效的排序，避免手寫排序邏輯帶來的錯誤與覆蓋資料的風險。
- 輸出格式統一，避免尾隨空格，符合題目對輸出格式的要求。
- 加入標頭註解，提升可讀性與維護性，同時確保註解包含檔案名稱、功能、作者與日期等資訊，滿足備註要求。
