改進建議：建議加入程式標頭註解、移除未使用變數並採用更高效的單次掃描演算法，同時用字更具可讀性。

優化說明與修改內容：
- 時間複雜度由原本的 O(n^2) 降為 O(n)。新演算法從右向左掃描，維護右側已知的最大值 maxSoFar，並把當前元素替換為該值，最後一個元素設為 -1。
- 移除未使用的變數（如 big、kal），提升可讀性與維護性。
- 加入程式標頭註解，描述檔案名稱、功能、作者、日期。
- 使用清楚的變數名稱與 StringBuilder 進行輸出，提升格式整潔度。
- 增加對 N=0 的情況的安全處理。

修改後完整程式碼如下：
/*
檔案名稱: CSIE_41343220_1031_2.java
功能: 依輸入長度 n 讀入 n 個整數，將陣列每個元素替換為其右邊元素的最大值（最後一個為 -1），最後印出結果。
作者: 教師
日期: 2025-11-14
說明: 遵循題目要求，不使用 EOF，且對 0 長度情況也能穩定執行。
*/
import java.util.Scanner;

public class CSIE_41343220_1031_2 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int n = 0;
        if (sc.hasNextInt()) {
            n = sc.nextInt();
        }

        int[] data = new int[n];

        for (int i = 0; i < n; i++) {
            if (sc.hasNextInt()) {
                data[i] = sc.nextInt();
            } else {
                data[i] = 0; // 若輸入不足，預設為 0，避免例外
            }
        }

        // 若長度為正，使用 O(n) 的右向最大值演算法
        if (n > 0) {
            int lastValue = data[n - 1];
            int maxSoFar = lastValue;
            data[n - 1] = -1;

            for (int i = n - 2; i >= 0; i--) {
                int current = data[i];
                data[i] = maxSoFar;
                if (current > maxSoFar) {
                    maxSoFar = current;
                }
            }
        }

        // 輸出結果
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(data[i]);
        }
        System.out.println(sb.toString());

        sc.close();
    }
}
