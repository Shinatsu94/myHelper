/*
 * 檔案名稱: SortArrayAscending.java
 * 功能: 建立任意長度的一維陣列，讀入資料並由小到大排序，最後輸出排序結果
 * 作者: 教師
 * 日期: 2025-11-12
 */
import java.util.Scanner;

public class SortArrayAscending {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        if (!scanner.hasNextInt()) {
            return;
        }
        int n = scanner.nextInt();
        int[] a = new int[n];
        for (int i = 0; i < n; i++) {
            if (scanner.hasNextInt()) {
                a[i] = scanner.nextInt();
            } else {
                a[i] = 0; // 若輸入不足，預設為0
            }
        }

        // 選取排序法（Selection Sort），逐步將最小元素放到前端
        for (int i = 0; i < a.length - 1; i++) {
            int minIndex = i;
            for (int j = i + 1; j < a.length; j++) {
                if (a[j] < a[minIndex]) {
                    minIndex = j;
                }
            }
            // 交換
            int tmp = a[i];
            a[i] = a[minIndex];
            a[minIndex] = tmp;
        }

        // 輸出排序後的陣列，元素以單一空格分隔，尾端不多輸入空格
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < a.length; i++) {
            if (i > 0) sb.append(' ');
            sb.append(a[i]);
        }
        System.out.println(sb.toString());
        scanner.close();
    }
}

修改說明：
- 正確性與邏輯：原作的排序邏輯有缺陷，會導致輸出並非由小到大，且最後輸出會出現 -1。改用選擇排序（Selection Sort）法，對未排序部分逐步找出最小值並交換，確保整體陣列從小到大排列。
- 輸出處理：移除原程式中最後設為 -1 的處理，直接輸出整個排序後的陣列，且元素之間用單一空格分隔，避免多餘符號。
- 格式與可讀性：加入清晰的檔案註解與標頭，並使用有意義的類別名稱與註解對排序過程做說明，提升維護性與可讀性。
- 邏輯與穩定性：雖然選擇排序非最有效率，但在規模較小或任意長度陣列時可靠且易於理解，符合教學與題意需求。

改進建議（一句話）：
建議使用穩定且簡潔的排序方式（如 Arrays.sort 或清楚的選擇排序實作），並確保輸出格式嚴謹以正確顯示排序結果。
