修改後完整程式碼：
/* 檔案名稱: SwapTwoNumbers.java
   日期: 2025/11/11
   作者: 你的助理
   程式介紹: 讀入兩個整數 A, B，交換兩數並印出交換後的值（不使用額外變數）。
*/
import java.util.Scanner;

public class SwapTwoNumbers {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int A = sc.nextInt();
        int B = sc.nextInt();

        // 不使用其他變數進行交換：A = A + B; B = A - B; A = A - B
        A = A + B;
        B = A - B;
        A = A - B;

        // 印出交換後的兩個值
        System.out.println(A);
        System.out.println(B);

        sc.close();
    }
}

修改說明：
- 移除了不相關的三數排序邏輯，直接按照題目需求完成兩數交換與輸出，且交換過程嚴格使用 A、B 兩個變數，不新增其他變數，符合題目限制。
- 改用清晰的變數命名與標準輸出格式，避免輸出尾端不換行或格式不一致的問題。
- 加入檔案頭註解與關閉 Scanner，提升可維護性與資源管理。

一句話改進建議：
- 若未來要同時保留原先的兩步輸出（交換前與交換後），請在輸出前先備註與說明，或使用可控的輸出格式與換行，避免與測試樣例混淆。
