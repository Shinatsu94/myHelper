題目：
a number is called a well-ordered number if all of its digits are strictly incressing from left to right.
For example, 258 (2 < 5 < 8) is a well-ordered number.
Write a program to determine a user input number whether it is a well-ordered number.

題目要求:
- 程式僅執行一次就結束，無需使用EOF

【INPUT-1】
258

【OUTPUT-1】
258 is well-ordered number.

【INPUT-2】
255

【OUTPUT-2】
255 is not well-ordered number.

【INPUT-3】
285

【OUTPUT-3】
285 is not well-ordered number.

學生作答程式：
import java.util.Scanner;

public class CSIE_41343131_1017_2 {
    public static void main(String args[]) {
        Scanner 輸入 = new Scanner(System.in);
        while(輸入.hasNext()) {
            int t = 輸入.nextInt();
            String str = String.format("%d is %swell-ordered number.", t, (isWellNumber((t)) ? "":"not "));
            System.out.println(str);
        }
        輸入.close();
    };

    public static boolean isWellNumber(int n) {
        if (n < 0) n *= -1;
        int min = n + 1;
        while(n > 0) {
            int t = n % 10;
            if (t >= min) return false;
            n /= 10;
            min = t;
        }
        return true;
    };
}
