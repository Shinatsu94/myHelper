```java
/*
 * 檔案名稱: SwapTwoNumbers.java
 * 功能: 從使用者讀入兩個整數 A, B，先印出原始值，然後在不新增第三個變數的情況下交換兩數，最後再印出交換後的值。
 * 作者: 批改系統
 * 日期: 2025-11-11
 */
import java.util.Scanner;

public class SwapTwoNumbers {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        // 讀取兩個整數
        long A = scanner.nextLong();
        long B = scanner.nextLong();

        // 輸出原始值
        System.out.println(A);
        System.out.println(B);

        // 不使用第三個變數，利用加法/減法進行交換
        A = A + B;
        B = A - B;
        A = A - B;

        // 輸出交換後的值
        System.out.println(A);
        System.out.println(B);

        scanner.close();
    }
}
```

- 不使用額外變數進行交換：採用 A = A + B; B = A - B; A = A - B 的交換法，符合題目限制（只使用現有的 A、B 變數）。
- 增加標頭註解與說明：提供檔名、功能、作者、日期等註解，提升可維護性與可讀性，符合備註評分標準。
