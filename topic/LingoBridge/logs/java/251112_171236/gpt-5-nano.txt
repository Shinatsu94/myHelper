```java
/*
 * 檔案名稱: CSIE_41343241_1031_1.java
 * 功能: 讀入長度與整數，輸出升冪排序結果
 * 作者: [請填入作者名稱]
 * 日期: 2025-11-12
 * 說明: 使用 Arrays.sort 進行排序，輸出時避免尾端空格。
 */

import java.util.Arrays;
import java.util.Scanner;

public class CSIE_41343241_1031_1 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        if (!sc.hasNextInt()) {
            sc.close();
            return;
        }

        int n = sc.nextInt();
        if (n <= 0) {
            sc.close();
            return;
        }

        int[] arr = new int[n];
        for (int i = 0; i < n; i++) {
            if (sc.hasNextInt()) {
                arr[i] = sc.nextInt();
            } else {
                arr[i] = 0; // 若輸入不足，以 0 代替，避免例外
            }
        }
        sc.close();

        // 排序並輸出，避免尾端多餘空格
        Arrays.sort(arr);
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(arr[i]);
        }
        System.out.print(sb.toString());
    }
}
```

修改原因說明
- 以 Arrays.sort 取代自訂的排序雙層迴圈，減少錯誤風險、提升穩健性與可讀性，並達到 O(n log n) 的時間複雜度。
- 移除不必要的變數（如 sum、max、min）與不直覺的邏輯，讓程式結構更清晰。
- 改用清晰的變數命名（如 n、arr），提升可讀性與維護性。
- 使用 StringBuilder 與無尾隨空白的輸出邏輯，符合題目輸出格式。
- 加入程式標頭註解，包含檔案名稱、功能、日期等，符合作業要求的備註規範。
- 加入基本輸入檢查與資源釋放（關閉 Scanner）。
