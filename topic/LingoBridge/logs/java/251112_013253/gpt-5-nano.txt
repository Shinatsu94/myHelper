檔案名稱: SwapTwoNumbers.java
功能: 讀入兩個整數 A, B，印出其值，之後在不使用額外變數的情況下交換並印出
作者: 你的名字
日期: 2025-11-11

import java.util.Scanner;

public class SwapTwoNumbers {
    public static void main(String[] args) {
        Scanner s = new Scanner(System.in);

        // 讀入兩個整數
        int A = s.nextInt();
        int B = s.nextInt();

        // 先印出原值
        System.out.println(A);
        System.out.println(B);

        // 不使用額外變數，使用 XOR 交換 A 與 B
        A = A ^ B;
        B = B ^ A;
        A = A ^ B;

        // 再印出交換後的值
        System.out.println(A);
        System.out.println(B);
    }
}

- 使用 XOR 交換法避免可能的整數溢位問題，且只使用 A、B 兩個變數，符合題目限制。
