修改後完整程式碼（符合題意的實作）：
/*
File: PiLeibniz.java
Description: 使用 Leibniz 公式計算 pi 值，輸出每 10000 項的近似值，i 為項數，pi 為近似值，格式化輸出。
Author: 教師
Date: 2025-11-11
*/
public class PiLeibniz {
    public static void main(String[] args) {
        int maxTerms = 100000; // 最大項數
        int step = 10000;      // 每隔多少項輸出一次結果
        double sum = 0.0;

        for (int k = 0; k < maxTerms; k++) {
            // 第 k+1 項，符號依序 + - + - ...
            double term = ((k % 2 == 0) ? 1.0 : -1.0) / (2.0 * k + 1.0);
            sum += term;

            if ((k + 1) % step == 0) {
                int i = k + 1;          // 目前的項數
                double pi = 4.0 * sum;   // 當前的 pi 近似值
                System.out.printf("i = %6d, pi = %.10f%n", i, pi);
            }
        }
    }
}

- 直接根據題意實作：以 Leibniz 公式逐項累加，當項數達到 10000 的倍數時輸出當前 i 與 pi 的值，符合題目給定的輸出需求。
- 使用漸增法避免重複計算：以 sum 變數保存前一階的和，迭代新增一項即可產生當前的 pi，效能較好且簡潔。
- 輸出格式化：i 使用六位寬度輸出以達成對齊需求，pi 使用十位小數輸出，符合「i 顯示到6位數、pi 顯示到小數點後10位」的要求；每行以換行結尾，且使用單一空白分隔符號。
