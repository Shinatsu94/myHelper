/*

功能說明: 建立任意長度的一維整數陣列，讀取長度與元素，對內容做升冪排序，輸出排序後的陣列
作者: [姓名]
日期: 2025-11-12
*/
import java.util.Scanner;
import java.util.Arrays;

public class CSIE_51015130_1031_3b {
	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);

		// 讀取陣列長度
		int n = 0;
		if (sc.hasNextInt()) {
			n = sc.nextInt();
		}

		// 對於非正長度直接結束或輸出空行
		if (n <= 0) {
			System.out.println();
			return;
		}

		// 讀取 n 個整數
		int[] arr = new int[n];
		for (int i = 0; i < n; i++) {
			if (sc.hasNextInt()) {
				arr[i] = sc.nextInt();
			} else {
				arr[i] = 0; // 若輸入不足，預設為 0
			}
		}

		// 升冪排序
		Arrays.sort(arr);

		// 輸出排序後的陣列，元素以空白分隔，末尾不多餘空格
		StringBuilder sb = new StringBuilder();
		for (int i = 0; i < n; i++) {
			if (i > 0) sb.append(' ');
			sb.append(arr[i]);
		}
		System.out.println(sb.toString());
	}
}

- 使用直覺且符合需求的變數命名：n 表示長度，arr 為整數陣列，以增進可讀性。
- 使用 Arrays.sort(arr) 進行排序，確保正確性與效率，並能處理負數與任意整數。

- 輸入控制與輸出格式：若長度為非正整數，直接輸出空行並結束；讀取不足時預設為 0，並以空格分隔輸出，符合一般競賽/作業輸出格式。
