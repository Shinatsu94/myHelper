```java
/*
 * 檔案名稱: Main.java
 * 功能說明: 讀入兩個整數 A, B，經過交換後分別印出交換後的值
 * 作者: 教師
 * 日期: 2025-11-11
 */

import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        // 讀取兩個整數 A 與 B
        if (in.hasNextInt()) {
            int A = in.nextInt();
            if (in.hasNextInt()) {
                int B = in.nextInt();
                // 使用 XOR 交換，符合不使用額外變數的限制
                A ^= B;
                B ^= A;
                A ^= B;
                // 輸出交換後的值
                System.out.println(A);
                System.out.println(B);
            }
        }
        in.close();
    }
}
```

修改重點與原因：
- 取代原本的 while 迴圈：題目僅需輸入兩個數字並輸出交換結果，使用兩個變數 A、B 直接讀取即可，避免無限輪詢與多組輸入處理造成的複雜度與風險。
- 修改類別名稱為 Main：提高與常見測試平台的相容性，避免命名衝突或編譯問題。
- 加入程式標頭註解：符合你提出的備註要求，方便他人閱讀與維護。
- 使用 XOR 交換法：符合「不可使用其他變數進行數字交換」的限制，並保有兩個變數即可完成交換。
- 變數命名與格式：改為常見且清晰的變數命名，提升可讀性；縮排與程式結構整理，避免結尾多餘的分號等語法問題。
