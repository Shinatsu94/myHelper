修改後完整程式碼（符合題意：讀入兩個數字並印出原值，再用不新增變數的方式交換，最後印出交換後的值）：
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        int A = scanner.nextInt();
        int B = scanner.nextInt();

        // 印出原始值
        System.out.println(A);
        System.out.println(B);

        // 不使用額外變數，利用算術交換法交換 A 與 B
        A = A + B;
        B = A - B;
        A = A - B;

        // 印出交換後的值
        System.out.println(A);
        System.out.println(B);

        scanner.close();
    }
}

修改原因說明：
- 將作業題意改為兩個整數的輸入與交換，並在輸出中依序顯示原值與交換後的值。
- 使用不新增變數的算術交換法（A = A + B; B = A - B; A = A - B），符合題目限制，且保持變數僅為 A、B 兩個。
- 去除與題意不符的額外邏輯與判斷，使程式更清晰、穩定。

改進建議（一句話）：
直接以兩個變數讀入並用算術交換法完成交換，避免不相關的邏輯，同時需留意整數溢位風險。
