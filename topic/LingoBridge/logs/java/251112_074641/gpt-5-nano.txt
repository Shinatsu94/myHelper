/**
 * File: LeibnizPi.java
 * 功能: 使用萊布尼茲級數近似 pi，輸出 i 為 10000, 20000, ..., 100000 的 pi 值。
 *       pi = 4 * sum_{k=0}^{i-1} (-1)^k / (2k+1)
 * 作者: 你的名字
 * 日期: 2025-11-11
 * 說明: 本程式不需要外部輸入，直接計算並輸出指定的 i 與對應的 pi，pi 顯示 10 位小數、i 顯示至 6 位數。
 */

public class LeibnizPi {
    public static void main(String[] args) {
        int maxI = 100000;
        int step = 10000;

        double sum = 0.0; // 部分和，將依序累加以避免重複計算
        for (int k = 0; k < maxI; k++) {
            // term = (-1)^k / (2k+1)
            double term = ((k % 2 == 0) ? 1.0 : -1.0) / (2.0 * k + 1.0);
            sum += term;

            int i = k + 1; // 當前使用的 i 值，從 1 開始累加
            if (i % step == 0) {
                double pi = 4.0 * sum;
                System.out.printf("i = %6d, pi = %.10f%n", i, pi);
            }
        }
    }
}

修改說明與原因：
- 修改部分 1：採用單次累加的方式計算萊布尼茲級數，避免重複重新計算整個前綴和，提升效率（從 O(n^2) 改為 O(n)）。
  - 原因：題目要求輸出多個 i 值的近似 pi，逐步累加能在達到下一個 i 時立即得到新近似值，效率較高。
- 修改部分 2：輸出格式採用 System.out.printf，明確控制 i 的寬度與 pi 的小數位數，符合題目對格式的要求：
  - i 顯示到 6 位數，使用 %6d。
  - pi 顯示到小數點後 10 位，使用 %.10f。
  - 結果輸出形式與題目給定的樣式一致，且對齊美觀。
- 修改部分 3：增加檔案頭註解，說明檔案名稱、功能、作者、日期與輸出行為，提升可維護性與可理解性。
