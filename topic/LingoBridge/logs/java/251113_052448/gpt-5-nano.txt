修改後完整程式碼（符合題意）
public class TigerAndNameCode {
    public static void main(String[] args) {
        // 先印出虎科大三個字及其對應內碼
        String[] first = {"虎", "科", "大"};
        // 再印學生姓名：林武杰 three characters
        String[] second = {"林", "武", "杰"};

        for (String ch : first) {
            printCode(ch);
        }
        for (String ch : second) {
            printCode(ch);
        }
    }

    // 輔助方法：輸出單個字與其 Unicode 十進位內碼
    private static void printCode(String ch) {
        int code = ch.codePointAt(0);
        System.out.println(ch + " " + code);
    }
}

說明修改部分的原因
- 原作業邏輯與題意嚴重不符，需改為固定輸出特定字與其內碼，才能符合題目的輸出格式與內容要求。
- 使用單一輔助方法 printCode 可避免重複程式碼，提升可維護性與可讀性；並以 codePointAt 取得字的 Unicode 內碼，確保跨字元集時的穩定性。
