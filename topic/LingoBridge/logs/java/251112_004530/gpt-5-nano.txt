import java.util.Scanner;

public class SwapTwoNumbers {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int a = sc.nextInt();
        int b = sc.nextInt();

        // 使用不使用暫存變數的異或交換法，符合題意限制
        a = a ^ b;
        b = a ^ b;
        a = a ^ b;

        System.out.println(a);
        System.out.println(b);
        sc.close();
    }
}

修改說明（重點改動與原因）：
- 針對題目要求，只讀入兩個整數並輸出交換後的結果，因此移除原程式中的多餘變數 c、max、min、mid，以及不相干的三數排序邏輯，避免混淆與錯誤輸出。
- 引入兩個變數 a、b 的交換，使用不需要暫存變數的異或交換法（XOR swap），以滿足「不可使用其他變數進行數字交換」的限制，同時避免算術溢位風險。
- 以兩行輸出交換後的結果，符合題意的輸出格式（先輸出交換後的 a，再輸出交換後的 b）。
