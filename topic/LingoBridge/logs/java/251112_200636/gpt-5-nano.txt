修改後完整程式碼（優化後）
/*
檔案名稱: CSIE_41343219_1031_3.java
功能: 讀取 n 與 n 個整數，排序後輸出
作者: 教授
日期: 2025-11-12
*/
public class CSIE_41343219_1031_3 {
    public static void main(String[] args) {
        java.util.Scanner sc = new java.util.Scanner(System.in);

        int n = sc.nextInt();
        int[] arr = new int[n];
        for (int i = 0; i < n; i++) {
            arr[i] = sc.nextInt();
        }

        java.util.Arrays.sort(arr);

        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(arr[i]);
        }

        System.out.print(sb.toString());
        sc.close();
    }
}

修改部分說明
- 將原本用於顯示三角矩陣星號的輸出替換為實作讀取、排序與輸出整數陣列的功能，符合「任意長度的一維陣列，排序後輸出」的題意。
- 使用 Arrays.sort 以簡潔且正確的方式實作升序排序，並使用 StringBuilder 組裝輸出字串，避免多次 I/O 呼叫。
