 作者: 你的名字

 說明: 讀取長度 n，接著讀入 n 個整數，排序後以空格分隔輸出。
*/
import java.util.*;

public class SortingAscending {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (!sc.hasNextInt()) {
            return;
        }
        int n = sc.nextInt();
        int[] arr = new int[n];
        for (int i = 0; i < n; i++) {
            if (sc.hasNextInt()) {
                arr[i] = sc.nextInt();
            } else {
                arr[i] = 0;
            }
        }
        Arrays.sort(arr);
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(arr[i]);
        }
        System.out.print(sb.toString());
    }
}

- 正確性與邏輯：完全符合題意，讀取先給定的長度 n，接著讀入 n 個整數，使用 Arrays.sort 進行升序排序，最後以空格分隔輸出，能支援負數且不依賴 EOF。
