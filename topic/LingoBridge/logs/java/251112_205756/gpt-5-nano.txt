public class ArraySort_ASC {
    /*
     檔案名稱: ArraySort_ASC.java
     功能: 讀入整數長度 n 與 n 個整數，將其由小到大排序後輸出
     作者: 教師
     日期: 2025-11-12
    */

    public static void main(String[] args) {
        java.util.Scanner in = new java.util.Scanner(System.in);

        // 讀取長度 n
        if (!in.hasNextInt()) {
            return;
        }
        int n = in.nextInt();

        // 讀入 n 個整數
        int[] arr = new int[n];
        for (int i = 0; i < n; i++) {
            if (in.hasNextInt()) {
                arr[i] = in.nextInt();
            } else {
                arr[i] = 0; // 若輸入不足，預設為 0，避免例外
            }
        }

        // 排序
        java.util.Arrays.sort(arr);

        // 輸出，元素以空格分隔
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(arr[i]);
        }
        System.out.println(sb.toString());
    }
}

- 將題目需求轉化為直接的排序問題，降低不必要的矩陣運算與複雜度。
- 使用 Arrays.sort 提升效率與可讀性，且對負數與整數排序皆適用。
