// File: LeibnizPiPrinter.java
/**
 * 檔案名稱: LeibnizPiPrinter.java
 * 功能說明: 計算並輸出使用 Leibniz 公式近似的圓周率(pi)。
 *           當 i = 10000, 20000, ..., 100000 時輸出：
 *           i = <6位右對齊>, pi = <小數點後10位>
 * 作者: 你可自行填寫
 * 日期: 2025-11-12
 */
public class LeibnizPiPrinter {
    public static void main(String[] args) {
        int maxI = 100000;          // 需要計算到的最大 i 值
        double sum = 0.0;             // Leibniz 級數的部分和
        for (int i = 1; i <= maxI; i++) {
            double term = (i % 2 == 1 ? 1.0 : -1.0) / (2.0 * i - 1.0); // (+) 第1,3,5...項，(-) 第2,4,6...項
            sum += term;
            if (i % 10000 == 0) { // 每 10000 筆輸出一次
                double pi = 4.0 * sum;
                System.out.printf("i = %6d, pi = %.10f%n", i, pi);
            }
        }
    }
}

- 漸增式計算與效能：利用累加的方式逐步累計 sum，避免每次都重新計算前面的項目，提升效率。
- 輸出格式化：採用 System.out.printf 配合格式字串，能同時滿足 i 顯示為6位寬度並在左對齊/右對齊，以及 pi 顯示至小數點後10位的需求。使用 "i = %6d, pi = %.10f" 能與題目樣例相符。
