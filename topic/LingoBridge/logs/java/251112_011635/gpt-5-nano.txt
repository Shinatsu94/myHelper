優化後程式碼（符合題意：輸入兩個數字 A、B，先印出，再交換並印出，且不使用第三變數）：
public class SwapTwoNumbers {
    public static void main(String[] args) {
        java.util.Scanner in = new java.util.Scanner(System.in);
        int A = in.nextInt();
        int B = in.nextInt();

        // 先印出原值
        System.out.println(A);
        System.out.println(B);

        // 不使用額外變數，使用「加法與減法」交換 A 與 B
        A = A + B;
        B = A - B;
        A = A - B;

        // 印出交換後的值
        System.out.println(A);
        System.out.println(B);

        in.close();
    }
}

修改原因說明：
- 將程式改為符合題意：先印出原本的 A、B，然後交換再印出，去除原先對三個數的排序邏輯。
- 交換實作採用不使用第三變數的算術交換法，符合「不可使用其他變數進行數字交換」的限制，使得僅用 A 與 B 兩個變數完成交換。
- 保留 Scanner 物件的使用，並在結尾關閉輸入以提升資源管理。
