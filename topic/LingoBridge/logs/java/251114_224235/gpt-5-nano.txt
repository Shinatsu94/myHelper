```java
/*
 * 檔案名稱: Main.java
 * 功能說明: 讀取整數長度 n，接著讀入 n 個整數，將其升冪排序，最後以空格分隔輸出排序後的陣列內容。
 * 作者: 你的名字
 * 日期: 2025-11-14
 * 備註: 盡量避免使用 EOF，依題意輸入結束。
 */

import java.util.Arrays;
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (!sc.hasNextInt()) {
            sc.close();
            return;
        }

        int n = sc.nextInt();
        int[] arr = new int[n];

        for (int i = 0; i < n; i++) {
            if (sc.hasNextInt()) {
                arr[i] = sc.nextInt();
            } else {
                // 若輸入不足，填充為 0（根據需求也可選擇拋出例外或其他處理）
                arr[i] = 0;
            }
        }

        // 排序
        Arrays.sort(arr);

        // 輸出
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(arr[i]);
        }
        System.out.println(sb.toString());

        sc.close();
    }
}
```

修改重點與原因
- 使用一維陣列與 Arrays.sort 的組合，讓實作更符合題意、簡潔且效率高。
- 輸出格式改為單行、用空格分隔，符合題目「1 3 5 8 12」的要求。
- 增加清晰的標頭與註解，提升可讀性與維護性。
- 提供基本的輸入穩健性處理，避免因輸入不完整而造成不可預期的例外。
