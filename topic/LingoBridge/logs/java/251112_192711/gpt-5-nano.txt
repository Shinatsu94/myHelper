修改後的完整程式碼（使用內建排序，確保正確性，並改善輸出與註解）：
import java.util.Scanner;
import java.util.Arrays;

/*
檔案名稱: CSIE_41343150_1031_2.java
功能: 讀取一個整數長度 n 與 n 個整數，輸出其升序排序結果
作者: 教授
日期: 2025-11-12
說明: 使用 Arrays.sort 進行排序，輸出以空格分隔的整數序列，避免使用 EOF。
*/
public class CSIE_41343150_1031_2 {
  public static void main(String[] args) {
    Scanner scn = new Scanner(System.in);
    int n = scn.nextInt();
    int[] a = new int[n];
    for (int i = 0; i < n; i++) {
      a[i] = scn.nextInt();
    }

    // 進行升序排序
    Arrays.sort(a);

    // 輸出，元素之間以單一空格分隔，末尾不多餘空格
    StringBuilder sb = new StringBuilder();
    for (int i = 0; i < n; i++) {
      if (i > 0) sb.append(' ');
      sb.append(a[i]);
    }
    System.out.print(sb.toString());

    scn.close();
  }
}

修改部分說明與原因
- 將原本不正確的「每次取後綴最大值再寫回前綴」的作法改為使用 Arrays.sort(a)，確保排序結果正確且對負數亦適用。
- 移除原程式中的 a[n-1] = -1 的特殊值設定與不必要的輸出處理，避免干擾輸出內容與正確性。
- 增加頭部註解，說明檔案名稱、功能、作者與日期，提升可維護性。
- 使用 StringBuilder 組裝輸出，避免尾端多餘空格，且輸出格式更符合題意。
- 關閉 Scanner，養成良好資源管理習慣。
