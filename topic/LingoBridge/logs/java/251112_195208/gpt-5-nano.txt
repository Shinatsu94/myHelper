/*
 * 檔案名稱: CSIE_41343109_1031_03.java
 * 功能: 讀入使用者輸入的任意長度一維陣列（先輸入長度n，接著輸入n個整數），將陣列內容由小到大排序，並輸出排序後的陣列。
 * 作者: 學生
 * 日期: 2025-11-12
 */

import java.util.Arrays;
import java.util.Scanner;

public class CSIE_41343109_1031_03 {
    public static void main(String[] args) {
        Scanner scn = new Scanner(System.in);

        // 讀取陣列長度
        if (!scn.hasNextInt()) {
            scn.close();
            return;
        }
        int n = scn.nextInt();

        // 讀取陣列內容
        int[] arr = new int[n];
        for (int i = 0; i < n; i++) {
            if (scn.hasNextInt()) {
                arr[i] = scn.nextInt();
            } else {
                // 若輸入不足，預設為0，避免例外
                arr[i] = 0;
            }
        }

        // 進行升冪排序
        Arrays.sort(arr);

        // 輸出排序後的陣列，元素以空格分隔，末尾換行
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(arr[i]);
        }
        System.out.println(sb.toString());

        scn.close();
    }
}

修改說明：
- 將原先的 2D char 陣列與輸出無關的邏輯移除，改用整數一維陣列來儲存使用者輸入的數值，符合題意。
- 使用 Arrays.sort(arr) 進行升冪排序，簡單且可靠，支援負數與任意長度的陣列。
- 改善輸出格式，使用空格分隔且不產生多餘空格，輸出與題目範例相符。
- 進行基本的輸入檢查與資源釋放（關閉 Scanner），提高穩定性。
- 程式命名與註解新增檔案名稱、功能說明，提升可讀性與維護性。
