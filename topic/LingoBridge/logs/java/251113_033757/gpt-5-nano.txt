public class PrintNameCode {
    public static void main(String[] args) {
        // 檔案名稱：PrintNameCode.java
        // 功能：印出虎科大與學生姓名每一字及其 Unicode 內碼，先印校名再印學生姓名
        // 作者：黃榮裕
        // 日期：2025-11-12

        // 校名與學生姓名，先拼接再逐字輸出對應內碼
        String schoolName = "虎科大";
        String studentName = "黃榮裕";
        String text = schoolName + studentName;

        // 逐字輸出每個字及其 Unicode 內碼
        for (int i = 0; i < text.length(); i++) {
            char c = text.charAt(i);
            int code = (int) c; // Unicode 內碼
            System.out.println(c + " " + code);
        }
    }
}

修改原因說明
- 對齊檔案命名與公開類別名稱：Java 要求 public 類別名稱與檔案名稱一致，否則編譯會失敗。把類別名稱改為 PrintNameCode，並建議將檔案命名為 PrintNameCode.java，以確保能正確編譯與執行。
- 提高可讀性與維護性：將中文姓名與校名分別存於 schoolName 與 studentName 變數，讓未來若要改名或做其他字串組合時更易於維護。
