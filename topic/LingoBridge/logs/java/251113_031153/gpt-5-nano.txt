修改後完整程式碼
/*
檔案名稱: WordCodePrinter.java
功能描述: 印出「虎科大」與使用者輸入的中文姓名的每個字及其對應的內碼；每行格式為「字 空格 編碼」。
作者: 你的姓名
日期: 2025-11-12
*/
public class WordCodePrinter {
    public static void main(String[] args) {
        // 先印出虎科大
        String tiger = "虎科大";
        printCharsWithCodes(tiger);

        // 再將學生的中文姓名輸出（若有輸入），把所有參數合併為一個字串
        if (args != null && args.length > 0) {
            String name = String.join("", args);
            printCharsWithCodes(name);
        }
    }

    // 輔助方法：印出字串中每一個字與其對應的內碼，一行一字
    private static void printCharsWithCodes(String s) {
        for (int i = 0; i < s.length(); i++) {
            char ch = s.charAt(i);
            System.out.println(ch + " " + (int) ch);
        }
    }
}

- 引入輔助方法 printCharsWithCodes 加強可讀性與重用性，同時改善未來維護性。
