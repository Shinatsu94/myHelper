檔案名稱: CSIE_51015130_1003_1.java
功能: 讀入兩個整數，印出其值後，使用不使用額外變數的 XOR 交換法互換兩個變數，並印出交換後的值。
作者: 51015130
日期: 2025-11-15
說明: 本範例透過 a = a ^ b; b = a ^ b; a = a ^ b; 完成交換，且僅使用 a、b 兩個變數。
*/

import java.util.Scanner;

public class CSIE_51015130_1003_1 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int a = sc.nextInt();
        int b = sc.nextInt();

        // 使用不產生額外變數的 XOR 交換
        a = a ^ b;  // a 變為 a ^ b
        b = a ^ b;  // b 變為 (a ^ b) ^ b = a
        a = a ^ b;  // a 變為 (a ^ b) ^ a = b

        System.out.println(a);
        System.out.println(b);

        sc.close();
    }
}
