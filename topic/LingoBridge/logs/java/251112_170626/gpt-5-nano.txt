 * 作者: 學生

 * 備註: 符合題目需求；使用選擇排序，輸出格式改為不帶尾端空格，結尾換行
 */

import java.util.Scanner;

public class CSIE_41343232_1031_1 {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);

        int n = input.nextInt();
        int[] arr = new int[n];

        for (int i = 0; i < n; i++) {
            arr[i] = input.nextInt();
        }

        // 選擇排序：由小到大
        for (int i = 0; i < n - 1; i++) {
            int min = i;
            for (int j = i + 1; j < n; j++) {
                if (arr[j] < arr[min]) {
                    min = j;
                }
            }
            if (min != i) {
                int tmp = arr[i];
                arr[i] = arr[min];
                arr[min] = tmp;
            }
        }

        // 輸出：避免尾端多餘的空格
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(arr[i]);
        }
        System.out.println(sb.toString());

        input.close();
    }
}

- 變數命名與可讀性：將長度變數改為 n，陣列改為 arr，提升命名意義與閱讀性。
- 保留演算法邏輯：保留原本的選擇排序實作，簡單且易於理解，符合「簡單、合理、正確的邏輯」要求。
