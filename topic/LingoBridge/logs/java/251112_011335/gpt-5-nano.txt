// 檔案名稱: SwapTwoNumbers.java
// 功能: 讀入兩個整數，交換兩數並輸出交換後的結果。僅使用 Scanner 讀取輸入，交換僅用現有變數 a, b。
import java.util.Scanner;

public class SwapTwoNumbers {
  public static void main(String[] args) {
    Scanner sc = new Scanner(System.in);

    int a = sc.nextInt();
    int b = sc.nextInt();

    // 使用不額外宣告變數的交換方法
    a = a + b;
    b = a - b;
    a = a - b;

    // 輸出交換後的結果
    System.out.println(a);
    System.out.println(b);

    sc.close();
  }
}

修改之處與原因說明
- 移除第三個輸入變數與不必要的最大/中位數/最小值計算，直接滿足「交換並印出」的需求，避免與題意衝突。
- 使用兩個變數之間的加減法交換，滿足「不可使用其他變數進行交換」的限制。
- 加入檔案級註解與更清晰的類別命名，提升可讀性與維護性。
- 統一簡潔的輸出格式，直接輸出交換後的兩個數值，各佔一行，符合題目樣本。
