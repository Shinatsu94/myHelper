- 注：內碼以 Unicode code point（十進位）表示，使用 Java 字元直取做法，程式可於 UTF-8 編碼環境下正確執行。

程式碼：
```java
/**
 * 程式名稱: 印出虎科大及姓名內碼
 * 功能: 將「虎科大」及學生姓名的每個字及其對應的 Unicode 內碼印出（十進位）。
 * 作者: 教學助理
 * 日期: 2025/11/12
 */
public class PrintTigerAndNameCode {
    public static void main(String[] args) {
        // 先印出虎科大，再印學生姓名的每個字及其內碼
        String text = "虎科大林武杰"; // 六個字，順序為：虎 科 大 林 武 杰
        for (int i = 0; i < text.length(); i++) {
            char ch = text.charAt(i);
            int codePoint = ch; // Unicode 內碼（十進位表示）
            System.out.println(ch + " " + codePoint);
        }
    }
}
```

修改說明原因
- 移除原程式的無限迴圈與輸入依賴，改為一次性輸出符合題意的內容，避免無法結束的執行與不必要的輸入處理。

- 將內碼以 Unicode 十進位表示，直接反映 Java 的內部編碼機制，避免依賴特定編碼頁面，適用於跨平台的輸出。
