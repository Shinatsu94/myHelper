```java
/*
 * File: ArraySortAscending.java
 * 功能: 讀取整數陣列長度與元素，將其由小到大排序，並輸出排序結果
 * 作者: 教授
 * 日期: 2025-11-14
 * 版本: 1.0
 */

import java.util.Scanner;
import java.util.Arrays;

public class ArraySortAscending {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 讀取陣列長度
        if (!sc.hasNextInt()) {
            sc.close();
            return;
        }
        int n = sc.nextInt();

        // 處理無效長度
        if (n <= 0) {
            sc.close();
            System.out.println();
            return;
        }

        int[] arr = new int[n];

        // 讀入 n 個整數
        for (int i = 0; i < n; i++) {
            if (sc.hasNextInt()) {
                arr[i] = sc.nextInt();
            } else {
                // 若輸入不足，預設為 0，避免程式執行失敗
                arr[i] = 0;
            }
        }
        sc.close();

        // 排序
        Arrays.sort(arr);

        // 輸出，避免尾端多餘空格
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(arr[i]);
        }
        System.out.print(sb.toString());
    }
}
```

說明
- 本版本以整數陣列與 Arrays.sort 作為核心，能正確處理包含負數的情況，並符合題目「任意長度的一維陣列從小到大排序」的要求。
- 輸出格式與題目示例相符，且不輸出不必要的分隔符或額外換行。若實際測資需要換行，可以在 System.out.print(sb.toString()) 後加 System.out.println();。
