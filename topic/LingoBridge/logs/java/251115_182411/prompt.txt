題目：
a number is called a well-ordered number if all of its digits are strictly incressing from left to right.
For example, 258 (2 < 5 < 8) is a well-ordered number.
Write a program to determine a user input number whether it is a well-ordered number.

題目要求:
- 程式僅執行一次就結束，無需使用EOF

【INPUT-1】
258

【OUTPUT-1】
258 is well-ordered number.

【INPUT-2】
255

【OUTPUT-2】
255 is not well-ordered number.

【INPUT-3】
285

【OUTPUT-3】
285 is not well-ordered number.

學生作答程式：
import java.util.Scanner;

public class WellOrderedChecker {

    public static boolean isWellOrdered(String numberStr) {
        int n = numberStr.length();
        
        if (n <= 1) {
            return false;
        }
        
        for (int i = 1; i < n; i++) {
            if (numberStr.charAt(i - 1) >= numberStr.charAt(i)) {
                return false;
            }
        }
        
        return true;
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        if (scanner.hasNext()) {
            String inputStr = scanner.next();
            
         
            try {
                int inputNumber = Integer.parseInt(inputStr);
                
                if (isWellOrdered(inputStr)) {
                    System.out.println(inputNumber + " is well-ordered number.");
                } else {
                    System.out.println(inputNumber + " is not well-ordered number.");
                }
            } catch (NumberFormatException e) {
       
                System.out.println(inputStr + " is not a valid number.");
            }
        }
        
        scanner.close();
    }
}
