// 檔案名稱示例：SortingAscending.java
// 功能: 讀取長度為 n 的整數陣列，將內容升序排序並輸出
// 作者: 教師姓名
// 日期: 2025-11-12

import java.util.Arrays;
import java.util.Scanner;

public class SortingAscending {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int n = sc.nextInt();
        int[] arr = new int[n];

        for (int i = 0; i < n; i++) {
            arr[i] = sc.nextInt();
        }

        // 使用 Java 標準排序提高穩定性與可讀性
        Arrays.sort(arr);

        // 使用 StringBuilder 組字串，避免多次輸出造成效能下降
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < arr.length; i++) {
            if (i > 0) sb.append(' ');
            sb.append(arr[i]);
        }

        System.out.println(sb.toString());

        sc.close();
    }
}

修改部分說明與理由：
- 使用 Arrays.sort(arr) 替代自訂的選擇排序：簡化實作、降低錯誤機率，且對整數陣列提供標準且高效的排序，提升程式的可讀性與維護性。
- 輸出改為使用 StringBuilder：避免在迴圈中多次呼叫 System.out.print，提升效能，且可以更整齊地處理間格放置，輸出結果與格式更穩定。
- 增加檔案頭註解（檔案名稱、功能、作者、日期）：符合備註要求，方便日後維護與評分。
- 變數與類別命名改為描述性名稱：例如 class SortingAscending，讓程式意圖更清楚，便於他人閱讀。
