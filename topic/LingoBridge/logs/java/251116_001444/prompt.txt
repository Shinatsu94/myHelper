題目：
【題目】
輸入一個數字A, 找尋 1 到 100000之間有多少數字每一個位數的加總結果等於A。
例如：當A=6, 則數字 123 -> 1+2+3=6 符合要求。
程式輸出為1-100000之間，符合要求的數字數量。

【INPUT】
6

【OUTPUT】
210

學生作答程式：
import java.util.Scanner;

public class csie_41343235_1003_3 {
    public static void main(String[] args) {
        int B = 0, sum, d;

        Scanner sc = new Scanner(System.in);
        d = sc.nextInt();

        for (int i = 1; i <= 100000; i++) {
            int num = i;
            sum = 0;

            // 將數字 i 的每一位數加總起來
            while (num > 0) {
                sum += num % 10;
                num /= 10;
            }

            if (sum == d)
                B++;
        }

        System.out.println(B);
    }
}
