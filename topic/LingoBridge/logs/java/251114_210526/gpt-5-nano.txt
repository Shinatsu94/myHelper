修改後完整程式碼（已優化）：
```java
/*
檔案名稱: CSIE_41343219_1031_1.java
功能: 讀入整數陣列長度與元素，對元素由小到大排序，輸出排序結果
作者: 您的姓名
日期: 2025-11-14
說明: 不使用 EOF，執行一次即可結束；支援負數；輸出格式為以空格分隔的排序結果，末尾換行。
*/

import java.util.*;

public class CSIE_41343219_1031_1 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        if (!sc.hasNextInt()) {
            sc.close();
            return;
        }

        int n = sc.nextInt();
        int[] ary = new int[Math.max(0, n)];

        for (int i = 0; i < ary.length && sc.hasNextInt(); i++) {
            ary[i] = sc.nextInt();
        }

        // 使用內建排序提升效率
        Arrays.sort(ary);

        // 輸出排序結果，數字以單一空格分隔，末尾換行
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < ary.length; i++) {
            if (i > 0) sb.append(' ');
            sb.append(ary[i]);
        }
        sb.append('\n');
        System.out.print(sb.toString());

        sc.close();
    }
}
```
