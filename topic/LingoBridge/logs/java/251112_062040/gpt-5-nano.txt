/*
 * File: LeibnizPi.java
 * 功能: 計算 pi 的莱布尼茲級數近似，輸出 i 為 10000, 20000, ..., 100000 時的值，並符合指定輸出格式。
 * 作者: 教師
 * 日期: 2025-11-11
 */
public class LeibnizPi {
    public static void main(String[] args) {
        int maxI = 100000;
        double sum = 0.0;
        // 累積求和，逐項加入以提升效率，並在每滿足條件時輸出結果
        for (int k = 0; k < maxI; k++) {
            double term = (k % 2 == 0) ? 1.0 / (2.0 * k + 1.0) : -1.0 / (2.0 * k + 1.0);
            sum += term;
            if ((k + 1) % 10000 == 0) {
                int iVal = k + 1;
                double pi = 4.0 * sum;
                // 輸出格式：i 以六位寬顯示，pi 保留小數點後十位
                System.out.printf("i = %6d, pi = %.10f%n", iVal, pi);
            }
        }
    }
}

- 修改說明
  - 直接針對題意要求實作：使用莱布尼茲級數逐項累加並輸出 i 為 10000 的倍數時的 pi 值，符合「i 顯示到6位數、pi 顯示到小數點後10位」的格式要求。
  - 采用增量累加的做法避免每次都重新計算整個和，提升效率。
  - 使用 System.out.printf 採用寬度格式化，確保輸出符合題目給定的對齊與格式。
