- 保留兩數互換的 XOR 交換法，因題目限制不得使用其他變數進行交換；並加入註解說明交換思路及變數用途。
- 程式結尾加上 scn.close()，以釋放 Scanner 資源。

   程式介紹: 讀入兩個整數，印出原值，交換後再印出新值

import java.util.Scanner;
public class csie_41343233_1003_1{
    public static void main(String[] args){
        Scanner scn = new Scanner(System.in);
        int firstValue = scn.nextInt();
        int secondValue = scn.nextInt();

        // 使用位元異或運算交換兩個變數的值，僅使用原本的兩個變數
        firstValue = firstValue ^ secondValue;
        secondValue = firstValue ^ secondValue;
        firstValue = firstValue ^ secondValue;

        System.out.println(firstValue);
        System.out.println(secondValue);

        scn.close();
    }
}
