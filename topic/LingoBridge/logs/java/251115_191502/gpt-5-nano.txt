修改後完整程式碼
```java
/*
檔案名稱: ReverseDigits.java
功能: 將輸入的整數的位數反轉，保留符號；例如 123 -> 321、-123 -> -321、2500 -> 52。
作者: 教授
日期: 2025-11-15
*/
import java.util.Scanner;

public class ReverseDigits {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        long number = scanner.nextLong();
        scanner.close();

        long reversed = 0;
        long n = number;

        // 使用同一迴圈解決正負號與位數反轉
        while (n != 0) {
            long digit = n % 10;
            reversed = reversed * 10 + digit;
            n /= 10;
        }

        System.out.println(reversed);
    }
}
```

說明修改部分的原因
- 使用 long 避免在反轉後超出 int 的表示範圍，提升穩定性與適用性。
- 以單一迴圈處理正負號，簡化邏輯並降低維護成本。
- 改用符合命名慣例的類別名稱，提升可閱讀性與與其他專案的整合性。
- 增加標頭註解，符合作業要求與方便日後維護。
