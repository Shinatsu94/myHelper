修改後完整程式碼（Java）：
- 功能說明：計算 π 的 Leibniz 公式近似值。輸出 i = 10000, 20000, ..., 100000 對應的 pi，小數點後十位。之後找出最小的 i，令 pi 的四捨五入到小數點後五位等於 3.14159，並輸出該 i，格式符合題目要求。
- 使用逐步累加的方式以避免每次重新計算整個級數，提高效率。

/*
File: LeibnizPiCalculator.java
Description: 使用 Leibniz 公式計算 π 的近似值，輸出指定 i 值的 pi 值，並找出使 pi 四捨五入到小數點後五位為 3.14159 的最小 i。
Author: Professor
Date: 2025-11-11
*/
public class LeibnizPiCalculator {
    public static void main(String[] args) {
        // Section 1: 輸出 i = 10000, 20000, ..., 100000 的 pi 值，pi 保留十位小數
        double sum = 0.0;
        int sign = 1; // k=1 時的符號，因為公式以 (2k-1) 分母
        for (int i = 1; i <= 100000; i++) {
            sum += sign * (1.0 / (2.0 * i - 1.0));
            sign = -sign;
            if (i % 10000 == 0) {
                double piApprox = 4.0 * sum;
                System.out.printf("i = %6d, pi = %.10f%n", i, piApprox);
            }
        }

        // Section 2: 找出最小的 i，使 pi 四捨五入到小數點後五位等於 3.14159
        sum = 0.0;
        sign = 1;
        int iTarget = -1;
        for (int i = 1; ; i++) {
            sum += sign * (1.0 / (2.0 * i - 1.0));
            sign = -sign;
            double piApprox = 4.0 * sum;
            // 四捨五入到五位小數，結果若為 3.14159 則符合條件
            if (Math.round(piApprox * 100000.0) == 314159) {
                iTarget = i;
                break;
            }
            // 防止無限迴圈，設定上限保護
            if (i > 5_000_000) {
                break;
            }
        }
        System.out.println();
        System.out.printf("i = %6d%n", iTarget);
    }
}
