/*
 * File: LeibnizPi.java
 * 功能: 使用勒讓得級數 (pi = 4 * sum_{k=0}^{i-1} (-1)^k / (2k+1)) 逐步計算
 *       i = 10000, 20000, ..., 100000，輸出 i 與 pi，i 以 6 位寬度輸出，pi 以小數點後 10 位輸出
 * 作者: 資工系教授
 * 日期: 2025-11-11
 * 內容說明: 使用單一迴圈遞增累加，避免每次都從頭計算，以提升效能。
 */
public class LeibnizPi {
    public static void main(String[] args) {
        final int maxI = 100000;
        double sum = 0.0;

        // k 從 0 增加到 maxI-1，每次加上一項
        for (int k = 0; k < maxI; k++) {
            double term = (((k % 2) == 0) ? 1.0 : -1.0) / (2.0 * k + 1.0);
            sum += term;

            // 當已加到 i 條時，輸出結果
            if ((k + 1) % 10000 == 0) {
                int i = k + 1;
                double pi = 4.0 * sum;
                System.out.printf("i = %6d, pi = %.10f%n", i, pi);
            }
        }
    }
}

說明修改原因：
- 直接針對題意實作勒讓得級數，輸出格式與精度需求與題目要求一致。
- 使用遞增累加法（一次迴圈完成所有 i 的計算），避免對每個 i 重新從頭計算，提升效能。
- 輸出格式採用 System.out.printf，確保 i 以寬度 6 右對齊，pi 以小數點後 10 位顯示，且在每行的前後有單一空白分隔，符合題目要求。

改進建議（一句話）：
- 建議採用單一遞增累加的方式計算，避免重複計算，並在同一循環內同時輸出多個 i 的結果，以提高效能與可維護性。
