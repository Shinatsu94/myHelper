改進意見：移除不必要的 import、加上程式標頭與註解、優化選取排序迴圈範圍與輸出格式以提升可讀性與效率。

修改說明與原因：
- 移除未使用的 import（import java.sql.Array;），避免混淆與潛在編譯問題。
- 增加清晰的標頭註解，說明檔案名稱、功能與日期，提升可維護性。
- 選取排序的內層迴圈改為從 i+1 開始，外層迴圈遍歷到 arr.length-1，減少不必要的比較。
- 輸出改用 StringBuilder 組字串，避免多次輸出造成 I/O 開銷，且可避免尾端多空格問題。
- 保留負數與任意長度支援，且對長度為 0 的情況保持穩定輸出（空行）。

修改後完整程式碼：
/*
檔案名稱: CSIE_41343212_1031_1.java
功能: 將任意長度的一維整數陣列作升序排序，輸入長度與元素，輸出排序後結果
作者: 請填寫
日期: 2025-11-14
*/

import java.util.Scanner;

public class CSIE_41343212_1031_1 {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);
        int length = input.nextInt();
        int[] arr = new int[Math.max(0, length)];
        for (int i = 0; i < length; i++) {
            arr[i] = input.nextInt();
        }

        // Selection sort: find the minimum in arr[i..] and swap with arr[i]
        for (int i = 0; i < arr.length - 1; i++) {
            int minIndex = i;
            for (int j = i + 1; j < arr.length; j++) {
                if (arr[j] < arr[minIndex]) {
                    minIndex = j;
                }
            }
            if (minIndex != i) {
                int tmp = arr[i];
                arr[i] = arr[minIndex];
                arr[minIndex] = tmp;
            }
        }

        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < arr.length; i++) {
            if (i > 0) sb.append(' ');
            sb.append(arr[i]);
        }
        System.out.println(sb.toString());

        input.close();
    }
}
