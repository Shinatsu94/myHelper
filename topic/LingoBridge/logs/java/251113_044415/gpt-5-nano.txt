註解說明: 使用直觀的字元陣列，逐一印出字元及其編碼，便於驗證內碼是否正確。
*/
public class PrintNameEncoding {
    public static void main(String[] args) {
        // 依序的中文字元，先印虎科大，後印學生中文姓名林武杰
        char[] chars = {'虎', '科', '大', '林', '武', '杰'};

        for (char ch : chars) {
            // 以十進位輸出字元的 Unicode 編碼
            System.out.println(ch + " " + (int) ch);
        }
    }
}
