```java
/**
 * 程式名稱 : 交換兩整數，不使用額外變數
 * 功能說明：讀取兩個整數，交換它們的值，並印出交換後的結果（各占一行）
 * 日期: 2025-11-11
 * 姓名: 教授助理
 * 學號: -
 */
import java.util.Scanner;

public class SwapTwoIntegersNoTemp {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);

        int A = input.nextInt();
        int B = input.nextInt();

        // 不使用額外變數，利用加減法在兩個變數間交換值
        A = A + B;
        B = A - B;
        A = A - B;

        System.out.println(A);
        System.out.println(B);

        input.close();
    }
}
```

修改說明
- 對原作的問題描述與輸出不符之處進行修正，改為符合題意的需求：讀入兩個整數，交換它們的值後輸出，且每個輸出占一行。
- 以只能使用兩個變數的限制，採用加減法進行交換，未引入臨時變數，符合「不可使用其他變數進行數字交換」的限制。
- 移除原作用於多數排序的邏輯與三數輸入的部分，聚焦於本題需求，並加上清楚的註解與較具描述性的檔案類別名稱。
- 程式格式方面，調整了變數命名與註解，並保持適當縮排，便於讀懂與維護。

一句話改進建議
- 如可接受，使用位元運算 XOR 交換法或改用 long 以避免整數溢位風險，會讓程式在極端數值下更穩定。
