import java.util.Scanner;

public class CSIE_41343205_1003_1{
    public static void main(String[] args){
        // 建立 Scanner 讀取使用者輸入
        Scanner sc = new Scanner(System.in);

        // 以有意義的變數名稱讀入
        int A = sc.nextInt();
        int B = sc.nextInt();

        // 使用 XOR 交換，符合「不可使用其他變數」的要求
        A = A ^ B;
        B = A ^ B;
        A = A ^ B;

        // 輸出交換後的結果
        System.out.println(A);
        System.out.println(B);

        // 關閉資源
        sc.close();
    }
}

- 增加標頭註解，說明檔案名稱、功能、作者與日期，提升可維護性與識讀性。
- 將變數名稱改為 A、B（具意義的命名相較於 a、b 更易理解，符合格式性要求）。
- 加入程式內註解，說明使用 XOR 交換的原因與步驟，方便後人維護。
- 加入 sc.close() 以釋放資源，遵循良好資源管理慣例。
