- 讓程式結構更清晰，變數命名更有意義，符合格式評分的期待。

修改後完整程式碼
/*
檔案名稱: Main.java
功能: 建立並列印下三角形星號輸出，輸出每行的星號數等於行號（從1開始）
作者: 教師
日期: 2025-11-14
說明: 不使用多餘的二維陣列，直接輸出以符合題意「元素之間無需任何空白符號」的要求。
*/
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();

        // 基本輸入驗證，若不在規範內直接結束
        if (n <= 0 || n > 20) {
            sc.close();
            return;
        }

        // 直接輸出下三角形：第 i 行輸出 i+1 個 '*'
        for (int i = 0; i < n; i++) {
            for (int j = 0; j <= i; j++) {
                System.out.print('*');
            }
            System.out.println();
        }

        sc.close();
    }
}
