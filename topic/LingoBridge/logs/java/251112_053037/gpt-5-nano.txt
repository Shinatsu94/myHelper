/*

Description: 使用勒布尼茲級數計算 pi。對 i = 10000, 20000, ..., 100000 計算 pi，並輸出 i（6 位寬）與 pi（小數點後10位）。
作者: 教師姓名
日期: 2025-11-11
*/

import java.io.PrintStream;

public class LeibnizPi {
    public static void main(String[] args) {
        double sum = 0.0; // 累積的和，等於 sum_{k=1..i} (-1)^{k+1} / (2k-1)
        PrintStream out = System.out;

        // 逐步計算，當 i 為 10000 的倍數時輸出結果
        for (int i = 1; i <= 100000; i++) {
            double term = 1.0 / (2 * i - 1);
            if ((i & 1) == 1) { // 奇數項，正值
                sum += term;
            } else { // 偶數項，負值
                sum -= term;
            }

            if (i % 10000 == 0) {
                double pi = sum * 4.0;
                out.printf("i = %6d, pi = %.10f%n", i, pi);
            }
        }
    }
}

修改說明與原因：
- 將原有的與題目無關的程式碼改為實作勒布尼茲級數求 pi 的邏輯，符合作業需求與評分標準的正確性與邏輯。
- 使用累積和的方式逐步計算，避免每次都重新計算整個和，提升效率；每當 i 為 10000 的倍數時輸出結果，符合題目「輸出 i 為 10000、20000、…、100000 時的 pi 值」的要求。
- 輸出格式部分：

  - pi 使用十進位小數點後10位輸出，符合「pi 顯示到小數點後10位」。
  - 逐行輸出格式為 "i = %6d, pi = %.10f"，與題目給定的示例格式高度一致。
