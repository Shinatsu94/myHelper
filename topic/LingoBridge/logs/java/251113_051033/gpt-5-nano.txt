修改後完整程式碼
以下為修改後的完整 Java 程式，直接輸出「虎科大」與「林武杰」三字各自的 Unicode 碼點。請以實際的學生姓名替換 name 字串中的內容。

/*
 * 檔案名稱: CSIE_PrintNameCodes.java
 * 功能: 依序輸出「虎科大」與學生姓名的每個字及其 Unicode 內碼
 * 作者: 教師
 * 日期: 2025-11-12
 */

public class CSIE_PrintNameCodes {
    public static void main(String[] args) {
        String school = "虎科大";
        String name = "林武杰"; // 請依實際學生姓名修改此字串

        // 逐字輸出「虎科大」的每個字及其 Unicode 內碼
        for (int i = 0; i < school.length(); i++) {
            char ch = school.charAt(i);
            System.out.println(ch + " " + (int) ch);
        }

        // 逐字輸出學生姓名的每個字及其 Unicode 內碼
        for (int i = 0; i < name.length(); i++) {
            char ch = name.charAt(i);
            System.out.println(ch + " " + (int) ch);
        }
    }
}

修改說明要點
- 直接以字元與其整數型內碼輸出，避免依賴特定編碼表與外部輸入，確保在不同環境下的可預期性與穩定性。
- 整潔的程式結構與可讀性：用清晰的變數名與註解標示，方便後續維護與他人閱讀。
