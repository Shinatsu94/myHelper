/**
 * 程式名稱: 讀入兩個整數並交換並印出（不使用額外變數）
 * 功能說明： 讀入變數 A、B，印出交換後的值（每行一個）
 * 日期: 2025/11/11
 * 姓名: 教授範例
 * 學號: XXXXX
 */
import java.util.Scanner;

public class CSIE_41343224_1003_2 {

    public static void main(String[] args) {
        // 使用 try-with-resources 以自動關閉 Scanner
        try (Scanner input = new Scanner(System.in)) {
            int A = input.nextInt();
            int B = input.nextInt();

            // 不使用額外變數，利用算術交換法：
            // A = A + B; B = A - B; A = A - B;
            // 注意：此法可能因極大數值而造成溢位風險
            A = A + B;
            B = A - B;
            A = A - B;

            // 交換後的值即為 swapped 結果，依序印出
            System.out.println(A);
            System.out.println(B);
        }
    }
}

修改原因：
- 針對題目要求，必須把兩個數字交換並印出交換後的結果；原本的程式讀取三個數字並輸出排序結果，與需求不符，因此需要改為讀取兩個變數、在不使用額外變數的條件下完成交換，並依題意印出交換後的值。

改進建議（一句話）：
- 為避免溢位風險，長遠看可以改用位元異或交換法（若確保整數型別不影響位元運算），或在可接受的情況下使用一個暫存變數以提升穩定性與可讀性；同時建議對輸入範圍做基本檢核。
