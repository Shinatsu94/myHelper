import java.util.Scanner;

/**
 * 檔案名稱: CSIE_51015130_1003_3.java
 * 功能說明: 讀入兩個整數 A, B，先印出原值，再交換兩數後印出新值。
 * 作者: 論壇使用者
 * 日期: 2025-11-11
 * 備註: 僅使用 Scanner 讀取輸入，透過不使用臨時變數 (使用 XOR 交換) 來完成交換。
 */
public class CSIE_51015130_1003_3{
	public static void main(String[] args){
		Scanner sc = new Scanner(System.in);
		int A = sc.nextInt();
		int B = sc.nextInt();

		// 先印出原值
		System.out.println(A);
		System.out.println(B);

		// 使用 XOR 交換 A 與 B，符合「不可使用其他變數進行數字交換」的要求
		A = A ^ B;
		B = A ^ B;
		A = A ^ B;

		// 印出交換後的值
		System.out.println(A);
		System.out.println(B);

		sc.close();
	}
}

修改說明與原因：
- 將原本與題目不符的邏輯全部替換成符合題意的實作：讀入兩個數字，先輸出原值，然後交換再輸出。原程式在執行時與題意不符，且主要功能是計算數字和等，不符合需求。
- 變數命名與結構調整：使用 A、B 兩個整數變數分別存放兩個輸入值，讓流程變得清晰易懂。
- 交換策略：採用無臨時變數的 XOR 交換法（A = A ^ B; B = A ^ B; A = A ^ B），符合「不可使用其他變數進行數字交換」的限制，同時避免了可能的整數溢位問題（相較於某些加減法交換方案）。
- 程式格式與註解：加入標頭註解，說明檔案名稱、功能、作者、日期及交換方法，提升可讀性與維護性；並保持適當縮排與註解，符合格式評分標準。

改進建議或反饋（一句話）：
