修改後完整程式碼（符合題意與限制）
以下程式會讀入兩個整數 A 與 B，先以 XOR 交換法在原地交換兩者值，然後輸出交換後的結果（每個值各佔一行），不使用額外變數做交換。

/*
檔案名稱: SwapTwoNumbers.java
功能: 讀入兩個整數，交換兩個變數的值並印出交換後的結果。
作者: 你的老師
日期: 2025-11-11
*/
import java.util.Scanner;

public class SwapTwoNumbers {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int a = sc.nextInt();
        int b = sc.nextInt();

        // 使用 XOR 交換，不使用額外變數
        a = a ^ b;
        b = a ^ b;
        a = a ^ b;

        // 輸出交換後的兩個值，符合題目樣例
        System.out.println(a);
        System.out.println(b);

        sc.close();
    }
}
