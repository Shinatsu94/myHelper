作者: 請填寫
日期: 2025-11-15
描述: 提升可讀性，將數位和的計算提取成方法，範圍修正為 1..100000，變數命名改為有意義名稱，並加入註解
*/
import java.util.Scanner;

public class CSIE_41343121_1003_3 {
  public static void main(String[] args) {
    Scanner scn = new Scanner(System.in);
    int target = scn.nextInt();
    int count = 0;

    // 範圍調整為 1 至 100000，符合題目要求
    for (int number = 1; number <= 100000; number++) {
      if (digitSum(number) == target) {
        count++;
      }
    }

    System.out.println(count);
  }

  // 計算整數 x 的各位數字總和
  private static int digitSum(int x) {
    int sum = 0;
    while (x > 0) {
      sum += x % 10;
      x /= 10;
    }
    return sum;
  }
}

- 範圍正確性：改為從 1 開始計數，避免包含 0；100000 仍然包含在範圍內，符合題意。
- 可維護性與可讀性：把 digitSum 獨立成方法，命名清楚，方便日後維護或單元測試。
