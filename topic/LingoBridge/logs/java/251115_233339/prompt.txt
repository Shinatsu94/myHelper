題目：
【題目】
請撰寫一個 Java 程式，讓使用者輸入三個整數。
程式需利用兩種不同的方式來找出這三個數的最大值(max)、中間值(mid)和最小值(mix)。
使用三元運算子 ? : 找出三個數中的最大值和最小值。
注意：不可使用 if-else

【INPUT】
15
23
8

【OUTPUT】
8
15
23

學生作答程式：
import java.util.Scanner;
public class CSIE_41343150_1003_2{
public static void main(String[] args){
Scanner scanner=new Scanner(System.in);
System.out.print("輸入 A: ");
 int A = scanner.nextInt();
System.out.print("輸入 B: ");
 int B = scanner.nextInt();
System.out.print("輸入 C: ");
 int C = scanner.nextInt();
 int max = (A > B) 
            ? (A > C ? A : C) 
            : (B > C ? B : C);
  int min = (A < B) 
                    ? (A < C ? A : C) 
                    : (B < C ? B : C);
int mid = A + B + C - max - min;
  System.out.println("最大值 max = " + max);
        System.out.println("最小值 min = " + min);
        System.out.println("中間值 mid = " + mid);

        scanner.close();
}
}
