```java
/**
 * 檔案名稱: DigitSumCounter.java
 * 功能: 計算 1..100000 之間，各位數字之和等於輸入值 target 的數字個數
 * 作者: 你的學生
 * 日期: 2025-11-15
 * 備註: 以直覺清晰的寫法提升可讀性與可維護性
 */

import java.util.Scanner;

public class DigitSumCounter {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int target = sc.nextInt();
        sc.close();

        int count = 0;

        // 1..100000 的範圍內，若 target 不在可能的範圍內，直接輸出 0
        if (target > 0 && target <= 45) {
            for (int num = 1; num <= 100000; num++) {
                int sum = 0;
                int tmp = num;
                while (tmp > 0) {
                    sum += tmp % 10;
                    tmp /= 10;
                }
                if (sum == target) {
                    count++;
                }
            }
        } else {
            count = 0;
        }

        System.out.println(count);
    }
}
```

說明修改部分的原因與效果：
- 將 i/sum 變數命名改為更具描述性的命名，提升可讀性與維護性。
- 加入 target 的有效性檢查，避免不必要的計算並在極端情況下直接回傳結果，提升效率。
- 移除原程式中「int tmp = i++;」的混淆寫法，改成清楚的 for 迴圈使用，降低邏輯錯誤風險。
- 加入檔案頭註解與內部註解，符合作業要求的備註規範。
