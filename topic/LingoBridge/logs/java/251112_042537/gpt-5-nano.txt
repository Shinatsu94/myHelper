     程式名稱：SwapTwoNumbers
     功能：讀入兩個整數 A、B；先印出 A 與 B；再用不使用額外變數的交換法交換兩者，並印出交換後的值。
     
     內容如下（直接以檔案形式撰寫即可）：
     
     /**
      * 程式名稱 : SwapTwoNumbers
      * 功能說明： 讀入兩個整數 A、B；先印出 A 與 B 的值，之後在不使用額外變數的情況下交換兩者，並再印出交換後的值。
      * 日期: 2025/11/11
      * 姓名/學號: [學生資訊]
      */
     import java.util.Scanner;
     
     public class SwapTwoNumbers {
     
         public static void main(String[] args) {
             
             Scanner input = new Scanner(System.in);
             
             int A = input.nextInt();
             int B = input.nextInt();
             
             // 先印出原始值
             System.out.println(A);
             System.out.println(B);
             
             // 使用不需要第三個變數的交換法 (XOR swap)
             A = A ^ B;
             B = A ^ B;
             A = A ^ B;
             
             // 印出交換後的值
             System.out.println(A);
             System.out.println(B);
             
             input.close();
         }
     }

     修改原因說明：
     - 符合題目需求：兩數輸入、兩次輸出（原值與交換後值），且不使用額外變數協助交換。
     - 使用 XOR 交換法可在不另外宣告第三個變數的前提下完成交換，符合題意限制，同時避免整數溢位問題（相對於加減法交換）。
