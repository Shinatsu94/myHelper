/*
檔案名稱: SwapTwoNumbers.java
功能: 讀入兩個整數，輸出先前兩個數的交換結果
作者: CSIE_教學範例
日期: 2025-11-11
說明: 使用兩個變數 A, B，並在不新增額外變數的情況下交換兩數值，
      此處採用 XOR 交換法以避免整數溢出風險，並加上適當註解與格式化。
*/
import java.util.Scanner;

public class SwapTwoNumbers {
    public static void main(String[] args) {
        Scanner scn = new Scanner(System.in);
        int A = scn.nextInt();
        int B = scn.nextInt();

        // 使用 XOR 交換法：不需第三個變數，符合題意要求
        A ^= B;
        B ^= A;
        A ^= B;

        System.out.println(A);
        System.out.println(B);

        scn.close();
    }
}

修改部分說明
- 將變數名稱改為 A、B，並改用清晰的標註與註解，提升可讀性與維護性。
- 以 XOR 交換法替代原本的加減法交換，減少溢位風險，且符合「不使用額外變數進行交換」的限制。
- 加入標頭註解與內部註解，並統一縮排與排版，符合格式評分標準。
- 設置資源釋放（scn.close()），提高程式健壯性。
