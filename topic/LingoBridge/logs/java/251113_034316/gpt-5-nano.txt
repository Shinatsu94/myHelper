/**
 * 檔案名稱：CSIE_Unicode_Print.java
 * 功能：印出虎科大與學生中文姓名每個字及對應的內碼，順序為虎科大 -> 學生姓名
 * 作者：教師
 * 日期：2025-11-12
 * 說明：改用字元陣列與迴圈，避免硬編碼多個變數，輸出格式符合題意。
 */
public class CSIE_Unicode_Print {
    public static void main(String[] args) {
        // 虎科大三個字元
        char[] school = {'虎', '科', '大'};
        // 學生中文姓名三個字元（請依實際姓名修改）
        char[] name = {'林', '武', '杰'};

        // 先印出虎科大，並附上對應的內碼
        for (char c : school) {
            System.out.println(c + " " + (int) c);
        }

        // 再印出學生姓名，並附上對應的內碼
        for (char c : name) {
            System.out.println(c + " " + (int) c);
        }
    }
}

修改原因說明：
- 將單一字元變數改為陣列並使用迴圈，可以讓程式更容易維護與擴充（如字數改為四字、五字時僅需修改陣列內容，而不必新增多條輸出語句）。
- 重新排序輸出，使輸出順序符合題意「先印虎科大，再印學生中文姓名」。
- 移除不必要的標頭文字，避免與題目要求的輸出格式衝突。
