```java
/*
檔案名稱: SortArrayAscending.java
功能說明: 讀取整數 n 代表陣列長度，接著讀入 n 個整數，將陣列內容由小到大排序，最後輸出排序後的陣列（以空格分隔）。
作者: 你的名字
日期: 2025-11-12
*/
import java.util.Arrays;
import java.util.Scanner;

public class SortArrayAscending {
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);

        // 讀取陣列長度
        if (!in.hasNextInt()) {
            in.close();
            return;
        }
        int n = in.nextInt();
        int[] arr = new int[n];

        // 讀取 n 個整數，若不足以讀入 n 個，依需求可改成拋出例外或補 0；此處以 0 作為預設
        for (int i = 0; i < n; i++) {
            if (in.hasNextInt()) {
                arr[i] = in.nextInt();
            } else {
                arr[i] = 0;
            }
        }

        in.close();

        // 排序
        Arrays.sort(arr);

        // 輸出，元素之間以單一空格分隔，末端不留多餘空格
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(arr[i]);
        }
        System.out.println(sb.toString());
    }
}
```

- 邏輯與設計
  - 使用簡單直接的排序方法 Arrays.sort，時間複雜度 O(n log n)，適用於任意長度的整數陣列，且支援負數。
  - 輸出格式避免尾端多餘空格，與樣例輸出一致。

  - 采用清晰的類別名稱 SortArrayAscending，符合一般 Java 命名規範；檔案名稱與 public 類別一致。
