題目：
【題目】
請撰寫一個 Java 程式，建立一個下三角形的二維陣列。

陣列的高度 n 由使用者輸入。
陣列的第 i 列（從 1 開始）有 i 個元素。
數字從 1 開始，依序從上到下、從左到右填入陣列。
讓使用者輸入一個要尋找的數字 (num < 100000)。
建立完整的下三角形二維陣列 (不用印出三角形內容)。
找出 num 在陣列中的位置（列編號和行編號，皆從 1 開始），並將其印出。
如果 num 超出陣列範圍，則印出相應的提示訊息。

【input】
10
55

【output】
55[10, 10]

【input】
5
99

【output】
not find 99 in array.

學生作答程式：
import java.util.Scanner;
public class CSIE_51015120_1031_3b {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int[][] arr = new int[n][];
        int num = 1;

        for (int i = 0; i < n; i++) {
            arr[i] = new int[i + 1];
            for (int j = 0; j <= i; j++) {
                arr[i][j] = num++;
            }
        }
        int target = sc.nextInt();

        int maxValue = n * (n + 1) / 2;
        if (target > maxValue || target <= 0) {
            System.out.println("not find " + target + " in array.");
            return;
        }

        boolean found = false;
        for (int i = 0; i < n && !found; i++) {
            for (int j = 0; j <= i; j++) {
                if (arr[i][j] == target) {
                    System.out.println(target + "[" + (i + 1) + ", " + (j + 1) + "]");
                    found = true;
                    break;
                }
            }
        }
        if (!found) {
            System.out.println("not find " + target + " in array.");
        }
        sc.close();
    }
}
