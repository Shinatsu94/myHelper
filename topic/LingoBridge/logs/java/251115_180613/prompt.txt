題目：
對於以下公式: 
pi = 4(1-1/3+1/5-1/7+1/9-1/11+1/13-...-1/2i-1) 
(a) 當i = 10000, 20000, ...到 100000,所求到的pi值為多少?
(b) 若要得到 pi = 3.14159, i 值最小為多少?

題目要求:

- i 顯示到6位數

public class csie_41343150_1017_1 {

    public static double cal(int n) {
        double u = 0.0;
        for (int i = 0; i <= n; i++) {
            if (i % 2 == 0) {
                u += 1.0 / (2 * i + 1);
            } else {
                u -= 1.0 / (2 * i + 1);
            }
        }
        return u * 4;
    }

    public static void main(String[] args) {
        for (int i = 10000; i <= 100000; i += 10000) {
            double p = cal(i);
            System.out.printf("i = %d, π ≈ %.15f%n", i, p);
        }

        int i = 0;
        while (true) {
            double p = cal(i);
            if (Math.round(p * 100000.0) / 100000.0 == 3.14159) {
                System.out.printf("i = %d, π ≈ %.15f%n", i, p);
                break;
            }
            i++;
        }
    }
}
