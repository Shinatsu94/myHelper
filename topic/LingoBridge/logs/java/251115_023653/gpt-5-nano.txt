修改重點

- 增加檔案標頭註解，說明檔案名稱、功能、作者、日期等。

修改後完整程式碼
```java
/*
檔案名稱: TrianglePrinter.java
功能: 建立並列印下三角形二維陣列，高度 n 由使用者輸入，範圍 1~20。輸出中不加入多餘空白。
作者: 教授
日期: 2025-11-14
*/

import java.util.Scanner;

public class TrianglePrinter {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (!sc.hasNextInt()) {
            return;
        }
        int n = sc.nextInt();

        // 依題意，範圍為 1~20，若不在範圍內則不輸出內容
        if (n < 1 || n > 20) {
            sc.close();
            return;
        }

        // 建立 n x n 的二維陣列，僅下三角設定為 '*'
        char[][] tri = new char[n][n];
        for (int i = 0; i < n; i++) {
            for (int j = 0; j <= i; j++) {
                tri[i][j] = '*';
            }
        }

        // 組裝輸出內容
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            for (int j = 0; j <= i; j++) {
                sb.append(tri[i][j]);
            }
            sb.append('\n');
        }

        System.out.print(sb.toString());
        sc.close();
    }
}
```
