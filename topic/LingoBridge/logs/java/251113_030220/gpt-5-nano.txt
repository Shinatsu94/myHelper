/*
 檔案名稱: CSIE_41343107_0926_1.java
 作者: 需填寫的學生姓名
 功能: 印出虎科大與學生姓名三個字的每個字及其對應的 Unicode 內碼點
 日期: 2025/09/26
*/
public class CSIE_41343107_0926_1 {
    public static void main(String[] args) {
        // 先印出虎科大三個字及其內碼
        char[] tigerKeDa = {'虎', '科', '大'};
        // 再印出學生中文姓名，這裡以林武杰為例
        char[] studentName = {'林', '武', '杰'};

        // 輸出每個字及其對應的 Unicode 內碼點（十進位）
        for (char ch : tigerKeDa) {
            System.out.println(ch + " " + (int) ch);
        }
        for (char ch : studentName) {
            System.out.println(ch + " " + (int) ch);
        }
    }
}

修改說明：
- 將要輸出的字符分成兩個陣列：虎科大與學生姓名，透過迴圈逐字輸出，符合「一行印一個中文字與其對應的內碼」的規格。
- 使用 (int)char 取得 Unicode 內碼點的十進位值，避免字元串連造成混淆，且輸出格式清楚、易於比對。
- 這樣的寫法比直接硬寫多個 System.out.println 更具擴充性，若未來姓名或字數變動，只需調整陣列內容即可。
- 保留了原本註解與檔案頭，描述檔案名稱、功能、日期等，符合備註要求；變數命名改為具語意的陣列名稱，提升可讀性。

改進建議或反饋（一句話）：
務必嚴謹遵循題目輸出格式，並以可擴充的資料結構（如陣列/清單與迴圈）處理字數變動的情況，同時使用清晰的變數命名與註解。
