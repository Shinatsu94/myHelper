/*
 * file: SwapTwoNumbers.java
 * author: 李品劭
 * date: 2025-11-11
 * description: 讀入兩個整數，印出交換後的結果；交換過程不使用額外的變數
 */

import java.util.Scanner;

public class SwapTwoNumbers {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);
        int a = input.nextInt();
        int b = input.nextInt();

        // 使用不需額外變數的算術交換
        a = a + b;
        b = a - b;
        a = a - b;

        // 印出交換後的結果 (符合題目輸出樣本)
        System.out.println(a);
        System.out.println(b);

        input.close();
    }
}

修改原因說明
- 目的性：改用能正確滿足題目需求的實作，確保讀入兩個數字並在交換後輸出正確結果。
- 限制遵循：採用算術交換避免引入第三個臨時變數，符合「不可使用其他變數進行數字交換」的條件。
- 可讀性：變數命名採用小寫結構 a、b，且加入清晰註解與檔案標頭，讓程式更易理解與維護。
