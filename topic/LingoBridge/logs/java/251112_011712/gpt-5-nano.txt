修改後完整程式碼
import java.util.Scanner;

/**
 * 需求：讀入兩個整數 A, B，先印出原本的兩個數值，
 * 然後在不使用額外變數的情況下交換兩個數值，再印出交換後的結果。
 * 注意：僅使用 Scanner 讀取輸入，不可用其他變數作為暫存。
 */
public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        long A = sc.nextLong();
        long B = sc.nextLong();

        // 印出原始順序
        System.out.println(A);
        System.out.println(B);

        // 不使用額外變數，使用加法與減法交換 A 與 B 的值
        A = A + B;
        B = A - B;
        A = A - B;

        // 印出交換後的順序
        System.out.println(A);
        System.out.println(B);

        sc.close();
    }
}

修改說明
- 符合題意：原程式讀入三個數字並依序輸出最小/中位數/最大值，與題目要求的讀兩數、先印原值再印交換後值的需求不符，故改為讀取兩個數字並按題意輸出。
- 交換實作：改用不使用第三個變數的原地交換技巧（A = A + B; B = A - B; A = A - B;），符合「不可使用其他變數進行數字交換」的限制。
- 輸入與輸出：使用最基本的輸入輸出路徑，確保輸入順序與輸出格式與題目完全一致。
- 型別與可讀性：使用 long 以降低整數溢位風險，並加上程式註解提升可讀性；同時維持簡潔清晰的變數命名。

改進建議
- 建議在實作前仔細對照題目要求，避免偏離題意與多餘的輸入輸出，並選用符合限制的交換方法以提升魯棒性。
