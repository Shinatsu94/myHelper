修改後完整程式碼（Java）
/*
File: LeibnizPiPrinter.java
功能: 依題意輸出 i 為 10000, 20000, ..., 100000 時，對應的 pi 值，pi 計算使用 Leibniz 公式
作者: 盧冠宇
日期: 2024-10-17
說明: 使用累加法逐步計算前 k 項的和，並在每 10000 項輸出一次結果，格式符合題目要求。
*/
public class LeibnizPiPrinter {
    public static void main(String[] args) {
        int maxI = 100000;
        double sum = 0.0;

        // k 從 0 到 maxI-1，每次增加一項
        for (int k = 0; k < maxI; k++) {
            // 第 k 項的符號與分母
            double term = ((k % 2 == 0) ? 1.0 : -1.0) / (2.0 * k + 1.0);
            sum += term;

            // 每增加 10000 項，輸出一次結果
            if ((k + 1) % 10000 == 0) {
                int i = k + 1;
                double pi = sum * 4.0;
                System.out.printf("i = %6d, pi = %.10f%n", i, pi);
            }
        }
    }
}

- 改善正確性：直接依題目要求實作 Leinniz 公式計算與輸出，確保每個 i 對應的 pi 值正確計算。
- 提升邏輯與效能：改用增量累加法，以單次迭代更新 sum，避免重複計算全部前項。
- 輸出格式化：使用 printf 直接控制 i 的欄寬與 pi 的小數位數，符合題目範例。
- 程式結構與註解：新增檔案頭註解，變數命名清楚，方便日後維護。
