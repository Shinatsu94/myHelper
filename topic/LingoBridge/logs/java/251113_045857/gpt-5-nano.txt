修改後的完整程式碼（可直接複製執行）
/*
檔案名稱: PrintUnicodeCode.java
功能: 印出「虎科大」三字及學生姓名每個字的 Unicode 內碼（十進位），每字一行，先印「虎科大」再印學生姓名。
作者: （可修改為實際作者）
日期: 2025-11-12
說明: 本程式預設學生姓名為林武杰。若要改成其他姓名，請修改 name 變數的字串內容。
*/
public class PrintUnicodeCode {
    public static void main(String[] args) {
        // 学生姓名預設值；可依需求修改為其他姓名
        String name = "林武杰";

        // 先輸出虎科大三字及其內碼
        printCharWithCode('虎');
        printCharWithCode('科');
        printCharWithCode('大');

        // 再輸出學生的中文姓名每個字及其內碼
        for (int i = 0; i < name.length(); i++) {
            char ch = name.charAt(i);
            printCharWithCode(ch);
        }
    }

    // 輔助方法：輸出單一字及其十進位的 Unicode 內碼
    private static void printCharWithCode(char c) {
        System.out.println(c + " " + (int) c);
    }
}

修改部分的原因
- 將不符合題意的輸入輸出邏輯移除，改以直接輸出「虎科大」與姓名的每個字及其內碼，符合題目要求與自動評分的預期輸出格式。
- 使用十進位 Unicode 內碼輸出，讓結果在不同文字編碼環境下具有穩定的可驗證性。
