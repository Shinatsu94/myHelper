功能: 讀入兩個整數，A 與 B，印出其值，將兩數交換後再印出（不使用額外變數進行交換）。

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int a = sc.nextInt();
        int b = sc.nextInt();

        // 使用 XOR 交換法，交換 a 與 b，且不使用其他變數

        b = a ^ b;
        a = a ^ b;
