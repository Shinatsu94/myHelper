// 文件名稱: CSIE_41343136_1031_2.java
// 功能說明: 將輸入的一維整數陣列由小到大排序，並輸出排序結果
// 作者: 學生原作，優化後版本
// 日期: 2025-11-12

import java.util.Arrays;
import java.util.Scanner;

public class CSIE_41343136_1031_2 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 讀取陣列長度
        if (!sc.hasNextInt()) {
            sc.close();
            return;
        }
        int n = sc.nextInt();
        int[] arr = new int[n];

        // 依序讀入陣列元素
        for (int i = 0; i < n; i++) {
            if (sc.hasNextInt()) {
                arr[i] = sc.nextInt();
            } else {
                arr[i] = 0; // 如果輸入不足，給予預設值以避免例外
            }
        }

        // 排序（簡單且可靠，支援負數）
        Arrays.sort(arr);

        // 輸出排序結果，使用單行、以空格分隔
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(arr[i]);
        }
        System.out.print(sb.toString());

        sc.close();
    }
}

修改說明與原因：
- 將原程式中錯誤的排序邏輯改為使用 Arrays.sort()，確保正確性、可讀性與對負數的支援，避免手寫排序邏輯造成資料覆寫與錯誤結果。
- 增加標頭註解與更清晰的程式註解，符合題目對程式標頭與註解的要求，提升可維護性。
- 使用 StringBuilder 組裝輸出，避免在迴圈中頻繁輸出造成效能問題，且確保輸出為單行且元素間以空格分隔。
- 加入基本的輸入檢查與資源關閉，提升健壮性。
- 變數命名與排版統一，提升格式美觀與可讀性。
