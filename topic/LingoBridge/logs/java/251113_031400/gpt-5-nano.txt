public class CSIE_41343150_0919_1 {
    public static void main(String[] args) {
        // 先印出虎科大及其對應的內碼
        printWithCode("虎科大");
        // 再印學生中文姓名及其對應的內碼
        printWithCode("林武杰");
    }

    // 輔助方法: 對字串逐字輸出「字元 線內碼(Unicode 編碼點)」
    private static void printWithCode(String text) {
        for (int i = 0; i < text.length(); i++) {
            char ch = text.charAt(i);
            System.out.println(ch + " " + (int) ch);
        }
    }
}

修改說明
- 以字串分別存放要輸出的兩段文字（虎科大與林武杰），避免未來改成其他姓名時需要修改多個輸出點。
- 使用逐字列印方式，直接輸出每個字的 Unicode 編碼點，避免硬編碼“xxx”內碼，提升正確性與可維護性。
- 新增一個輔助方法 printWithCode，將輸出邏輯集中，方便日後擴展或修改輸出格式。
- 程式結構清晰，保留基本縮排與註解，便於他人閱讀。
