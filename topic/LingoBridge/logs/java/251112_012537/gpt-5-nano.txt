```java
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        // 讀入兩個整數，分別存放於 A, B
        int A = sc.nextInt();
        int B = sc.nextInt();

        // 印出初始值
        System.out.println(A);
        System.out.println(B);

        // 不使用其他變數，使用「加法-減法」的方式交換 A 與 B
        A = A + B;
        B = A - B;
        A = A - B;

        // 印出交換後的值
        System.out.println(A);
        System.out.println(B);

        sc.close();
    }
}
```
修改說明（重點與原因）
- 將原作業改為符合題意的功能：先輸出初始的 A、B，然後交換兩者再輸出交換後的值，避免跑出與題目不符的結果。
- 使用不新增其他變數的交換方法：A = A + B; B = A - B; A = A - B，這樣只有 A、B 兩個變數在使用，符合「不可使用其他變數進行數字交換」的限制。
- 程式結構與命名改為較通用與清晰的形式（class Main、main 方法簽名、適當註解），以提升可讀性與可移植性。
