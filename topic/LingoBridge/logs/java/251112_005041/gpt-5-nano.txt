檔案名稱: CSIE_41343119_1003_2.java
功能: 讀入兩個整數，交換兩數，並輸出交換後的結果（僅使用兩個變數，無額外變數）
作者: 金宗翰
日期: 2025/10/03
*/
import java.util.Scanner;

public class CSIE_41343119_1003_2 {
    public static void main(String[] args) {
        Scanner scn = new Scanner(System.in);
        long a = scn.nextLong();
        long b = scn.nextLong();

        // 使用算術交換，僅使用兩個變數，避免使用額外暫存變數
        a = a + b;
        b = a - b;
        a = a - b;

        // 輸出交換後的兩個值，符合題意樣本
        System.out.println(a);
        System.out.println(b);

        scn.close();
    }
}

- 移除不必要的三個變數與多餘輸出，改以兩個變數直接完成交換，符合題目的限制與輸出格式。

改進建議（一句話）：
