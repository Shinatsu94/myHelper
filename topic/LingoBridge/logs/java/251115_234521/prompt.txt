題目：
【題目】
請撰寫一個 Java 程式，讓使用者輸入三個整數。
程式需利用兩種不同的方式來找出這三個數的最大值(max)、中間值(mid)和最小值(mix)。
使用三元運算子 ? : 找出三個數中的最大值和最小值。
注意：不可使用 if-else

【INPUT】
15
23
8

【OUTPUT】
8
15
23

學生作答程式：
/*檔案名稱:class csie_41343233_1003_2
  日期:2025/10/03
  作者:陳韋廷
  程式介紹:輸入a和b跟c的值,找出最大值與中間值跟最小值
*/
import java.util.Scanner;
public class csie_41343233_1003_2{
    public static void main(String[] args){
        Scanner scn=new Scanner(System.in);
        int max=0,min=0,mid=0;
        int a=scn.nextInt();
        int b=scn.nextInt();
        int c=scn.nextInt();
        max=a>b?a:b;
        max=c>max?c:max;
        min=a<b?a:b;
        min=c<min?c:min;
        mid=a+b+c-max-min;
        System.out.println(min);
        System.out.println(mid);
        System.out.print(max);
    }
}
