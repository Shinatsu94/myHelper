修改後的完整程式碼如下（已符合上述優化）：
```java
/*
File: Main.java
功能: 讀取長度 n 與 n 個整數，將它們從小到大排序，輸出排序後的序列，輸出以空格分隔，末尾換行。
作者: 你的名字（可自行填寫）
日期: 2025-11-14
*/

import java.util.Arrays;
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        // 使用 try-with-resources 自動關閉 Scanner
        try (Scanner sc = new Scanner(System.in)) {
            int n = sc.nextInt();
            int[] a = new int[n];
            for (int i = 0; i < n; i++) {
                a[i] = sc.nextInt();
            }

            Arrays.sort(a);

            StringBuilder sb = new StringBuilder();
            for (int i = 0; i < n; i++) {
                if (i > 0) sb.append(' ');
                sb.append(a[i]);
            }
            System.out.println(sb.toString());
        }
    }
}
```
