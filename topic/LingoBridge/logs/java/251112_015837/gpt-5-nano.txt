import java.util.Scanner;

public class SwapTwoNumbers {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        int A = scanner.nextInt();
        int B = scanner.nextInt();

        scanner.close();

        // 使用 XOR 交換，僅使用 A、B 兩個變數，避免使用暫存變數
        A = A ^ B;
        B = A ^ B;
        A = A ^ B;

        // 輸出交換後的結果
        System.out.println(A);
        System.out.println(B);
    }
}

修改說明：
- 將作業改為符合題目需求：改為讀入兩個整數 A, B，並在交換後輸出，輸出順序與題目範例相符（先印出交換後的 A，再印出交換後的 B）。
- 使用只有 A、B 的 XOR 交換法，避免引入額外變數，遵循「不可使用其他變數進行數字交換」的限制，並避免整數溢位風險（相較於加減法在極端情況下可能溢位）。
- 程式結構與命名採用符合 Java 規範的慣例：類別名稱 SwapTwoNumbers、檔案名稱 SwapTwoNumbers.java、適當的註解與排版。
- 移除原始程式中的不相關邏輯與多餘變數，使得程式更加清楚、易於核對。

一句話改進建議：
如果希望提高可讀性，建議在交換過程配合清晰註解或改用具名方法（如使用暫存變數或位操作的清晰函式）以提升維護性與可讀性。
