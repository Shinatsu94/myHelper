import java.util.Scanner;

public class CSIE_41343223_1003_1 {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        // 使用 long 以擴大表示範圍，並用位元運算交換以避免溢位
        long A = scanner.nextLong();
        long B = scanner.nextLong();

        // XOR 交換，不使用額外變數
        A = A ^ B;
        B = A ^ B;
        A = A ^ B;

        System.out.println(A);
        System.out.println(B);

        scanner.close();
    }
}

修改說明：
- 將整數型別由 int 改為 long，提升對極值輸入的容許度，減少溢位風險。
- 使用位元交換（XOR swap）取代相加相減的交換方式，避免可能的溢位問題，同時仍不需要額外變數。
- 加入程式標頭註解，說明檔案名稱、功能、作者與日期，提升可讀性及維護性。
- 程式結尾加入 scanner.close()，符合資源管理的好習慣。
