修改後完整程式碼
// 檔案名稱: CSIE_Optimized_Exercise3.java
// 功能: 讀入兩個整數 A, B，先印出 A 與 B 的值，然後在不使用額外變數的情況下交換兩者，最後再印出交換後的值。
// 作者: 你的姓名
// 日期: 2025-11-11

import java.util.Scanner;

public class CSIE_Optimized_Exercise3 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int A = sc.nextInt();
        int B = sc.nextInt();
        sc.close();

        // 印出原始值
        System.out.println(A);
        System.out.println(B);

        // 使用位元運算 (XOR) 交換 A 與 B，且不使用額外變數
        A = A ^ B;
        B = A ^ B;
        A = A ^ B;

        // 印出交換後的值
        System.out.println(A);
        System.out.println(B);
    }
}

修改說明
- 以符合題意的輸入/輸出方式重構程式：先讀入兩個整數，依序印出原始值，接著利用不使用額外變數的 XOR 交換法交換兩個整數，最後印出交換後的值。
- 使用清晰命名與註解：變數名稱 A、B 符合題意，並在程式中添加註解，提升可閱讀性與維護性。
