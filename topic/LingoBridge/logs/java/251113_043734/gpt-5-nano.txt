/*
 * 檔案名稱: Main.java
 * 功能: 印出「虎科大」以及對應的內碼，逐字輸出每個字的 Unicode 內碼。
 * 作者: 自動評分系統
 * 日期: 2025-11-12
 * 說明: 直接輸出固定字串與其內碼，不透過使用者輸入。
 */
public class Main {
    public static void main(String[] args) {
        String[] chars = {"虎","科","大","林","武","杰"};
        for (String ch : chars) {
            int codePoint = ch.codePointAt(0);
            System.out.println(ch + " " + codePoint);
        }
    }
}

修改說明：
- 變更目的：題目要求固定輸出六個中文字及其對應的內碼，原程式透過使用者輸入分數並輸出評語，與題意嚴重不符，因此需改寫為直接輸出固定字與內碼。
- 主要改動：
  - 移除 Scanner、分數輸入與條件判斷，改用固定字元陣列来表示需要輸出的六個字。
  - 使用每個字元的 Unicode 內碼（透過 ch.codePointAt(0) 得到整數值）輸出，確保內碼的正確性且與字元一一對應。
  - 加入程式頭註解，說明檔案名稱、功能、作者與日期，提高可維護性。
- 為何這樣修改：直接符合作業需求與輸出格式，避免硬編碼數值，且能動態地取得正確的內碼，利於後續維護與擴充。
