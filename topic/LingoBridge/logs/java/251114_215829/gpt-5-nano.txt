```java
/*
File: CSIE_41343203_1031_3a.java
Description: 讀取使用者輸入的長度 n，接著讀入 n 個整數，將其由小到大排序，最後輸出排序結果（以空格分隔）。
Author: 教師
Date: 2025-11-14
*/

import java.util.Arrays;
import java.util.Scanner;

public class CSIE_41343203_1031_3a {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        if (!sc.hasNextInt()) {
            sc.close();
            return;
        }

        int n = sc.nextInt();
        if (n <= 0) {
            sc.close();
            return;
        }

        int[] arr = new int[n];
        for (int i = 0; i < n; i++) {
            if (sc.hasNextInt()) {
                arr[i] = sc.nextInt();
            } else {
                // 若輸入不足，預設為 0（可依需求改為丟出例外或終止）
                arr[i] = 0;
            }
        }

        // 將陣列從小到大排序
        Arrays.sort(arr);

        // 輸出排序結果，以空格分隔，末尾換行
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(arr[i]);
        }
        System.out.println(sb.toString());

        sc.close();
    }
}
```

說明修改部分的原因
- 導入 Arrays.sort 以簡單且高效地完成排序，確保正確性與可讀性。
- 以動態陣列存放輸入的整數，避免固定長度限制，符合「任意長度」的要求。
- 輸出格式改為單行、以空格分隔，符合題目輸出需求。
- 增加檔案標頭註解與清晰的變數命名，提升可維護性與可讀性。
