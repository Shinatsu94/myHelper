```java
/*
 * File: CSIE_51015130_1031_1.java
 * Title: 一維陣列排序（從小到大）
 * Description: 讀取整數長度 n，接著讀入 n 個整數，將它們排序為升序，輸出以空格分隔的序列，最後換行。
 * Author: 學生
 * Date: 2025-11-14
 */

import java.util.Scanner;
import java.util.Arrays;

public class CSIE_51015130_1031_1 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();

        // 邊界處理：若長度為 0 或負數，直接輸出空行並結束
        if (n <= 0) {
            System.out.println();
            sc.close();
            return;
        }

        int[] numbers = new int[n];
        for (int i = 0; i < n; i++) {
            numbers[i] = sc.nextInt();
        }

        Arrays.sort(numbers); // 進行升序排序

        // 使用 StringBuilder 避免尾端有空格，並加上結尾換行
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(numbers[i]);
        }
        System.out.println(sb.toString());

        sc.close();
    }
}
```

說明：
- 主要改動在輸出部分改用 StringBuilder，避免在輸出最後出現尾端空格，且最後以 println 輸出換行，符合常見測資的比對方式。
- 增加了對 n <= 0 的邊界處理，避免不合法輸入造成的異常行為。
- 加入檔案標頭註解，符合題目對程式標頭註解的要求。
