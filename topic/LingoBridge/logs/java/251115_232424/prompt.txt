題目：
【題目】
讀入使用者輸入的二個數字存在變數 A, B 並印出其值，將這二個變數的值交換並印出。
注意：程式除了建立 Scanner  物件讀取使用者輸入之外，不可使用其他變數進行數字交換。

【INPUT-1】
3
5

【OUTPUT-1】
5
3

【INPUT-2】
234
789

【OUTPUT-2】
789
234

學生作答程式：
import java.util.Scanner;

public class test {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        //System.out.print("請輸入兩個數字 A B: ");
        int A = sc.nextInt();
        int B = sc.nextInt();

        //System.out.println("交換前: A = " + A + ", B = " + B);

        // 不可使用額外變數 → 用加減法交換
        A = A + B;  
        B = A - B;  
        A = A - B;

        //System.out.println("交換後: A = " + A + ", B = " + B);
        System.out.println(A);
        System.out.println(B);

        sc.close();
    }
}
