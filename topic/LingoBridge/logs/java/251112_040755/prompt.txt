【題目】
讀入使用者輸入的二個數字存在變數 A, B 並印出其值，將這二個變數的值交換並印出。
注意：程式除了建立 Scanner  物件讀取使用者輸入之外，不可使用其他變數進行數字交換。

【INPUT-1】
3
5

【OUTPUT-1】
5
3

【INPUT-2】
234
789

【OUTPUT-2】
789
234

學生作答程式：
import java.util.Scanner;
public class CSIE_41343107_1003_3
{
	public static void main(String args[])
	{
	Scanner same=new Scanner(System.in);
	int a;
	int sum=0;
 	a = same.nextInt();
	for(int i=1;i<=100000;i++){
	int c=0,b=i;
	        while (b!=0) {
                c = c+(b % 10);
                b = b/10;
            }
	if(c==a) {sum=sum+1;}
	}
	System.out.println(sum);
	}
}
