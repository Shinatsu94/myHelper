修改與說明
- 作業要點理解與實作
  - 原 student 程式僅有一個無窮迴圈的分數判定範例，與題目「印出虎科大及學生中文姓名各字的內碼」完全不符，且缺少符合 Java main 方法簽名與輸出需求的實作。
  - 優化後的程式直接輸出虎科大與林武杰每個字及其對應的 Unicode 內碼，符合題意且不需使用外部輸入。

- 為何這樣改
  - 使用固定字串與固定輸出，避免不必要的輸入與控制流程，使程式的行為可預期、穩定，且容易測試。
  - 透過 (int) char 對每個字取得十進位的 Unicode 內碼，輸出格式與題目範例相符，且具有自動化對應，不需要手動填寫代碼。

- 修改的重點
  - main 方法簽名改為 public static void main(String[] args)。
  - 移除 Scanner 與不可預測的輸入流程，改以固定字串輸出。
  - 新增輔助方法 printChars(String s) 逐字印出字與對應的碼點，提升可維護性與可重用性。
  - 註解說明與整體排版，確保可讀性。

- 修改後完整程式碼
public class CSIE_41343111_0926_3 {
    public static void main(String[] args) {
        // 依題意：先輸出虎科大，再輸出林武杰，每個字與其Unicode內碼
        printChars("虎科大");
        printChars("林武杰");
    }

    private static void printChars(String s) {
        for (int i = 0; i < s.length(); i++) {
            char ch = s.charAt(i);
            int code = (int) ch;
            System.out.println(ch + " " + code);
        }
    }
}

說明修改部分的原因
- 將 main 方法簽名正規化，確保程式能在各種 Java 執行環境正確啟動。
- 以固定字串直接輸出符合題意，避免與題目無關的輸入處理與邏輯，提升正確性與可維護性。
- 使用輔助函式 printChars 以模組化處理字元與碼點對映，便於日後擴充或變更輸出內容。
