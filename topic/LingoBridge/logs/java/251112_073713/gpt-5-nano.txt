public class J_1011_01 {
    /*
     檔案名稱: J_1011_01.java
     功能: 使用李布尼茲級數近似 π，輸出前 100000 項的 pi 值與對應的 i，並找出使 pi ∈ [3.14159, 3.14160) 的最小 i。
     作者: 教師
     日期: 2025-11-11
     版本: 1.0
     說明: 程式包含適當的註解與易讀的區塊劃分。
     */
    public static void main(String[] args) {
        int i;
        double val;

        // 使用符號變數避免每次都判斷 i 的奇偶性，提高可讀性與效能
        val = 0.0;
        int sign = 1;
        for (i = 1; i <= 100000; i++) {
            val += sign * (1.0 / (2 * i - 1));
            sign = -sign;
            double pi = 4.0 * val;
            if (i % 10000 == 0) {
                System.out.printf("i = %6d, pi = %.10f%n", i, pi);
            }
        }

        System.out.println();

        // 找出使 pi 屬於 [3.14159, 3.14160) 的最小 i
        double acc = 0.0;
        int s = 1;
        int minI = 0;
        for (int t = 1; ; t++) {
            acc += s * (1.0 / (2 * t - 1));
            double currentPi = 4.0 * acc;
            if (currentPi >= 3.14159 && currentPi < 3.14160) {
                minI = t;
                break;
            }
            s = -s;
        }

        System.out.printf("i = %6d%n", minI);
    }
}

- 新增程式標頭與內部註解：提供檔案名稱、功能、作者、日期等資訊，提升可維護性與可讀性。
- 使用單一符號變數 sign 決定正負號，取代在迴圈內使用多個 if-else 的寫法，減少分支判斷，提升效能與可讀性。
- 將主要區塊分成兩個清晰段落：
  - 逐步計算前 100000 項的 pi 值並每 10000 顯示一次，符合題目輸出需求。
  - 單獨找出最小的 i 使 pi 落在 [3.14159, 3.14160) 的區間，並輸出該 i。
- 輸出格式保持原本要求，i 使用寬度 6 的欄位，pi 使用小數點後 10 位，且以換行分隔，確保與題目樣例相容。
