題目：
reverse digit order of an integer given by the user.
e.g. 123 --> 321, 2500 --> 52, 8531 --> 1358

題目要求:
- 程式僅執行一次就結束，無需使用EOF

【INPUT】
123

【OUTPUT】
321

【INPUT】
2500

【OUTPUT】
52

【INPUT】
8531

【OUTPUT】
1358


學生作答程式：
import java.util.Scanner;

public class CSIE_51015130_1017_3{
	public static void main(String[] args){
		Scanner sc= new Scanner(System.in);
		int s= sc.nextInt();
		int a[]={0,0,0,0}, b=0, sum;
		
		while(s!=0){
			b=0; sum=0;
			while(s!=0){
				a[b]=s%10;
				s=s/10;
				b++;
			}
		
			for(int i=0;i<b;i++){
				sum=sum*10+a[i];
			}

			System.out.println(sum); 
			s= sc.nextInt();
		}		
	}
}
