```java
/*
檔案名稱: PrintNameWithCodes.java
功能說明: 輸入使用者的中文姓名，先輸出「虎科大」三個字及其對應的 Unicode 內碼，
再逐字輸出使用者姓名的每一個字及其對應的 Unicode 內碼（內碼以十進位表示）。
作者: 論文/課程助教
日期: 2025-11-12
說明: 程式包含標頭註解與清晰的註解，變數命名有意義，格式整齊。
*/
import java.util.Scanner;

public class PrintNameWithCodes {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        // 提示使用者輸入中文姓名
        System.out.print("請輸入你的中文姓名（僅包含中文字符；可留空以結束程式）：");
        String name = scanner.nextLine();
        // 移除可能的空白字元，以避免影響輸出
        name = name == null ? "" : name.replaceAll("\\s+", "");

        // 先輸出「虎科大」3個字及其對應的內碼
        printCharWithCode('虎');
        printCharWithCode('科');
        printCharWithCode('大');

        // 再輸出學生的中文姓名每個字及其對應的內碼
        for (char ch : name.toCharArray()) {
            printCharWithCode(ch);
        }

        scanner.close();
    }

    // 協助方法：輸出單一字與其十進位內碼
    private static void printCharWithCode(char ch) {
        System.out.println(ch + " " + (int) ch);
    }
}
```

- 允許使用者輸入姓名：新增輸入機制，讓程式能動態輸出任意使用者的中文姓名。
