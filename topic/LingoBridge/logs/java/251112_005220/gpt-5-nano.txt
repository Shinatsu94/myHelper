/*
File: SwapTwoNumbers.java
Description: 讀入兩個整數 A, B，先輸出原值，接著在不使用其他變數的情況下交換兩數，並輸出交換後的值。
Author: 教師
Date: 2025-11-11
*/
public class SwapTwoNumbers {
    public static void main(String[] args) {
        java.util.Scanner s = new java.util.Scanner(System.in);
        int A = s.nextInt();
        int B = s.nextInt();

        // 先輸出原值
        System.out.println(A);
        System.out.println(B);

        // 使用 XOR 交換，僅用 A, B 兩個變數，且不額外宣告新變數
        A = A ^ B;
        B = A ^ B;
        A = A ^ B;

        // 輸出交換後的值
        System.out.println(A);
        System.out.println(B);

        s.close();
    }
}

修改原因說明：
- 移除了原作中多餘的變數（只有 A 與 B），且直接按照題意先輸出原值，再進行交換並輸出，符合題目限制。
- 使用 XOR 交換法在不使用額外變數的情況下完成交換，避免整數溢位風險（相較於 A = A + B / B = A - B 之類的做法）。
- 增加了程式頭註解與適當的程式內容註解，並確保類別名稱與檔案名稱對應，提升可閱讀性與維護性。

改進建議（一句話）：
建議同時提供更易讀的交換寫法說明與補充輸入驗證，讓學生了解不同交換方式的可讀性與邊界情況。
