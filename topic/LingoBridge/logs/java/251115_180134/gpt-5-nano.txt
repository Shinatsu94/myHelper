public class PiLeibniz {
    /*
     * 檔案名稱: PiLeibniz.java
     * 功能: 使用 Leibniz 公式近似圓周率，輸出 i=10000..100000 的 pi 值，並找出最小 i 使 pi ≥ 3.14159
     * 作者: 你可以填寫
     * 日期: 2025-11-15
     */
    public static void main(String[] args) {
        // Part (a): 輸出 i = 10000, 20000, ..., 100000 對應的 pi 值
        for (int i = 10000; i <= 100000; i += 10000) {
            double sum = 0.0;
            int sign = 1;
            for (int k = 1; k <= i; k++) {
                sum += 4.0 * sign / (2.0 * k - 1.0);
                sign = -sign;
            }
            // 輸出格式：i =  10000, pi = 3.1414926536
            System.out.printf("i =  %d, pi = %.10f%n", i, sum);
        }

        // 空一行，分隔兩部分
        System.out.println();

        // Part (b): 找出最小的 i 使得 pi >= 3.14159
        double target = 3.14159;
        double sum2 = 0.0;
        int sign2 = 1;
        int i2 = 0;
        while (sum2 < target) {
            i2++;
            sum2 += 4.0 * sign2 / (2.0 * i2 - 1.0);
            sign2 = -sign2;
        }
        System.out.printf("i = %d%n", i2);
    }
}

- 依照題意，(a) 部分輸出 10 行對應的 pi 值，格式嚴格符合「i =  10000, pi = 3.1414926536」的樣式。
- (b) 部分找出第一個使累積和(pi)≥3.14159 的 i，輸出格式為「i = 136121」。
