import java.util.Scanner;

public class CSIE_41343247_1003_2{
	public static void main(String[] args){
		Scanner scanner = new Scanner(System.in);

		// 讀取兩個整數
		int A = scanner.nextInt();
		int B = scanner.nextInt();

		// 不使用額外變數，利用算術交換法變更 A, B 的值
		A = A + B;
		B = A - B;
		A = A - B;

		// 輸出交換後的結果
		System.out.println(A);
		System.out.println(B);
	}
}

修改說明與原因：
- 移除原程式中的多餘變數與三值排序邏輯（max/min/mid），因題目只需兩個數字的交換，原邏輯與題意不符。
- 使用不新增額外變數的算術交換法，符合「除了 Scanner 之外不可使用其他變數進行數字交換」的要求；讓兩數直接在原變數中完成交換。

- 變數命名使用 A、B 以清楚表示第一與第二個輸入值，增進可讀性。
