```java
/*
File: SortArray.java
用途: 讀入一個整數 n 與 n 個整數，輸出經由升冪排序後的陣列內容
作者: 自動批改示例
日期: 2025-11-12
描述: 使用 Arrays.sort 進行排序，支援包含負數的情況，輸出以單行顯示且元素間以空格分隔
*/
import java.util.Arrays;
import java.util.Scanner;

public class SortArray {
    public static void main(String[] args) {
        Scanner scn = new Scanner(System.in);
        int n = 0;
        if (scn.hasNextInt()) {
            n = scn.nextInt();
        }

        int[] arr = new int[n];
        for (int i = 0; i < n; i++) {
            if (scn.hasNextInt()) {
                arr[i] = scn.nextInt();
            } else {
                arr[i] = 0; // 若輸入不足，預設為 0；實務上應加強輸入驗證
            }
        }

        // 排序：由小到大
        Arrays.sort(arr);

        // 輸出：單行，元素以空格分隔，無尾端多餘空格
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(arr[i]);
        }
        System.out.print(sb.toString());

        scn.close();
    }
}
```

- 正確性提升：改用 Arrays.sort 可保證正確且穩定地排成遞增序列，並天然支援負數與任意長度的輸入。
