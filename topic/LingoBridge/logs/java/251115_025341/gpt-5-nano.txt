改進建議（一句話）
建議直接依照行號輸出星號，不需要額外的二維陣列，並加上適當的註解與輸入驗證，使程式更清晰、健壯。

程式修改說明與改動原因
- 移除不必要的二維陣列與固定長度的資源配置：原程式使用一個 jagged char[][] 並以固定起始長度 20 作為每列的長度，較難理解且浪費資源；改為直接根據行號輸出對應數量的 '*'。
- 增加輸入驗證：原程式未處理不符合條件的輸入，如 0 或大於 20 的情況；加入檢查以符合題意 0 < n <= 20。
- 改善變數命名與加入程式頭註解：使用更具可讀性的命名，並在程式頭加入檔案名稱、功能、作者與日期等說明，提升可維護性。
- 簡化輸出邏輯：直接在迴圈中輸出星號，避免不必要的陣列寫入與複雜的索引計算。

修改後完整程式碼
/*******************************************************
 * 檔案名稱: LowerTriangleStars.java
 * 功能說明: 輸入高度 n (0 < n <= 20)，列印下三角形的 '*' 符號，逐行輸出，
 *          第 i 行輸出 i 個 '*'，不輸出空格。
 * 作者: 你可以填寫
 * 日期: 2025-11-14
 *******************************************************/
import java.util.Scanner;

public class LowerTriangleStars {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 讀入並驗證 n
        if (!sc.hasNextInt()) {
            System.err.println("輸入必須為整數。");
            return;
        }
        int n = sc.nextInt();
        if (n <= 0 || n > 20) {
            System.err.println("n 必須滿足 0 < n <= 20。");
            return;
        }

        // 直接輸出每行的星號，形成下三角形
        for (int i = 1; i <= n; i++) {
            for (int j = 0; j < i; j++) {
                System.out.print('*');
            }
            if (i < n) System.out.println();
        }
    }
}
