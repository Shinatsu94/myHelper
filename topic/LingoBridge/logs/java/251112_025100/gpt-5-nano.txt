/*
檔案名稱: CSIE_41343119_1003_1.java
功能: 讀入兩個整數 A、B，先印出 A、B 的原始值，之後不新增變數地交換其值並印出交換後的結果
作者: 金宗翰
日期: 2025/10/03
*/
import java.util.Scanner;

public class CSIE_41343119_1003_1 {
    public static void main(String[] args) {
        Scanner scn = new Scanner(System.in);
        int a = scn.nextInt();
        int b = scn.nextInt();

        // 印出原始值
        System.out.println(a);
        System.out.println(b);

        // 不新增變數，使用加法-減法交換法
        a = a + b;
        b = a - b;
        a = a - b;

        // 印出交換後的值
        System.out.println(a);
        System.out.println(b);

        scn.close();
    }
}

修改說明
- 主要改動是加入原始值的輸出，以符合題意需求；同時將標頭註解移到檔案頂部、加入資源釋放與改善整體格式，提升可讀性與可維護性。若要避免整數溢位風險，亦可改用 XOR 交換法（A ^= B; B ^= A; A ^= B;）而不需額外變數。
