題目：
【題目】
請撰寫一個 Java 程式，建立一個下三角形的二維陣列。

陣列的高度 n 由使用者輸入 (0 < n <= 20)。
依序從上到下、從左到右填入'*'符號進入陣列。
建立並列印完整的下三角形二維陣列。

備註: 元素之間無需任何空白符號，避免未通過測資。

【input】
10

【output】
*
**
***
****
*****
******
*******
********
*********
**********


學生作答程式：
import java.util.Scanner;

public class csie_1031_41343214_3a {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int n;

       
        do {

            n = sc.nextInt();
        } while (n <= 0 || n > 20);

     
        char[][] tri = new char[n][n];

      
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                if (j <= i) {
                    tri[i][j] = '*';
                } else {
                    tri[i][j] = ' ';
                }
            }
        }

   

        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                System.out.print(tri[i][j]);
            }
            System.out.println();
        }

        sc.close();
    }
}
