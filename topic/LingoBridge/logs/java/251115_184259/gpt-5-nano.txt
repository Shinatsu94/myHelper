- 將判斷條件改為僅判斷相鄰兩位是否嚴格遞增（s[i] < s[i+1]），避免把遞減也當作符合的錯誤邏輯。
- 使用字串處理以支援任意長度的數字，避免固定為三位數的限制，並提升健壯性。
- 增加清晰的程式標頭註解與具描述性的變數名稱，提升可讀性與維護性。
- 輸出格式保持題目要求的語句，方便比對。

修改後完整程式碼
/*
 File: WellOrderedNumber.java
 功能: 判斷輸入的十進位數字其各位數是否嚴格遞增（從左到右）
 作者: (你的名字)
 日期: 2025-11-15
 說明: 程式僅執行一次，輸入可為任意長度的正整數，若各位數字嚴格遞增則輸出
      "<input> is well-ordered number."，否則輸出
      "<input> is not well-ordered number."
*/
import java.util.Scanner;

public class WellOrderedNumber {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String input = sc.next(); // 讀取輸入字串
        boolean isWellOrdered = true;

        // 判斷是否為純數字與每兩鄰位是否嚴格遞增
        for (int i = 0; i < input.length() - 1; i++) {
            char c1 = input.charAt(i);
            char c2 = input.charAt(i + 1);

            // 確保都是數字
            if (!Character.isDigit(c1) || !Character.isDigit(c2)) {
                isWellOrdered = false;
                break;
            }

            // 檢查相鄰兩位是否嚴格遞增
            if (c1 >= c2) {
                isWellOrdered = false;
                break;
            }
        }

        if (isWellOrdered) {
            System.out.println(input + " is well-ordered number.");
        } else {
            System.out.println(input + " is not well-ordered number.");
        }
    }
}
