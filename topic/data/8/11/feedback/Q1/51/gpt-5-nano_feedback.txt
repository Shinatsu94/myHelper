```java
import java.util.Scanner;

public class SwapTwoNumbers {
    public static void main(String[] args) {
        try (Scanner scan = new Scanner(System.in)) {
            int A = scan.nextInt();
            int B = scan.nextInt();

            // 兩數值使用不額外變數的 XOR 交換法
            A = A ^ B;
            B = A ^ B;
            A = A ^ B;

            System.out.println(A);
            System.out.println(B);
        }
    }
}
```

- 將變數名稱改為 A、B：讓變數名稱更符合題目敘述與常見慣例，提升可讀性。

- 使用 try-with-resources 包裝 Scanner：自動關閉資源，避免忘記手動關閉造成的資源洩漏。
- 保留原有的 XOR 交換法，符合「不可使用其他變數進行數字交換」的限制，同時以註解說明交換步驟。
