```java
import java.util.Scanner;

public class SwapTwoNumbers {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 讀取兩個輸入數字，符合題意
        int A = sc.nextInt();
        int B = sc.nextInt();

        // 印出原始值
        System.out.println(A);
        System.out.println(B);

        // 不使用額外變數進行交換（加法法）
        A = A + B;
        B = A - B;
        A = A - B;

        // 印出交換後的結果，順序依「先印出交換後的兩個值」之需求
        System.out.println(A);
        System.out.println(B);

        sc.close();
    }
}
```

修改部分說明
- 移除多餘的第三個變數 c，符合輸入需求與題意。
- 將變數名稱改為 A、B，並加入註解，提升可讀性。
- 使用不額外變數的加法交換法，符合「不可使用其他變數進行數字交換」的限制，同時保留簡單直觀的實作；也可替換為 XOR 交換以避免整數溢位風險。
- 加入程式關閉 (sc.close())，提升資源管理。
