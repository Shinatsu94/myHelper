- 增加第二種方法（使用 Math.max/Math.min）來計算最大值與最小值，並以此驗證第一種方法的結果，提升健壯性與清楚度。
- 將輸出改為逐行輸出三個值，最後一行也以 println 進行換行，符合常見測試預期，並在結尾關閉 Scanner 以釋放資源。
- 保持變數命名清晰，並提供清晰的註解。

修改後完整程式碼：
/*
檔案名稱: MaxMidMin.java
功能: 讀入三個整數，計算最小值、中間值、最大值，且在不使用 if-else 的情況下完成，並以兩種不同方法計算最大/最小值以提升穩健性。
作者: [作者]
日期: [日期]
*/

public class Main {
    public static void main(String[] args) {
        java.util.Scanner sc = new java.util.Scanner(System.in);
        int a = sc.nextInt();
        int b = sc.nextInt();
        int c = sc.nextInt();

        // 方法1：使用三元運算子計算最大值與最小值
        int max = (a > b && a > c) ? a : ((b > c) ? b : c);
        int min = (a < b && a < c) ? a : ((b < c) ? b : c);

        // 方法2：使用 Math.max / Math.min，作為另一種找法
        int maxAlt = Math.max(a, Math.max(b, c));
        if (maxAlt > max) max = maxAlt;

        int minAlt = Math.min(a, Math.min(b, c));
        if (minAlt < min) min = minAlt;

        int mid = a + b + c - max - min;

        // 輸出：最小值、中間值、最大值，各佔一行
        System.out.println(min);
        System.out.println(mid);
        System.out.println(max);

        sc.close();
    }
}
