- 修改內容與原因

  - 輸出改為符合題目樣例：先輸出交換後的兩個值（A 與 B），分別印在兩行，以符合樣例輸出需求。
  - 增加完整標頭註解與必要的註解，提升可讀性與可維護性。
  - 為了與一般評測保持一致，使用主程式類名 Main，確保檔案命名與類別對應。
- 修改後程式碼（完整，已含標頭與註解）
/*
檔案名稱: Main.java
功能: 讀入兩個整數 A, B，交換其值並印出交換後的結果
作者: 授課教授
日期: 2025-11-11
說明: 不使用額外變數進行交換，採用 XOR 交換法，輸出順序符合題目樣例。
*/
public class Main {
    public static void main(String[] args) {
        java.util.Scanner sc = new java.util.Scanner(System.in);
        int A = sc.nextInt();
        int B = sc.nextInt();

        // 使用不引入額外變數的交換方法：XOR 交換
        A = A ^ B;
        B = A ^ B;
        A = A ^ B;

        System.out.println(A);
        System.out.println(B);

        sc.close();
    }
}
