檔案名稱: CSIE_Optimized_Swap.java
功能: 讀取使用者輸入的兩個整數 A 與 B，透過不使用額外變數的方式交換兩值，並輸出交換後的結果（先輸出原本的第二個值，再輸出原本的第一個值）。
作者: 你
日期: 2025-11-11
*/
import java.util.Scanner;

public class CSIE_Optimized_Swap {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int A = sc.nextInt();
        int B = sc.nextInt();

        // 以 XOR 交換法，僅使用 A、B 兩個變數，避免使用額外變數
        A = A ^ B;
        B = A ^ B;
        A = A ^ B;

        // 輸出交換後的結果：先印出原本的 B，後印出原本的 A
        System.out.println(B);
        System.out.println(A);

        sc.close();
    }
}
