/**
 * 文件名稱: CSIE_41343223_1003_2.java
 * 功能說明: 讀入兩個整數 A, B，交換後輸出新順序的兩個值（不使用其他變數進行交換）。
 * 作者: 教師姓名
 * 日期: 2025-11-11
 */

import java.util.Scanner;

public class CSIE_41343223_1003_2 {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        // 讀取兩個整數
        int A = scanner.nextInt();
        int B = scanner.nextInt();

        // 使用位元運算交換兩個整數，且不使用額外變數
        A = A ^ B;
        B = A ^ B;
        A = A ^ B;

        // 輸出交換後的結果（符合題目樣本：先輸出原本的 B，即現在的 A）
        System.out.println(A);
        System.out.println(B);

        scanner.close();
    }
}

修改部分原因說明：
- 移除了不必要的第三個讀入變數 C，因題目只需兩個數字，因此移除相關運算以避免歧義並提升正確性。
- 以 XOR 交換法取代原程序中的多重變數運算，符合「不可用額外變數」的要求，同時避免整數溢位問題（相較於加減法）。
- 程式輸出改為交換後的值，以符合題目提供的輸出樣本。
- 加入完整檔案標頭與適當註解，提升可讀性與維護性。
