修改後程式碼（SwapTwoNumbers.java）
import java.util.Scanner;

/*
Filename: SwapTwoNumbers.java
功能: 讀取兩個整數，交換後輸出交換後的值（先輸出交換後的第一個，再輸出第二個）。
作者: 教師
日期: 2025-11-11
說明: 不使用第三個變數進行交換，使用 XOR 交換法以避免暫存變數。
*/
public class SwapTwoNumbers {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        long a = sc.nextLong();
        long b = sc.nextLong();

        // 使用 XOR 交換法，無需第三個變數
        a = a ^ b;
        b = a ^ b;
        a = a ^ b;

        // 輸出交換後的值：先輸出 a (原本的 B)，再輸出 b (原本的 A)
        System.out.println(a);
        System.out.println(b);

        sc.close();
    }
}

修改說明
- 這版把原先的三個輸入改為兩個，並且用不需要額外變數的交換法完成需求，輸出順序與題意相符（交換後的值，兩條分行）。
- 加入程式標頭註解，提升可讀性與維護性；變數命名改為具意義的 a 與 b，且使用 long 類型提高數字範圍的穩健性。

改進建議或反饋（一句話）
建議在實作時嚴格依題意輸出並採用不需要額外暫存變數的交換方法（如本次的 XOR 交換），以確保行為符合題目要求且提升程式穩健性與可讀性。
