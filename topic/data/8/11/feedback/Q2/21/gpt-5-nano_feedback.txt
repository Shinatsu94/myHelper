修改後程式碼（檔案名稱建議：CSIE_41343119_1003_2.java）：
/*
檔案名稱: CSIE_41343119_1003_2.java
功能: 讀入兩個整數，交換其值，並印出交換後的結果（每行一個數字）
作者: 金宗翰
日期: 2025/10/03
*/
import java.util.Scanner;

public class CSIE_41343119_1003_2 {
    public static void main(String[] args) {
        Scanner scn = new Scanner(System.in);

        int a = scn.nextInt();
        int b = scn.nextInt();

        // 使用兩個變數完成交換，不新增額外變數
        a = a + b;
        b = a - b;
        a = a - b;

        System.out.println(a);
        System.out.println(b);

        scn.close();
    }
}

- 更符合題意：直接交換兩個數字並輸出結果，且輸出符合範例（先輸出交換後的第一個數，再輸出第二個數）。
- 不使用額外變數進行交換：採用算術交換法，僅用 a、b 兩個變數完成交換，符合題目的限制。
- 程式可讀性提升：加入標準化的檔案註解與簡短註解，命名與排版更清晰，方便後續維護。
