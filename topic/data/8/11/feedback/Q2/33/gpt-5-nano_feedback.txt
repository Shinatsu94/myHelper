/*

import java.util.Scanner;

public class CSIE_41343208_1003_2 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int A = sc.nextInt();
        int B = sc.nextInt();

        // 不使用臨時變數，透過加減法在原地交換 A 與 B 的值
        A = A + B;
        B = A - B;
        A = A - B;

        System.out.println(A);
        System.out.println(B);

        sc.close();
    }
}

修改說明與原因：
- 移除原程式中的不必要第三個輸入 C，改為只讀取兩個整數 A、B，符合題目要求。
- 以不使用臨時變數的方式完成交換：A = A + B; B = A - B; A = A - B，達成題目限制。
- 交換完成後才印出兩個值，並且輸出順序與樣例相符（交換後的值）。
- 增加程式開頭註解與說明，並在結尾加入 sc.close()，提升可讀性和資源釋放。

改進建議或反饋（一句話）：
若輸入數值可能很大而造成溢位，建議改用位元運算 XOR 的交換法（A = A ^ B; B = A ^ B; A = A ^ B）以避免整數溢位風險，且仍不使用臨時變數。
