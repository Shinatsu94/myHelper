/*
  檔案名稱：CSIE_41343121_1003_2.java
  功能說明：讀入兩個整數，輸出交換後的兩個數字；實作不使用額外變數直接交換（使用 XOR 交換法以避免整數溢位風險）。
  作者：CSIE_41343121_1003_2
  日期：改寫版本
*/
import java.util.Scanner;

public class CSIE_41343121_1003_2 {
  public static void main(String[] args) {
    Scanner scn = new Scanner(System.in);
    int a = scn.nextInt();
    int b = scn.nextInt();

    // 不使用額外變數的交換方式：XOR 交換
    a = a ^ b;
    b = a ^ b;
    a = a ^ b;

    System.out.println(a);
    System.out.println(b);
  }
}

修改說明：
- 將原本的三個臨時變數和排序邏輯移除，改以符合題意：先輸入兩個數字，再交換並輸出。為避免加減法交換可能造成的整數溢位，改用位元異或（XOR）交換法，且不使用額外變數，符合題目要求的「不使用其他變數進行數字交換」。
- 增加程式標頭註解，說明檔案名稱、功能、作者、日期，並保留適當的內部註解以提升可讀性。
- 保留 Scanner 作為輸入來源，符合題目限制。

改進建議（一句話）：
- 若對可讀性有更高要求且輸入範圍已知，考慮使用更直覺的交換方式並加上輸入範圍檢查，或在極端情況下採用可靠的交換實作以避免誤解。
