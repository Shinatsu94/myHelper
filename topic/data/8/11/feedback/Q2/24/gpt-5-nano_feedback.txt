import java.util.Scanner;

public class SwapTwoNumbers {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int a = sc.nextInt();
        int b = sc.nextInt();

        // 使用不另行宣告新變數的交換方式：a = a + b; b = a - b; a = a - b
        // 注意：此方法可能在非常大的整數下發生溢位，若有溢位風險可改用暫存同步的方法，但題目要求不得使用額外變數。
        a = a + b;
        b = a - b;
        a = a - b;

        System.out.println(a);
        System.out.println(b);

        sc.close();
    }
}

說明修改原因：
- 將原程式中的多餘變數 c、max/min 等邏輯移除，改以簡潔的交換方法直接滿足「兩數交換並輸出」的需求，符合正確性與邏輯評分重點。
- 使用只有兩個變數 a、b 進行交換，符合題目「不可使用其他變數進行數字交換」的限制。
- 加入符合題意的輸出（交換後的兩個值各佔一行），並以容易閱讀的變數名稱命名，提升程式可讀性。
