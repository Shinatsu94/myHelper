```java
/**
 * File: CSIE_41343205_1003_3.java
 * Description: Count numbers from 1 to 100000 whose digits sum equals A
 * Author: (Your Name)
 * Date: 2025-11-15
 */

import java.util.Scanner;

public class CSIE_41343205_1003_3 {

    // Return the sum of digits of n
    private static int digitSum(int n) {
        int sum = 0;
        while (n > 0) {
            sum += n % 10;
            n /= 10;
        }
        return sum;
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int a = sc.nextInt();

        int count = 0;

        // 遍歷範圍從 1 到 100000（含），符合條件的數字累計
        for (int i = 1; i <= 100000; i++) {
            if (digitSum(i) == a) {
                count++;
            }
        }

        System.out.println(count);
        sc.close();
    }
}
```

說明修改部分的原因
- 對範圍進行修正：為符合題目「1 到 100000 之間」，避免少算 100000 的情況，確保一般輸入都能得到正確結果。
- 提升可讀性與維護性：將方法命名改為 digitSum，避免與局部變數 sum 同名造成混淆；新增清晰的程式標頭註解與必要內部註解。
- 簡化與穩定性：維持原有簡單迴圈與數字位元分解邏輯，僅提升結構與可讀性，不影響效率。
