import java.util.Scanner;
public class SwapTwoNumbers {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int A = scanner.nextInt();
        int B = scanner.nextInt();

        // 印出原始值
        System.out.println(A);
        System.out.println(B);

        // 不使用額外變數，直接交換 A 與 B
        A = A + B;
        B = A - B;
        A = A - B;

        // 印出交換後的值
        System.out.println(A);
        System.out.println(B);

        scanner.close();
    }
}

修改說明：
- 針對題目需求，將輸入改為讀取兩個整數並分別印出其原值，接著在不使用其他變數的限制下完成 A、B 的交換，最後再印出交換後的值。
- 交換實作採用不需要第三個變數的常見做法：A = A + B；B = A - B；A = A - B；此法僅使用 A、B 兩個變數，符合題意的限制。
- 如考量溢位風險，亦可改為使用 XOR 交換法或改用較大型別（如 long）以避免溢位，並適當加上註解與輸出格式說明。
