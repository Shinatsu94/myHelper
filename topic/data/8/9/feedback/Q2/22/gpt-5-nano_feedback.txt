public class CSIE_41343121_0926_2 {

    public static void main(String[] args) {
        printWithCode("虎科大");
        printWithCode("林武杰");
    }

    // 輪廓：逐字輸出字元及其內碼（Unicode碼點， BMP 範圍的字元直接轉為 int 即可顯示碼點）
    private static void printWithCode(String s) {
        for (int i = 0; i < s.length(); i++) {
            char ch = s.charAt(i);
            System.out.println(ch + " " + (int) ch);
        }
    }
}

修改說明與原因
- 原程式僅顯示進度條，與題目需求「印出虎科大與學生姓名的每個字及對應內碼」不符，故改用直接輸出每個字及其 Unicode 碼點的方式，確保正確性與可讀性。
- 將輸出邏輯抽象為 printWithCode 方法，提升重用性與可維護性；主入口只需呼叫兩次即可打印所需字串及對應碼點。
- 使用十進位的 Unicode 碼點輸出，對於 BMP 字元（如中文漢字）直接等同於其碼點，避免混淆，同時保留結構清晰與易於驗證的輸出格式。
