// 檔案名稱: CSIE_41343219_0926_2.java
// 功能: 印出「虎科大」及學生中文姓名對應的內碼（以 Unicode 代碼點表示）
// 作者: 學生
// 日期: 2025-11-12
public class CSIE_41343219_0926_2 {
    public static void main(String[] args) {
        // 依序印出虎 科 大 林 武 杰，並顯示其 Unicode 代碼點（int）
        char[] chars = {'虎','科','大','林','武','杰'};
        for (char ch : chars) {
            System.out.println(ch + " " + (int) ch);
        }
    }
}

修改說明與原因：
- 移除原本作業中不相關的動畫與佔用 CPU 資源的迴圈與睡眠邏輯，直接輸出所需結果，符合「正確性」與「邏輯性」的要求。

- 以 Unicode 代碼點 (int) 的方式輸出每個字元的內碼，避免與特定編碼（如 Big5、GBK 等）綁定，提升可移植性與一致性，同時滿足「對應的內碼」的需求。
