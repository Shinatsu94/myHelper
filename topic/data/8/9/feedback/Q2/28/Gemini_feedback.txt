**3. 改進的程式碼**

```java
public class CSIE_41343143_0926_2 {
    public static void main(String[] args) throws Exception {
        char[] spinner = {'|', '/', '-', '\\'};
        int total = 100;
        int i = 0; // 初始化 i，用于追踪当前进度
        while (true) { // 使用无限循环，实现持续旋转

            char spinChar = spinner[i % spinner.length]; //根据进度取得不同的旋轉樣態
            System.out.print("\r" + spinChar + " " + (i % (total+1)) + "%"); //取i除以(total+1)的餘數來達成迴圈進度

            Thread.sleep(100);
            i++; // 增加进度
            
            //讓 i 持續增加，但是當 i 超過 total 時，將 i 設為 0，讓進度從 0 開始
            if(i > total){
                i = 0;
            }
        }

        //System.out.println("\n載入完成！"); // 移除載入完成，因為不需要跑完就結束
    }
}
```

**4. 修改說明**

*   **使用`while(true)`迴圈:**  將原本的`for`迴圈改為`while(true)`迴圈，實現程式持續運行，達成持續旋轉的效果。
*   **增加`i`的初始化:** 原本的程式碼中`i`只在`for`迴圈中存在，修改後的程式碼在迴圈外新增`int i = 0;`，用於追蹤進度。
*   **使用`i % (total+1)`來顯示進度:** 由於需要持續旋轉，每次旋轉後進度都會增加，當超過100時，需要將其歸零，因此使用取餘數的方式來達成迴圈進度的效果。
*   **新增`if`判斷式:**  讓 i 持續增加，但是當 i 超過 total 時，將 i 設為 0，讓進度從 0 開始。
*   **移除`System.out.println("\n載入完成！");`:** 因為題目要求是持續旋轉，因此不需要跑完就結束。
