/*
檔案名稱: TigerUnicodePrinter.java
功能: 印出「虎科大」與對應的內碼，依序印出每個字元及其 Unicode 內碼
作者: 教授
日期: 2025-11-12
*/
public class TigerUnicodePrinter {
  public static void main(String[] args) {
    // 順序按照題目要求：虎、科、大、林、武、杰
    char[] chars = {'虎','科','大','林','武','杰'};
    for (char ch : chars) {
      System.out.println(ch + " " + (int) ch);
    }
  }
}

修改部分說明與原因：
- 以直接輸出字元及其 Unicode 內碼的方式取代原程式中無關的字元與無目的的迴圈、sleep 與複雜輸出，符合「印出字元與對應內碼」的需求。
- 使用有意義的變數名稱與簡單明瞭的迴圈結構（chars 與 for-each），提升可讀性與維護性，符合格式與邏輯要求。
- 加上程式檔案標頭註解，說明檔案名稱、功能、作者與日期，符合備註評分標準。
- 輸出格式固定為每行一個字元及其對應的 Unicode 內碼，避免使用控制字元或不穩定的輸出格式。

改進建議或反饋（一句話）：
建議以固定字元陣列配合迴圈輸出 Unicode 內碼，避免不必要的複雜字元操作與延遲，直接滿足題意並提升可讀性與穩定性。
