```java
public class SpinnerAnimation {
    private static final char[] SPINNER = {'|', '/', '-', '\\'};
    private static volatile int progress = 0;
    private static volatile boolean running = true;

    public static void main(String[] args) {
        // 顯示程式功能
        System.out.println("此程式顯示一個旋轉的棒子，並在旁邊顯示進度百分比。");

        Thread spinner = new Thread(() -> {
            int i = 0;
            while (running) {
                System.out.printf("\r進度: %c %d%%", SPINNER[i++ % SPINNER.length], progress);
                try { Thread.sleep(100); } catch (InterruptedException e) { Thread.currentThread().interrupt(); break; } // 處理中斷異常
            }
        });

        Thread progressThread = new Thread(() -> {
            while (progress < 100) {
                try {
                    Thread.sleep(200);
                    // 調整進度更新幅度，使其更平滑
                    progress += 2; // 每次增加2，可以調整這個值
                    if (progress > 100) {
                        progress = 100; // 確保不超過100%
                    }
                } catch (InterruptedException e) { Thread.currentThread().interrupt(); break; } // 處理中斷異常
            }
            running = false;
            System.out.printf("\rCompleted! 100%%\n");
        });

        spinner.start();
        progressThread.start();
        try { spinner.join(); progressThread.join(); } catch (InterruptedException e) { Thread.currentThread().interrupt();}
    }
}
```

**4. 修改說明**

*   **加入程式功能說明:** 在main函式一開始，印出程式的目的，增加程式易讀性
*   **進度更新調整：**
    *   `progress += 2;`  將每次增加的進度改為 2，放慢進度條的速度，可以根據需求調整此值。
    *   `if (progress > 100) { progress = 100; }`  確保進度不會超過 100%。
*   **處理InterruptedException:**
    *   在 `catch` 塊中添加了 `Thread.currentThread().interrupt();`，當線程被中斷時，正確地設置中斷標誌，以便在必要時可以響應中斷。
