**3. 改進的程式碼**

```java
public class Main {
    public static void main(String[] args) throws Exception {
        String[] rotation = { "|", "/", "-", "\\" };
        int progress = 0; // 從0%開始
        int rotationIndex = 0; // 紀錄目前rotation的index
        while (true) { // 使用true，並用break跳出迴圈
            System.out.print("\r" + rotation[rotationIndex] + " 進度: " + progress + "%");
            Thread.sleep(100);
            progress += 10;
            rotationIndex = (rotationIndex + 1) % rotation.length; // 確保rotationIndex在0到3之間循環

            if (progress > 100) {
                progress = 100;
                System.out.print("\r" + rotation[rotationIndex] + " 進度: " + progress + "%"); // 確保最後一次輸出為100%
                break; // 跳出迴圈
            }
        }
        System.out.println("\n進度已達 100%，完成！");
    }
}
```

**4. 修改說明**

*   **進度初始化:** 將 `progress` 初始化為 0，確保從 0% 開始。
*   **進度迴圈的邏輯修改:**
    *   將`while (progress <= 100)`改為`while (true)`，讓迴圈可以持續執行。
    *   使用`rotationIndex`紀錄目前rotation的index，輪到下一個迴圈時，`rotationIndex + 1`，確保rotation的index可以正確的更新。
    *   當進度超過100%的時候，將進度設定為100%，並`break;`跳出迴圈。
*   **最後一次輸出:** 在進度達到 100% 前，確保輸出正確的進度百分比。
*   **移除多餘的判斷:** 移除了原本 `for` 迴圈，簡化了程式邏輯。
