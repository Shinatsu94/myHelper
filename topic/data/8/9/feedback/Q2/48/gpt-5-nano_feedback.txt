 * 說明: 直接輸出每個中文字及其 Unicode 內碼，符合題意需求，並移除原程式中與題意無關的 Spinner 顯示與嘗試。
 */
public class CSIE_41343227_0926_2 {
    public static void main(String[] args) {
        // 目標字元順序：虎 科 大 林 武 杰
        String target = "虎科大林武杰";

        // 逐字輸出該字的內碼（Unicode code point）
        for (int i = 0; i < target.length();) {
            int codePoint = target.codePointAt(i);
            String ch = new String(Character.toChars(codePoint));
            System.out.println(ch + " " + codePoint);
            i += Character.charCount(codePoint);
        }
    }
}
