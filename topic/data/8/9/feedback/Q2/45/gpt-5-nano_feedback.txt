修改後完整程式碼（保持原類別名 RoBar，便於評分與整合）：
/**
 * RoBar.java
 * 功能：輸出「虎科大」及學生中文姓名對應的內碼（Unicode 十進位表示）
 * 作者：資訊工程系教授
 * 日期：2025-11-12
 */
public class RoBar {
    public static void main(String[] args) {
        // 要輸出的六個中文字及其對應的 Unicode 十進位內碼
        char[] chars = {'虎','科','大','林','武','杰'};
        for (char ch : chars) {
            System.out.println(ch + " " + (int) ch);
        }
    }
}

修改部分說明
- 將原本的跑動畫面撤除，改為直接輸出六個字及其對應的內碼，符合題目需求的正確輸出格式。
- 使用 Unicode 字元的十進位碼點顯示內碼，透過 (int) ch 即可取得該字元的碼點值，確保輸出的一致性與可預期性。
- 程式結構簡潔，採用有意義的變數名稱與單一責任迴圈，易於閱讀與維護；新增程式檔案註解與標頭，以符合備註評分標準。
