**3. 改進的程式碼**

```java
public class RoBar {
    public static void main(String[] args) throws InterruptedException {

        char[] spin = {'|', '/', '-', '\\'};// 顯示的字元有哪幾種
        int i = 0;
        long startTime = System.currentTimeMillis(); // 紀錄開始時間
        long duration = 10_000; // 運行時間設定為10秒 (毫秒)
        long endTime = startTime + duration;

        while (System.currentTimeMillis() < endTime) {
            // 計算已過時間的百分比
            double progress = (double) (System.currentTimeMillis() - startTime) / duration;
            int percentage = (int) (progress * 100);

            System.out.print("\r" + spin[i % spin.length] + " " + percentage + "%"); // 顯示運行動畫跟時間進度
            System.out.flush();
            Thread.sleep(100);// 暫停0.1秒
            i++;
        }
        System.out.println();//結束後換行
    }
}
```

**4. 修改說明**

*   **新增變數`startTime`和`duration`**:startTime紀錄迴圈開始執行的時間，duration紀錄迴圈持續執行的時間長度。
*   **百分比計算:**
    *   使用 `double progress = (double) (System.currentTimeMillis() - startTime) / duration;` 計算已過時間佔總時間的比例。
    *   使用 `int percentage = (int) (progress * 100);` 將比例轉換為百分比（整數）。
    *   修改後，百分比顯示更加精確。
*   **將結束換行移到迴圈外面**:程式碼更加簡潔。
