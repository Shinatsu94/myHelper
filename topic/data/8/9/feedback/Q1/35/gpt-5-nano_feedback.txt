修改後完整程式碼（TigerKeDaPrinter.java）
/*
 * 檔案名稱: TigerKeDaPrinter.java
 * 功能說明: 逐字印出「虎科大」及學生姓名「林武杰」的每個字與對應的 Unicode 編碼（十進位）。
 * 作者: 教師
 * 日期: 2025-11-12
 */
public class TigerKeDaPrinter {
    public static void main(String[] args) {
        // 依需求順序：先印虎、科、大，再印林、武、杰
        char[] chars = {'虎', '科', '大', '林', '武', '杰'};

        // 逐字印出字元與其對應的 Unicode 編碼（十進位）
        for (char ch : chars) {
            System.out.println(ch + " " + (int) ch);
        }
    }
}

修改原因與說明
- 將輸出改為逐字列印：依題意，每一行輸出一個中文字與其對應的內碼，故採用逐字陣列與迴圈輸出，確保長度變動時仍可正確輸出。
- 使用 Unicode 編碼（十進位）：直接輸出 (int) ch 即可取得該字的 Unicode 編碼，保證在不同系統與編碼設定下的一致性，且避免手動硬編碼可能造成的錯誤。
- 改善命名與結構：將類別命名改為清晰的 TigerKeDaPrinter，並加上程式檔案註解，提升可讀性與維護性。
