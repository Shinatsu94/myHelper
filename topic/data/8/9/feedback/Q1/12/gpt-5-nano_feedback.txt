/**
 * 檔名: Main.java
 * 功能: 印出「虎科大」及學生中文姓名的內碼，先印虎科大再印學生姓名
 * 作者: 教師
 * 日期: 2025-11-12
 */
public class Main {
    public static void main(String[] args) {
        // 使用字元陣列提高可維護性，並符合題目需求的順序
        char[] school = {'虎', '科', '大'};
        char[] studentName = {'林', '武', '杰'};

        // 依序印出字元與對應的 Unicode 內碼
        for (char c : school) {
            System.out.println(c + " " + (int) c);
        }
        for (char c : studentName) {
            System.out.println(c + " " + (int) c);
        }
    }
}

修改說明與原因：
- 將六個單字元改為兩個字元陣列，並以迴圈輸出，降低重複程式碼、提升可維護性與可擴充性。
- 將變數命名改為符合 Java 規範的 lowerCamelCase，增進可讀性與一致性。
- 新增檔案開頭註解，說明檔案名稱、功能、作者與日期，符合備註要求。
- 將學生姓名改為林、武、杰，以符合題目指定的組成與輸出順序，避免混淆。
- 保留原本輸出格式（字元 + 空格 + 內碼），確保輸出格式與題意一致。

改進建議（一句話）：
透過陣列與迴圈統一處理輸出，同時確保姓名組成與順序正確，並補上完整的檔案註解以提升可維護性。
