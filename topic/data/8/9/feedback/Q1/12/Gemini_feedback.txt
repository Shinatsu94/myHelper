**3. 改進的程式碼：**

```java
public class Main {
    public static void main(String[] args) {
        // 學校名稱和學生姓名
        char[] schoolName = {'虎', '科', '大'};
        char[] studentName = {'林', '震', '宇'};

        // 印出學校名稱和對應的內碼
        System.out.println("【OUTPUT】");
        printCharacterAndCode(schoolName);

        // 印出學生姓名和對應的內碼
        printCharacterAndCode(studentName);
    }

    // 輔助函數：印出字元陣列中的每個字元及其內碼
    public static void printCharacterAndCode(char[] charArray) {
        for (char c : charArray) {
            System.out.println(c + " " + (int) c);
        }
    }
}
```

**4. 修改說明：**

*   **引入陣列：** 使用 `char[]` 陣列來儲存學校名稱和學生姓名，提高了程式碼的彈性和可維護性。如果需要更改輸出的文字，只需要修改陣列中的元素即可。
*   **引入輔助函數：** 使用 `printCharacterAndCode()` 函數來印出字元陣列中的每個字元及其內碼，避免了程式碼的重複。
*   **使用迴圈：** 使用 `for-each` 迴圈來遍歷字元陣列，簡化了程式碼。
*   **增加註解：** 增加了程式碼註解，提高了程式碼的可讀性。
