/**
 * 程式名稱 : 印出字元和字元內碼（改良版）
 * 功能說明：
 *   印出「虎科大」三個字以及使用者中文姓名三個字的各自字符與對應的Unicode內碼。
 *   先印虎科大，再印學生中文姓名，每一行輸出「字元 空格 內碼」。
 *
 * 日期: 2025/11/12
 * 作者: 教師
 *
 * 備註：原作業有命名與順序不符問題，本版本以陣列方式管理字元，確保輸出順序符合題目要求，並使用有意義的變數命名與清楚的註解。
 */
public class PrintCharCodesImproved {

    public static void main(String[] args) {
        // 定義要輸出的「虎科大」三字
        char[] campusChars = { '虎', '科', '大' };
        // 定義學生的中文姓名三字（符合題意「你的中文姓名」；此例以林武杰為示範）
        char[] studentNameChars = { '林', '武', '杰' };

        // 先印出「虎科大」及其對應的內碼
        for (char c : campusChars) {
            System.out.println(c + " " + (int) c);
        }

        // 再印出學生中文姓名及其對應的內碼
        for (char c : studentNameChars) {
            System.out.println(c + " " + (int) c);
        }
    }
}

修改說明
- 將三個虎科大字與三個學生姓名字元分別放到陣列中，避免逐字手寫重複列印，增加可維護性與正確性。
- 透過迴圈逐字輸出，確保輸出順序必然為「虎、科、大」再「林、武、杰」，並以 (int) 取得每個字元的Unicode內碼，符合題意。
- 使用有意義的變數名稱（campusChars、studentNameChars）與清楚的程式註解，提高可閱讀性與可維護性；修正了原程式中名字拼寫與輸出順序不符合題意的問題。
- 程式標頭加入說明，包含檔案目的、輸出內容、日期與作者，符合備註要求。

一句話改進建議
- 建議未來改用外部資料或常數字元陣列，讓同一份程式可套用於不同姓名與字串組合，提升可重用性與彈性。
