public class CSIE_41343124_0926_1 {
    /*
     檔案名稱: CSIE_41343124_0926_1.java
     功能: 印出「虎科大」及學生姓名每個字的內碼，格式為 "<字> <內碼>"
     作者:  [填寫作者姓名]
     日期: 2025-11-12
     說明: 使用字元陣列與迴圈輸出，確保順序與格式符合題意。
     */
    public static void main(String[] args) {
        // 將需要輸出的字元放入同一陣列，順序分別為「虎 科 大 林 武 杰」
        char[] chars = {'虎','科','大','林','武','杰'};

        // 逐一輸出字元與其對應的 Unicode 內碼（十進位）
        for (char ch : chars) {
            System.out.println(ch + " " + (int) ch);
        }
    }
}

- 使用陣列與迴圈可避免重複寫同樣的輸出語句，降低未來維護成本；並可確保輸出的順序嚴格符合題目要求（虎科大在前，林武杰為學生姓名）。
- 改善變數命名與可讀性，便於他人理解程式的目的與流程。
- 加入標頭註解，符合作業要求的備註部分，提升文件化程度。
