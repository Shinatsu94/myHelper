功能: 印出「虎科大」三個字與學生中文姓名三個字的內碼 (Unicode 編碼)，每行格式為「字元 空格 內碼」。

說明: 使用陣列與迴圈輸出，提升可維護性與可讀性。

public class CSIE_41343227_0926 {
    public static void main(String[] args) {
        // 以陣列儲存要輸出的字元，方便維護與擴充

        char[] nameChars = {'陳', '天', '霖'};

        // 逐字輸出字元及其對應的 Unicode 內碼

        }
    }
}

  - 原因：符合 Java 常見命名慣例，提升可讀性。
