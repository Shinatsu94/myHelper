優化後完整程式碼（可直接執行）

/*
 * File: ChineseCodePointPrinter.java
 * Description: 印出虎科大三字及林武杰三字，並附上對應的 Unicode 內碼（Code Point）。
 * Author: 你的姓名
 * Date: 2025-11-12
 */

public class ChineseCodePointPrinter {
    public static void main(String[] args) {
        // 要輸出的字元：先印虎科大，接著印學生姓名林武杰
        char[] chars = {'虎', '科', '大', '林', '武', '杰'};

        // 逐一印出字元與對應的內碼
        for (char c : chars) {
            System.out.println(c + " " + (int) c);
        }
    }
}

修改說明與原因
- 將多個分散的姓名字元改成單一陣列（char[]），統一管理，便於日後維護與擴充（如變更字元集合或順序）。
- 以 for-each 迴圈取代單獨的 print 語句，讓邏輯更簡單、可讀性更高，且降低人為錯誤機率。
- 更改類別名稱為通用且具描述性的名稱 ChineseCodePointPrinter，並加入標準檔案註解，提升專案可讀性與維護性。
- 輸出格式改為「字元 空格 內碼」，符合題意的輸出格式要求，避免使用不符合格式的分隔符（如髮出現的「-」）。

一句話改進建議
- 以陣列與迴圈取代逐字宣告與多條列印，讓程式更具擴充性與穩定性。
