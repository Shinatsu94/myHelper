**3. 改進的程式碼**

```java
public class Main {
    public static void main(String[] args) {
        String school = "虎科大";
        String chineseName = "丁祥誌";

        // 先印虎科大
        printCharacterAndCode(school);

        // 再印學生中文姓名
        printCharacterAndCode(chineseName);
    }

    // 輔助函式：印出字串中每個字的內碼
    public static void printCharacterAndCode(String str) {
        for (int i = 0; i < str.length(); i++) {
            char c = str.charAt(i);
            int unicode = (int) c; // 取得字元的 Unicode 內碼
            System.out.println(c + " " + unicode);
        }
    }
}
```
