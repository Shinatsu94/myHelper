## 2. 優缺點分析

### 優點
1. **變數使用**：已將學校名稱與中文姓名分別存於 `school`、`chineseName` 兩個變數，符合題目「使用變數」的基本要求。  
2. **基本結構**：程式具備正確的 `class`、`main` 方法結構，能成功編譯與執行。

### 缺點
| 項目 | 說明 |
|------|------|
| **缺少說明** | 完全沒有註解或說明，閱讀者無法快速了解程式意圖。 |
| **輸出格式錯誤** | 只一次性印出兩個字串，未依題目「一行印一個中文字與其對應的內碼」的格式。 |
| **未取得 Unicode 內碼** | 沒有將每個中文字符轉成 `int` 取得其 Unicode 編碼。 |
| **缺少迴圈** | 題目需要先印出「虎科大」的三個字，接著印出姓名的每個字，必須使用迴圈逐一處理。 |
| **排版不佳** | 縮排不一致，程式碼可讀性低。 |
| **缺少換行** | `printf` 只使用了空格分隔，未在每個字元後換行。 |
