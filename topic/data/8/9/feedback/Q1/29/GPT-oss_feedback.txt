## 3. 改進後的程式碼

```java
/**
 * CSIE_41343150_0919_1
 * 目的：印出「虎科大」與使用者中文姓名的每個字元及其 Unicode 內碼。
 * 輸出格式：每行「字元 空格 內碼」 ，先印「虎科大」再印姓名。
 */
public class CSIE_41343150_0919_1 {

    // 中文姓名可自行修改
    private static final String STUDENT_NAME = "林武杰";

    // 要印出的學校名稱
    private static final String SCHOOL = "虎科大";

    public static void main(String[] args) {
        // 先印學校名稱
        printCharsWithCode(SCHOOL);
        // 再印學生姓名
        printCharsWithCode(STUDENT_NAME);
    }

    /**
     * 逐一印出字串中每個字元以及其 Unicode 內碼。
     *
     * @param text 要處理的字串
     */
    private static void printCharsWithCode(String text) {
        // 使用 for‑each 迴圈遍歷字元
        for (char ch : text.toCharArray()) {
            // (int) ch 會得到該字元的 Unicode 編碼（十進位）
            System.out.println(ch + " " + (int) ch);
        }
    }
}
```
