// 檔案名稱: NameCodePrinter.java
// 功能: 印出「虎科大」與「林武杰」三個字的中文字符及其對應的內碼（十進位）
 // 作者: 教授
 // 日期: 2025-11-12
 // 編碼說明: 使用 UTF-8 編碼，確保中文字元正確顯示

public class NameCodePrinter {
    public static void main(String[] args) {
        // 先印「虎科大」
        char[] header = {'虎', '科', '大'};
        // 再印學生中文姓名（題目示例順序：林、武、杰）
        char[] name = {'林', '武', '杰'};

        // 輸出每個字及其內碼（Unicode 代碼點，十進位表示）
        for (char c : header) {
            System.out.println(c + " " + (int) c);
        }
        for (char c : name) {
            System.out.println(c + " " + (int) c);
        }
    }
}

修改說明與原因
- 移除與題目無關的輸入/輸出流程：原程式讀取分數並輸出成績，與題目要求「印出虎科大及姓名的內碼」毫無關聯。改為直接輸出指定字元及其內碼，符合作業要求。
- 使用固定字元陣列：透過 header 與 name 兩個 char 陣列，清楚呈現需要輸出的字與順序，方便日後如需改動姓名或順序時快速修改。
- 動態取得內碼：以 (int) c 取得字元的 Unicode 編碼點（十進位），避免手動寫死數值，減少錯誤且跨環境穩定。
- 程式註解與檔案說明：加入檔案名稱、功能、作者、日期等註解，符合備註部分的要求，方便未來維護與評分。
