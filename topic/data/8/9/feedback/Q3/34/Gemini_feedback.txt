```java
import java.util.Scanner;

public class CSIE_41343209_0926_3 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 讀取使用者輸入的三個分數，並根據分數判斷及格狀態
        for (int i = 0; i < 3; i++) {
            System.out.print("請輸入第 " + (i + 1) + " 個分數 (0-100): "); //提示使用者輸入
            if (sc.hasNextInt()) { // 檢查是否輸入的是整數
                int score = sc.nextInt();

                // 檢查輸入的分數是否在有效範圍內
                if (score >= 0 && score <= 100) {
                    if (score >= 60) {
                        System.out.println("及格");
                    } else if (score >= 50) { // 50 ~ 59
                        System.out.println("補考");
                    } else { // < 50
                        System.out.println("不及格");
                    }
                } else {
                    System.out.println("輸入的分數超出有效範圍 (0-100)，請重新輸入");
                    i--; // 讓迴圈重新執行一次，要求使用者重新輸入
                }
            } else {
                System.out.println("輸入的不是整數，請重新輸入");
                sc.next(); // 清空錯誤的輸入
                i--; // 讓迴圈重新執行一次，要求使用者重新輸入
            }
        }

        sc.close();
    }
}
```

**4. 修改說明**

*   **增加輸入提示:** 在 `sc.nextInt()` 前加上 `System.out.print` 提示使用者輸入的是第幾個分數，增加使用者體驗。
*   **增加輸入驗證:**
    *   使用 `sc.hasNextInt()` 檢查輸入是否為整數。如果不是整數，輸出錯誤訊息，並且使用 `sc.next()` 清空錯誤的輸入，`i--` 讓迴圈重新執行一次，要求使用者重新輸入。
    *   增加分數範圍驗證，檢查分數是否在 0-100 之間。如果超出範圍，輸出錯誤訊息，並且 `i--` 讓迴圈重新執行一次，要求使用者重新輸入。
*   **增加程式的說明:** 在for迴圈前增加程式的說明，說明程式的目的。
