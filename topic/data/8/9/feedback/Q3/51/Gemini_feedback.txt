```java
import java.util.Scanner;

public class CSIE_41343232_3_0926 {
    public static void main(String[] args) {
        Scanner scn = new Scanner(System.in);
        // 使用迴圈讀取多個成績，直到沒有輸入為止
        while (scn.hasNextInt()) {
            //System.out.print("輸入你的成績 "); // 移除此行，因為不需要每次都提示

            // 判斷成績等級並輸出
            if (grade >= 60) {
                System.out.println("及格"); // 使用println換行
            } else if (grade < 60 && grade >= 50) {
                System.out.println("補考"); // 使用println換行
            } else {
                System.out.println("不及格"); // 使用println換行
            }
        }
        scn.close();
    }
}
```

    *   原因：題目範例輸出中沒有提示訊息，而且多個提示訊息會影響程式的自動化測試。
*   **增加註解：**
