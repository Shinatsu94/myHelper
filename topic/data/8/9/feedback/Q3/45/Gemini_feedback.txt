**3. 改進的程式碼：**

```java
import java.util.Scanner;

public class CSIE_41343223_0926_3 {
    public static void main(String[] args) {
        // 建立 Scanner 物件，用於讀取使用者輸入
        Scanner scanner = new Scanner(System.in);

        System.out.println("請輸入成績 (輸入 -1 結束):");

        int score = scanner.nextInt(); // 輸入成績(整數)

        while(score != -1) { // 當輸入為 -1 時結束迴圈
            if (score >= 60) { // 判斷分數
                System.out.println("及格");
            } else if (score >= 50) {
                System.out.println("補考");
            } else {
                System.out.println("不及格");
            }

            score = scanner.nextInt(); // 再次讀取輸入
        }

        scanner.close(); // 結束 scanner
        System.out.println("程式結束");
    }
}
```

**4. 修改說明：**

*   **變數命名:** 將 `scn` 改名為 `scanner`，使其更具描述性。
*   **讀取多個分數:** 增加了 `while` 迴圈，迴圈終止條件為輸入 `-1`，程式可以持續讀取使用者的輸入並判斷分數，直到使用者輸入 `-1` 為止。
*   **程式結束提示:** 在程式結束前，印出 "程式結束"，讓使用者知道程式已完成。
