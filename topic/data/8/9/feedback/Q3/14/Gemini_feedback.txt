```java
import java.util.Scanner;

public class CSIE_41245139_0926_3 {
    public static void main(String[] args) {
        // 創建 Scanner 物件以讀取使用者輸入
        Scanner sca = new Scanner(System.in);

        // 循環讀取使用者輸入的分數，直到沒有更多輸入
        while (sca.hasNextInt()) {
            // 提示使用者輸入分數
            //System.out.print("請輸入分數:");  // 移除多餘的提示語句，因為題目只需要輸出結果

            // 讀取使用者輸入的分數
            int score = sca.nextInt();

            // 判斷分數等級
            if (score >= 60) {
                System.out.println("及格");
            } else if (score >= 50) {
                System.out.println("補考");
            } else {
                System.out.println("不及格");
            }
        }

        // 關閉 Scanner 物件
        sca.close();
    }
}
```

**4. 修改說明**

*   **加入 `while (sca.hasNextInt())` 迴圈：** 為了能夠處理多個輸入，使用 `while (sca.hasNextInt())` 迴圈來判斷是否還有下一個整數輸入。如果有的話，就繼續讀取並處理。
*   **移除多餘的提示語句:** 因為題目只需要輸出結果，並不需要提示使用者輸入，因此註解掉`System.out.print("請輸入分數:");`
