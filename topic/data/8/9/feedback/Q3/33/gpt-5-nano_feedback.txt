/*
    檔案名稱: NameCodePrinter.java
    功能說明: 印出「虎科大林武杰」六個字的字元與對應的內碼（Unicode 編碼點，十進位顯示）。
    作者: 教師姓名
    日期: 2025-11-12
    註解說明: 先印「虎科大」，再印「林武杰」；每行格式為 "<字> <內碼(十進位)>"
*/
public class NameCodePrinter {
    public static void main(String[] args) {
        // 依題目順序，先印虎、科、大，再印林、武、杰
        char[] chars = {'虎', '科', '大', '林', '武', '杰'};

        for (char c : chars) {
            // 將字元轉成整數，顯示其 Unicode 編碼點（十進位）
            System.out.println(c + " " + (int) c);
        }
    }
}

修改說明與原因：
- 正確性：用一個字元陣列依序輸出六個字，並以其 Unicode 編碼點（十進位）作為內碼，符合題目「一行印一個中文字與其對應的內碼」的需求，且先印虎科大再印林武杰。
- 邏輯：使用單一迴圈與字元陣列清楚描述輸出順序，邏輯簡單、穩定，易於未來擴充或改為其他字串。
- 格式：採用有意義的類別名稱 NameCodePrinter，並加入清楚的檔案註解與功能說明，方便他人理解與維護。
- 備註：加入程式頭註解，說明檔案名稱、功能、日期與簡要說明，提升可維護性。

建議在輸出前把需要輸出的字元與內碼關係做成常數或外部資料來源，方便未來維護與本地化成其他字集時的修改。
