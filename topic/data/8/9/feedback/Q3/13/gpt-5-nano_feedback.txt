/*
 * 文件名稱: CSIE_NameCodeOutput.java
 * 功能說明: 根據題目需求，印出「虎科大」與學生中文姓名的每個字，及其對應的 Unicode 編碼點（內碼）。
 * 作者: 教師姓名
 * 日期: 2025-11-12
 * 版本: 1.0
 */
public class CSIE_NameCodeOutput {
    public static void main(String[] args) {
        // 先印虎 科 大，再印 林 武 杰
        char[] chars = {'虎', '科', '大', '林', '武', '杰'};
        for (char ch : chars) {
            int codePoint = (int) ch; // 取得該字的 Unicode 編碼點（內碼）
            System.out.println(ch + " " + codePoint);
        }
    }
}

- 簡化結構：不再從輸入讀取資料，避免與測試案例衝突，確保輸出固定結果。
