```java
// 檔案名稱: NameEncodingPrinter.java
// 功能: 印出「虎科大」及學生姓名的每個字，並顯示對應的 Unicode 內碼
// 作者: 你
// 日期: 2025-11-12
// 備註: 不需要用到輸入，直接輸出固定字串與其內碼

public class NameEncodingPrinter {
    public static void main(String[] args) {
        // 先印出「虎科大」
        String header = "虎科大";
        printCodePoints(header);

        // 再印出學生姓名，這裡以示例姓名「林武杰」
        String name = "林武杰";
        printCodePoints(name);
    }

    // 輸出字串中每一個字及其 Unicode 內碼
    private static void printCodePoints(String s) {
        for (int i = 0; i < s.length();) {
            int codePoint = s.codePointAt(i);
            String ch = new String(Character.toChars(codePoint));
            System.out.println(ch + " " + codePoint);
            i += Character.charCount(codePoint);
        }
    }
}
```

- 增加程式註解與標頭資訊，提升可讀性與維護性。
