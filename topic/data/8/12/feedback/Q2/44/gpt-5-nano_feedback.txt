優化後的完成版程式碼與說明
以下提供一個可正確完成題目的版本，包含：逐項累加勒布尼茲級數以計算 pi，對 i = 10000, 20000, ..., 100000 輸出 pi，並找出使 pi 四捨五入到小數點後5位等於 3.14159 的最小 i；輸出格式符合題意，i 以六位寬度對齊，pi 以十位小數。

程式碼（PiLeibniz.java）：
/*
檔案名稱: PiLeibniz.java
功能說明: 使用勒布尼茲級數近似 pi，輸出 i=10000~100000（步長10000）對應的 pi 值，格式為
        i =  10000, pi = 3.1414926536
        i =  20000, pi = ...
        且在輸出結尾再列出最小使 pi 四捨五入至小數點後5位等於 3.14159 的 i 值。
作者: 教授
日期: 2025-11-12
*/
public class PiLeibniz {
    public static void main(String[] args) {
        // 先輸出 i=10000, 20000, ..., 100000 對應的 pi 值
        double sum = 0.0;
        int maxI = 100000;
        int step = 10000;

        for (int i = 1; i <= maxI; i++) {
            // 第 i 項： (-1)^(i-1) / (2*i - 1)
            sum += ((i % 2 == 1) ? 1.0 : -1.0) / (2.0 * i - 1.0);
            if (i % step == 0) {
                double pi = 4.0 * sum;
                // i 顯示到6位數，pi顯示到小數點後10位
                System.out.printf("i = %6d, pi = %.10f%n", i, pi);
            }
        }

        // 輸出空行，與下一行結果分開
        System.out.println();

        // 找出使 pi 四捨五入至小數點後5位等於 3.14159 的最小 i
        double sumForMinI = 0.0;
        int minI = -1;
        int limit = 2000000; // 足夠大以便找到第一個符合的 i
        for (int i = 1; i <= limit; i++) {
            sumForMinI += ((i % 2 == 1) ? 1.0 : -1.0) / (2.0 * i - 1.0);
            double pi = 4.0 * sumForMinI;
            long rounded5 = Math.round(pi * 100000.0); // 四捨五入到小數點後5位
            if (rounded5 == 314159) {
                minI = i;
                break;
            }
        }

        // 輸出最小 i（格式與前面一致，i 為六位寬度）
        System.out.println("i = " + String.format("%6d", minI));
    }
}
