import java.util.Scanner;

/**
 * 檔案名稱: WellOrderedNumber.java
 * 功能: 判斷輸入的十進位數字字串是否為嚴格遞增的數字（左到右的每位數字皆遞增）
 * 作者: YourName
 * 日期: 2025-11-15
 * 備註: 程式讀取一個字串，若其中每對相鄰字元都滿足前者 < 後者，輸出 "<input> is well-ordered number."
 *       否則輸出 "<input> is not well-ordered number."
 */
public class WellOrderedNumber {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 讀取單一輸入，程式執行一次即可結束
        String input = sc.next();

        boolean isWellOrdered = true;

        // 檢查每一對相鄰位元是否嚴格遞增
        for (int i = 0; i < input.length() - 1; i++) {
            char current = input.charAt(i);
            char next = input.charAt(i + 1);
            if (current >= next) {
                isWellOrdered = false;
                break; // 遇到第一個不符合就結束
            }
        }

        System.out.println(input + " " + (isWellOrdered
                ? "is well-ordered number."
                : "is not well-ordered number."));

        sc.close();
    }
}

說明：
- 正確性: 仍然以相鄰位元比較的方式判斷，對於單一位數或多位數皆正確，輸出格式與題目要求吻合。
- 邏輯: 直接且清晰的邏輯，遇到第一個不符合條件時立即結束迴圈，提升效率且易於閱讀。
- 格式: 程式碼有清楚的縮排與結構，使用較具描述性的變數名稱，且移除了不必要的輸出。
- 備註: 已加入標頭註解，描述檔案名稱、功能、作者與日期，以及輸出格式說明。
