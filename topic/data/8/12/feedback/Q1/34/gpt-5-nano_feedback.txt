```java
/*
File: LeibnizPi.java
功能: 計算萊布尼茲級數近似π
      (a) 輸出 i = 10000, 20000, ..., 100000 時的 pi 值（pi = 4 * sum_{k=0}^{i-1} (-1)^k /(2k+1)）
      (b) 找出使 pi >= 3.14159 的最小 i，並輸出該 i
作者: 你的名字
日期: 2025-11-11
說明: 程式使用單一遍歷計算級數，先輸出 (a) 的十個點，然後繼續計算直到找到最小的 i 使 pi >= 3.14159
*/
public class LeibnizPi {
    public static void main(String[] args) {
        // Part (a): 輸出 i = 10000, 20000, ..., 100000 的 pi 值
        double sum = 0.0;
        int maxIForPartA = 100000;
        int step = 10000;

        // 身為單次遍歷，逐步累加每一項
        for (int t = 0; t < maxIForPartA; t++) {
            // 第 t+1 項的分母為 2t+1，符號依序為 +, -, +, ...
            double term = (t % 2 == 0) ? 1.0 / (2.0 * t + 1.0) : -1.0 / (2.0 * t + 1.0);
            sum += term;

            // 當累加到第 i 個項時，輸出 pi
            if ((t + 1) % step == 0) {
                int i = t + 1;
                double pi = 4.0 * sum;
                System.out.printf("i =%7d, pi = %.10f%n", i, pi);
            }
        }

        // Part (b): 找出最小的 i 使 pi >= 3.14159
        int minimalI = -1;
        // sum 已累加到 i = maxIForPartA，繼續往後累加直到符合條件
        int t = maxIForPartA;
        while (t < 2000000) { // 安全上限，實務上會在較早就找到
            double term = (t % 2 == 0) ? 1.0 / (2.0 * t + 1.0) : -1.0 / (2.0 * t + 1.0);
            sum += term;
            double pi = 4.0 * sum;
            if (pi >= 3.14159) {
                minimalI = t + 1;
                break;
            }
            t++;
        }

        if (minimalI != -1) {
            System.out.printf("i =%7d%n", minimalI);
        } else {
            System.out.println("i =       -1"); // 理論上不會發生，僅作防呆
        }
    }
}
```

  - 對於 (b) 的最小 i，透過在每次加完一項後即計算 pi，當 pi 達到或超過 3.14159 時即記錄該 i 並停止，確保找到真正的最小 i。
