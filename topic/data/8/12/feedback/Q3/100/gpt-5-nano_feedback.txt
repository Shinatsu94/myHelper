  檔案名稱建議: LeibnizPi.java
  功能簡述: 使用勒貝格（Leibniz）級數逐步累計近似圓周率，輸出指定的 i 值與對應的 pi 值，格式符合題目要求。

檔案內容:
/*
檔名: LeibnizPi.java
功能: 使用 Leh-巴斯級數近似 pi，輸出 i=10000,20000,...,100000 的 pi 值，pi 保留十位小數；i 以寬度6顯示，便於對齊。
作者: 教授
日期: 2025-11-12
註解: 逐步累加避免重複計算，並以 printf 控制輸出格式。
*/
public class LeibnizPi {
    public static void main(String[] args) {
        // 逐步累加Leibniz級數：pi = 4 * sum_{k=0}^{i-1} (-1)^k / (2k+1)
        double sum = 0.0;
        // 從 k = 0 到 k = 99999，共計 100000 項
        for (int k = 0; k < 100000; k++) {
            double term = ((k & 1) == 0) ? 1.0 / (2.0 * k + 1.0) : -1.0 / (2.0 * k + 1.0);
            sum += term;

            // 當前項數 i 為 k+1，若是 10000 的整數倍，輸出一次結果
            int i = k + 1;
            if (i % 10000 == 0) {
                double pi = 4.0 * sum;
                // i: 寬度為6，pi: 小數點後10位
                System.out.printf("i = %6d, pi = %.10f%n", i, pi);
            }
        }
    }
}
