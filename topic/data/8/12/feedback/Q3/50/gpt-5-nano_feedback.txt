修改後完整程式碼（符合題目要求的輸出格式，使用 Leibniz 公式計算 pi，輸出 i 為 6 位寬、pi 為小數點後 10 位）：
/*
文件名稱: PiLeibniz.java
功能: 計算 pi 的近似值，對應 i = 10000, 20000, ..., 100000；輸出格式符合題目需求
作者: 以實作範例改寫
日期: 2025-11-11
說明: 使用 Leibniz 公式 pi = 4 * sum_{k=0}^{i-1} (-1)^k / (2k+1)，逐步輸出不同 i 的 pi 值
*/
public class PiLeibniz {
    public static void main(String[] args) {
        // 對每一個 i 從 10000 到 100000，每次增量 10000
        for (int i = 10000; i <= 100000; i += 10000) {
            double piEstimate = 0.0;
            // 以內層迴圈累加前 i 項
            for (int k = 0; k < i; k++) {
                double term = 1.0 / (2.0 * k + 1.0);
                if ((k & 1) == 0) { // k 偶數時為 +，奇數時為 -
                    piEstimate += term;
                } else {
                    piEstimate -= term;
                }
            }
            piEstimate *= 4.0; // Leibniz 對 pi 的近似
            // 輸出格式：i 為六位寬，pi 為小數點後 10 位
            System.out.printf("i = %6d, pi = %.10f%n", i, piEstimate);
        }
    }
}
