// 檔案名稱: Main.java
// 功能: 讀入一維整數陣列，將其升序排序，並以空格分隔輸出
// 作者: 你的名字
// 日期: 2025-11-12
import java.util.Arrays;
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int n;
        if (!sc.hasNextInt()) {
            sc.close();
            return;
        }
        n = sc.nextInt();
        int[] a = new int[n];
        for (int i = 0; i < n; i++) {
            if (sc.hasNextInt()) {
                a[i] = sc.nextInt();
            } else {
                a[i] = 0; // 預設，若輸入不足則用 0
            }
        }

        Arrays.sort(a); // 正確且簡潔地排序

        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(a[i]);
        }

        System.out.print(sb.toString());
        sc.close();
    }
}

修改部份說明
- 改為採用 Arrays.sort 以確保正確的升序排序，避免原程式邏輯錯誤與不必要的複雜性。
- 加入標頭註解與清晰的變數命名，提升可讀性與可維護性。
- 使用 StringBuilder 合併輸出，避免每次輸出都產生字串拼接的效能影響。

一句話改進建議
使用內建排序替代自訂排序邏輯，並補齊程式註解與輸出格式，能提升正確性與可維護性。
