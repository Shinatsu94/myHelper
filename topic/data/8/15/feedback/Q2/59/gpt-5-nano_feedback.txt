改進建議與評語（一句話）
程式雖能產生題目所需的替換結果，但會多印原始陣列、對負數情況的最大值計算不健全，且缺少標頭註解與清晰的變數命名，需修正以符合題意與可讀性。

改動與原因說明
- 移除多餘輸出：題目只需輸出轉換後的陣列，移除輸出原始陣列的部分，避免影響測資比對。
- 正確處理負數情況：將最大值的初始值由 0 改為 arr[i+1]，避免在後續元素全為負數時誤把最大值取成 0。
- 改善命名與可讀性：使用更具描述性的變數名稱，並在關鍵區塊加上註解。
- 增加標頭註解：加入檔案名稱、功能、作者、日期等，提升可維護性。
- 保持原始時間複雜度：仍採用 O(n^2) 的直覺做法，符合題意的直接實作。
- 格式與排版：統一縮排與輸出格式，避免多餘空格，輸出用空格分隔，結尾換行。

修改後完整程式碼
/*
檔案: CSIE_41343241_1031_2.java
功能: 依題意將陣列中每個元素替換為其之後出現的最大值，最後一個元素為 -1。
輸入: 先輸入長度 n，接著 n 個整數。
輸出: 轉換後的陣列內容（以空白分隔）。
作者: 教授
日期: 2025-11-14
*/
import java.util.Scanner;

public class CSIE_41343241_1031_2 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int[] arr = new int[n];

        // 讀取陣列
        for (int i = 0; i < n; i++) {
            arr[i] = sc.nextInt();
        }

        // 依題意計算每個位置之後的最大值，最後一個設定為 -1
        for (int i = 0; i < n - 1; i++) {
            int max = arr[i + 1];
            for (int j = i + 1; j < n; j++) {
                if (arr[j] > max) {
                    max = arr[j];
                }
            }
            arr[i] = max;
        }
        // 最後一個元素設為 -1
        if (n > 0) {
            arr[n - 1] = -1;
        }

        // 輸出結果（僅輸出轉換後的陣列）
        for (int i = 0; i < n; i++) {
            if (i > 0) System.out.print(" ");
            System.out.print(arr[i]);
        }
        System.out.println();
        sc.close();
    }
}
