/*
檔案名稱: CSIE_41343241_1031_2.java
功能: 讀取整數個數 n 及 n 個整數，輸出升序排序結果（以空格分隔，單行輸出）
作者: 教師
日期: 2025-11-12
說明: 使用 Arrays.sort 處理，支援負數，且僅輸出排序後的結果，符合題目要求。
*/
import java.util.Arrays;
import java.util.Scanner;

public class CSIE_41343241_1031_2 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 讀取長度
        int n = 0;
        if (sc.hasNextInt()) {
            n = sc.nextInt();
        } else {
            sc.close();
            return;
        }

        // 讀取 n 個整數
        int[] arr = new int[n];
        for (int i = 0; i < n; i++) {
            arr[i] = sc.nextInt();
        }

        // 進行排序（由小到大）
        Arrays.sort(arr);

        // 輸出排序結果，避免尾端多餘空格
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(arr[i]);
        }
        System.out.println(sb.toString());

        sc.close();
    }
}

修改部分說明
- 移除不必要的前置輸出，僅輸出排序後結果，符合題意與範例輸出。
- 用 Arrays.sort 統一且正確地排序整數陣列，對負數有正確支援，且時間複雜度為 O(n log n)。
- 使用 StringBuilder 連接輸出字串，避免每次輸出都帶有尾端空格，輸出格式更穩定。
- 增加檔案標頭註解，說明檔案名稱、功能、作者、日期，提升維護性。
