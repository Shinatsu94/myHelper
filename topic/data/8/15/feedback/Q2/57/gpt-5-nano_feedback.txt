修改後完整程式碼與說明
修改後的完整程式碼如下（檔名可為 SortArrayAscending.java，類別名稱對應檔案名稱）：

/**
 * 檔案名稱: SortArrayAscending.java
 * 功能: 讀取陣列長度與該長度的整數，輸出經升冪排序後的內容
 * 作者: (你可以填作者姓名)
 * 日期: (日期)
 * 備註: 不使用 EOF，程式執行一次即結束
 */
import java.util.Scanner;
import java.util.Arrays;

public class SortArrayAscending {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int length = sc.nextInt();
        int[] a = new int[length];

        for (int i = 0; i < length; i++) {
            a[i] = sc.nextInt();
        }

        // 使用內建排序確保正確且簡潔
        Arrays.sort(a);

        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < length; i++) {
            if (i > 0) sb.append(' ');
            sb.append(a[i]);
        }

        System.out.print(sb.toString());
        sc.close();
    }
}

修改說明：
- 引入 Arrays.sort 取代自訂排序邏輯，直接將整數陣列升冪排序，邏輯更加簡潔且正確。
- 移除最後一個元素被強制設為 -1 的做法，以及不必要的中間變數，減少出錯機會。
- 增加檔案頭註解，提升可讀性與專案管理品質。
- 輸出部分使用 StringBuilder 組字串，避免多次 System.out.print 呼叫，提升效能。
