程式修改說明與優化要點
- 正確性與邏輯
  - 原程式初始化最大值為 -1，若後綴皆小於 -1，會導致結果錯誤（特別是包含負數時）。改為以 a[i+1] 作為後綴最大值的初始值，並再比對後續元素，以正確取得最大值。
  - 最後一個元素必須直接設定為 -1，與題目一致。
- 輸出格式
  - 修改為單一空格分隔，整個陣列輸出完成後換行，避免前後多餘的空格。
- 可讀性與可維護性
  - 加上適當的程式標頭註解，變數命名清楚，整體排版整潔。
  - 以單一用途的迴圈清楚地完成任務，避免不必要的條件判斷。

修改後完整程式碼
import java.util.Scanner;

/*
檔案名稱: CSIE_41343109_1031_02.java
功能: 讀入任意長度的一維整數陣列，先輸入長度再輸入各元素，將陣列
      的每一個元素置換為其之後的元素中的最大值，最後一個元素設為 -1。
作者: 吳育丞
日期: 1031
*/

public class CSIE_41343109_1031_02 {
    public static void main(String[] args) {
        Scanner scn = new Scanner(System.in);

        int n = 0;
        if (scn.hasNextInt()) {
            n = scn.nextInt();
        }

        int[] a = new int[n];
        for (int i = 0; i < n; i++) {
            a[i] = scn.nextInt();
        }

        // 將每個元素替換為其之後的最大值；最後一個元素設為 -1
        for (int i = 0; i < n - 1; i++) {
            int maxVal = a[i + 1];
            for (int j = i + 2; j < n; j++) {
                if (a[j] > maxVal) {
                    maxVal = a[j];
                }
            }
            a[i] = maxVal;
        }
        if (n > 0) {
            a[n - 1] = -1;
        }

        // 輸出：單一空格分隔，末尾換行
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(a[i]);
        }
        System.out.println(sb.toString());

        scn.close();
    }
}

說明:
- 此版本在處理負數輸入時也能正確找出後綴最大值，不再受限於初始化為 -1 的問題。
- 輸出格式統一，符合題目樣例與一般自動測資的比對要求。
