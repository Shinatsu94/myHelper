修改後完整程式碼（以符合題意的正確做法）：
import java.util.Scanner;
import java.util.Arrays;

public class CSIE_41343109_1031_02 {
	public static void main(String[] args) {
		Scanner scn = new Scanner(System.in);
		// 讀取陣列長度
		if (!scn.hasNextInt()) {
			scn.close();
			return;
		}
		int n = scn.nextInt();
		int[] a = new int[n];
		// 讀入陣列元素
		for (int i = 0; i < n; i++) {
			a[i] = scn.nextInt();
		}
		// 依大小排序
		Arrays.sort(a);
		// 輸出排序結果，使用單一空白分隔，末尾無空白
		StringBuilder sb = new StringBuilder();
		for (int i = 0; i < n; i++) {
			if (i > 0) sb.append(' ');
			sb.append(a[i]);
		}
		System.out.print(sb.toString());
		scn.close();
	}
}

修改原因說明
- 將排序邏輯改為使用 Arrays.sort(a) 以確保正確性，且能同時正確處理負數。
- 使用 StringBuilder 組裝輸出字串，避免輸出尾端多餘的空白，且確保數字之間只有單一空白分隔。
- 增加必要的程式標頭與註解，並修正命名與縮排風格，提升可讀性與維護性。
- 關閉 Scanner，避免資源洩漏，整體程式結構更清晰。

單句改進建議或反饋
- 建議將排序與輸出封裝成獨立方法，未來若需要擴充或重複使用就更容易維護。
