/*
檔案名稱: CSIE_51015120_1031_5.java
功能說明: 讀入一個整數 n，接著讀入 n 個整數，將它們升序排序後輸出。
作者: 你可以填入姓名
日期: 2025-11-12
註解: 程式使用 1 維陣列，配合 Arrays.sort 進行排序；輸出依序以空格分隔，末尾換行。
*/
import java.util.Scanner;
import java.util.Arrays;

public class CSIE_51015120_1031_5 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 讀取長度 n
        if (!sc.hasNextInt()) {
            sc.close();
            return;
        }
        int n = sc.nextInt();

        // 讀取 n 個整數到一維陣列
        int[] arr = new int[n];
        int count = 0;
        while (count < n && sc.hasNextInt()) {
            arr[count++] = sc.nextInt();
        }
        sc.close();

        // 若實際讀入的數量小於 n，僅排序實際讀入的部分
        if (count > 0) {
            Arrays.sort(arr, 0, count);
        }

        // 輸出排序結果，元素之間以單一空格分隔
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < count; i++) {
            if (i > 0) sb.append(' ');
            sb.append(arr[i]);
        }
        System.out.println(sb.toString());
    }
}

- 資料結構改變：改用一維整數陣列 int[]，符合「任意長度的一維陣列」的要求，並簡化資料型別。
- 輸入處理與穩健性：先讀取長度 n，接著讀入最多 n 個整數，若輸入不足 n 個仍能正確輸出已讀入的部分並排序，提升健壯性；若需求一定有 n 個數，亦能正常工作。
- 排序與輸出：使用 Arrays.sort 直接對整數陣列進行升序排序，輸出時以空格分隔，避免尾端多餘空白。
