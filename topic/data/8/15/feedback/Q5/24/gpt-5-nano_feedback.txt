/*
檔案名稱: SortIntArray.java
功能: 建立任意長度的一維整數陣列，將內容從小到大排序，並輸出排序結果。
作者: 教師
日期: 2025-11-12
說明: 使用動態長度的整數陣列、Arrays.sort 進行排序，適用於正確處理負數與任意長度輸入。
*/
import java.util.Arrays;
import java.util.Scanner;

public class SortIntArray {
    public static void main(String[] args) {
        Scanner scn = new Scanner(System.in);

        // 讀取陣列長度
        if (!scn.hasNextInt()) {
            return;
        }
        int n = scn.nextInt();
        int[] arr = new int[n];

        // 依序讀取 n 個整數
        for (int i = 0; i < n; i++) {
            if (scn.hasNextInt()) {
                arr[i] = scn.nextInt();
            } else {
                arr[i] = 0; // 若輸入不足，預設為 0
            }
        }

        // 升冪排序
        Arrays.sort(arr);

        // 輸出，元素以單一空格分隔，結尾不多餘空格
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(arr[i]);
        }
        System.out.println(sb.toString());
    }
}
