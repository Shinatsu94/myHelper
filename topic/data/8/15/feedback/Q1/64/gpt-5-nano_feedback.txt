import java.util.Arrays;
import java.util.Scanner;

/**
 * 檔案名稱: CSIE_51015105_1031_1.java
 * 功能: 讀取長度 n，接著讀取 n 個整數，並輸出從小到大排序的結果
 * 作者: 教師
 * 日期: 2025-11-12
 * 備註: 使用內建 Arrays.sort 提高可讀性與穩定性
 */
public class CSIE_51015105_1031_1 {
    public static void main(String[] args) {
        try (Scanner scanner = new Scanner(System.in)) {
            if (!scanner.hasNextInt()) {
                return;
            }
            int n = scanner.nextInt();
            int[] arr = new int[n];
            for (int i = 0; i < n; i++) {
                arr[i] = scanner.nextInt();
            }
            Arrays.sort(arr);
            StringBuilder sb = new StringBuilder();
            for (int i = 0; i < n; i++) {
                if (i > 0) sb.append(' ');
                sb.append(arr[i]);
            }
            System.out.println(sb.toString());
        }
    }
}

修改說明
- 使用內建排序（Arrays.sort）取代自訂的選擇排序：提高可讀性與穩定性，負數與一般整數都能正確排序，且寫法更簡潔。
- 加入程式標頭註解：包含檔案名稱、功能、作者與日期，符合備註要求。
- 使用 try-with-resources 管理 Scanner：確保資源在程式結束時正確關閉，避免資源洩漏。
- 使用 StringBuilder 組裝輸出字串：避免多次 System.out.print 導致的效能問題，同時確保輸出格式為數字中間以單一空格分隔，最後再換行。

改進建議
