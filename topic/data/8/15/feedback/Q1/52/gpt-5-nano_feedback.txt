/*
 * File: CSIE_41343233_1031_2.java
 * 功能: 讀取整數 n 與 n 個整數，將陣列內容由小到大排序後輸出
 * 作者: 學生
 * 日期: 2025-11-12
 * 備註: 不使用 EOF，執行一次程式後結束。支援負數。
 */
import java.util.Arrays;

public class CSIE_41343233_1031_2 {
    public static void main(String[] args) {
        java.util.Scanner sc = new java.util.Scanner(System.in);

        // 讀取長度 n
        if (!sc.hasNextInt()) {
            return;
        }
        int n = sc.nextInt();
        if (n <= 0) {
            // 無元素時不輸出內容
            return;
        }

        // 讀取 n 個整數
        int[] arr = new int[n];
        for (int i = 0; i < n; i++) {
            if (sc.hasNextInt()) {
                arr[i] = sc.nextInt();
            } else {
                arr[i] = 0; // 若輸入不足，預設為 0
            }
        }

        // 進行升序排序
        Arrays.sort(arr);

        // 輸出排序後的陣列，元素以單一空格分隔
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(arr[i]);
        }
        System.out.print(sb.toString());

        sc.close();
    }
}

修改部分說明與原因：
- 取代原本的“對後面元素取最大值”的邏輯，改為使用 Arrays.sort 進行整數陣列的升序排序，確保輸出正確且時空複雜度穩定。這樣能保證正確性且簡化程式。
- 加入程式頭註解，清楚描述檔案名稱、功能、作者與日期，符合作業要求的備註部分。
- 使用正式且清晰的變數命名與適當的縮排，並以 StringBuilder 組裝輸出字串，提高可讀性與效率。
- 考慮負數及輸入不足情況，提供穩健的輸入處理與預設值，避免例外情況造成程式中斷。
