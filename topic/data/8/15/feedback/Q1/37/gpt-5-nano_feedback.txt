import java.util.Scanner;

/**
 * File: csie_1031_41343214.java
 * 功能: 依使用者輸入的長度與數值，對整數陣列進行升序排序，並輸出排序結果（以空格分隔，無尾隨空格）。
 * 作者: 你的名字
 * 日期: 2025-11-12
 */
public class csie_1031_41343214 {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int n = sc.nextInt();
        int[] arr = new int[n];

        for (int i = 0; i < n; i++) {
            arr[i] = sc.nextInt();
        }

        // 使用簡單的選取排序（Selection Sort）排序 arr
        for (int i = 0; i < n - 1; i++) {
            int minIndex = i;
            for (int j = i + 1; j < n; j++) {
                if (arr[j] < arr[minIndex]) minIndex = j;
            }
            int temp = arr[i];
            arr[i] = arr[minIndex];
            arr[minIndex] = temp;
        }

        // 輸出時避免尾隨空格
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(arr[i]);
        }
        System.out.println(sb.toString());

        sc.close();
    }
}

- 輸出格式修正：原程式在最後輸出時會在每個數字後補一個空格，造成尾隨空格與題目示例不同。改用 StringBuilder 先組裝輸出字串，僅在數字之間插入單一空格，避免尾端有多餘空格，符合常見自動化測資的格式檢查。

- 未改動核心演算法：保留原本的選取排序實作，保持與原作業意圖一致，確保正確性與可理解性。
