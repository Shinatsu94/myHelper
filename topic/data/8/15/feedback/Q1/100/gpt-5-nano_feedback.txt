```java
/*
 * 檔案名稱: Main.java
 * 功能: 讀取整數 n，輸入 n 個整數，排序後輸出升序結果
 * 作者: 教師
 * 日期: 2025-11-12
 * 版本: 1.0
 */

import java.util.Scanner;
import java.util.Arrays;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 讀取陣列長度
        int n = sc.nextInt();
        int[] nums = new int[n];

        // 讀取陣列元素
        for (int i = 0; i < n; i++) {
            nums[i] = sc.nextInt();
        }

        // 排序
        Arrays.sort(nums);

        // 輸出排序後的陣列，使用字串拼接提升效率
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(nums[i]);
        }
        System.out.println(sb.toString());

        // 注意：在某些執行環境下可能不建議顯式關閉 System.in
        // sc.close();
    }
}
```

- 將類別名稱改為 Main，符合 Java 的入口類型命名與大多數評測環境的要求，並以檔案名稱對應到檔案級別命名。
- 新增程式標頭註解，包含檔案名稱、功能、日期等，以符合備註需求。
- 移除不必要的關閉 Scanner 的呼叫，避免某些執行環境下的 System.in 關閉問題；輸出改用 StringBuilder 組裝，提升輸出效率與可讀性。
- 保留負數支援，使用 Arrays.sort 處理整數排序。
