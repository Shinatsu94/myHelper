// File: SortArray.java
// 功能: 讀取整數長度 n，接著讀入 n 個整數，對其升冪排序，輸出排序後的內容（以空格分隔）。
// 作者: 你的姓名
// 日期: 2025-11-12

import java.util.Arrays;
import java.util.Scanner;

public class SortArray {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);

        // 讀取長度 n
        if (!input.hasNextInt()) {
            input.close();
            return;
        }
        int n = input.nextInt();
        if (n <= 0) {
            // 若 n 非正，直接結束
            System.out.println();
            input.close();
            return;
        }

        // 讀取 n 個整數
        int[] arr = new int[n];
        for (int i = 0; i < n; i++) {
            if (input.hasNextInt()) {
                arr[i] = input.nextInt();
            } else {
                // 如果輸入不足，預設為 0
                arr[i] = 0;
            }
        }

        // 升冪排序
        Arrays.sort(arr);

        // 輸出排序結果，以空格分隔
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(arr[i]);
        }
        System.out.println(sb.toString());

        input.close();
    }
}

修改部分原因與說明
- 直接對題目需求作答：將「任意長度的一維陣列」的元素進行升冪排序並輸出，符合題意的最小實作。
- 使用 Arrays.sort 提升效率與穩定性：Java 標準庫的 Arrays.sort 對整數陣列的排序具備良好效能與穩定性，簡化程式邏輯。
- 清晰的輸入與輸出流程：先讀長度再讀取對應數量的元素，最終以空格分隔輸出結果，符合常見的競賽/作業輸出格式。
- 提升可維護性與可讀性：加入標頭註解，使用清晰的檔名與功能描述，便於他人閱讀與維護；變數命名與程式結構較為清晰。
