/*
 * File: Main.java
 * 功能: 從輸入讀取一維整數陣列，對其升冪排序，輸出排序後的內容
 * 作者: 教授
 * 日期: 2025-11-12
 */

import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (!sc.hasNextInt()) {
            sc.close();
            return;
        }
        int n = sc.nextInt();
        int[] a = new int[n];
        for (int i = 0; i < n; i++) {
            a[i] = sc.nextInt();
        }
        Arrays.sort(a);
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(a[i]);
        }
        System.out.println(sb.toString());
        sc.close();
    }
}

改動說明與原因
- 直接將輸入的 n 個整數排序並輸出，符合題目要求的「一次性輸入、升冪排序後輸出」的功能；原程式與題意相去甚遠，採用了不相關的三角形資料結構和查找流程。
- 使用 Arrays.sort 排序，語法清晰易懂，且對負數有完善支援，不需額外處理。
- 使用簡潔的輸入處理（Scanner），程式結構更直覺，便於維護。
