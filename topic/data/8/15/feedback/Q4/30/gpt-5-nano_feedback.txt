```java
/**
 * File: LowerTriangularMatrix.java
 * 功能: 建立一個下三角形的二維陣列，並找出指定數字在陣列中的位置
 * 作者: 教授
 * 日期: 2025-11-14
 * 備註: 程式包含註解，並遵循下三角形的填充規則與尋找邏輯
 */

import java.util.Scanner;

public class LowerTriangularMatrix {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        // 讀取高度 n 與要尋找的數字 num
        if (!scanner.hasNextInt()) {
            scanner.close();
            return;
        }
        int n = scanner.nextInt();

        if (!scanner.hasNextInt()) {
            scanner.close();
            return;
        }
        int num = scanner.nextInt();

        // 計算三角總數與範圍檢查
        int total = n * (n + 1) / 2;
        if (num < 1 || num > total) {
            System.out.println("not find " + num + " in array.");
            scanner.close();
            return;
        }

        // 建立下三角形二維陣列並填入數字
        int[][] tri = new int[n][];
        for (int i = 0; i < n; i++) {
            tri[i] = new int[i + 1];
        }
        int val = 1;
        for (int i = 0; i < n; i++) {
            for (int j = 0; j <= i; j++) {
                tri[i][j] = val++;
            }
        }

        // 在陣列中搜尋 num 的位置
        int foundRow = -1;
        int foundCol = -1;
        outer:
        for (int i = 0; i < n; i++) {
            for (int j = 0; j <= i; j++) {
                if (tri[i][j] == num) {
                    foundRow = i + 1; // 1-based
                    foundCol = j + 1; // 1-based
                    break outer;
                }
            }
        }

        if (foundRow != -1) {
            System.out.println(num + "[" + foundRow + ", " + foundCol + "]");
        } else {
            System.out.println("not find " + num + " in array.");
        }

        scanner.close();
    }
}
```

- 使用單次輸入處理，避免多組測資混淆，並與一般題目測資格式一致。
- 透過填充與搜尋確保在任何合法的 num 廣度下都能正確找出位置，且保留 1-based 的列/欄輸出格式。
