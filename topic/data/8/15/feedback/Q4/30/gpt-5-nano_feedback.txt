```java
/*
檔案名稱: CSIE_41343202_1031_3B.java
功能: 讀取一維整數陣列長度 n 與 n 個整數，輸出其升序排序的結果
作者: 教師
日期: 2025-11-12
*/
import java.util.Arrays;
import java.util.Scanner;

public class CSIE_41343202_1031_3B {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        // 讀取陣列長度 n
        if (!scanner.hasNextInt()) {
            scanner.close();
            return;
        }
        int n = scanner.nextInt();

        // 讀取 n 個整數
        int[] arr = new int[n];
        for (int i = 0; i < n; i++) {
            if (scanner.hasNextInt()) {
                arr[i] = scanner.nextInt();
            } else {
                // 若輸入不足，預設為 0 以避免例外
                arr[i] = 0;
            }
        }

        // 進行升序排序
        Arrays.sort(arr);

        // 輸出排序後的陣列，以空格分隔
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(arr[i]);
        }
        System.out.println(sb.toString());

        scanner.close();
    }
}
```

- 使用 Arrays.sort(arr) 來實作排序，簡潔且效率良好，適用於任意長度的整數陣列，且支援負數。
