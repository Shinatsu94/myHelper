修改後程式碼（可直接編譯執行，符合題目需求）：
/*
檔案名稱: SortArrayAscending.java
功能: 讀取整數個數 n，接著讀取 n 個整數，將它們升冪排序並輸出，以空格分隔。
作者: 你的姓名
日期: 2025-11-12
*/
import java.util.Arrays;
import java.util.Scanner;

public class SortArrayAscending {
    public static void main(String[] args) {
        Scanner scn = new Scanner(System.in);
        if (!scn.hasNextInt()) {
            scn.close();
            return;
        }

        int n = scn.nextInt();
        int[] arr = new int[n];

        for (int i = 0; i < n; i++) {
            if (scn.hasNextInt()) {
                arr[i] = scn.nextInt();
            } else {
                arr[i] = 0; // 若輸入不足，預設為 0，避免拋出異常
            }
        }

        Arrays.sort(arr);

        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(arr[i]);
        }

        System.out.println(sb.toString());
        scn.close();
    }
}

- 排序方式：使用 Arrays.sort(arr) 進行升冪排序，簡單且穩定，且能正確處理負數與任意整數範圍。
