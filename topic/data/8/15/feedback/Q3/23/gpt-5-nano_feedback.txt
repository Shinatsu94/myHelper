/*
檔案名稱：SortArray.java
功能：讀取長度 n，接著讀入 n 個整數，對陣列升冪排序，輸出排序後的陣列內容
作者：XXX
日期：YYYY-MM-DD
*/
import java.util.Arrays;
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (!sc.hasNextInt()) {
            sc.close();
            return;
        }
        int n = sc.nextInt();
        int[] a = new int[n];
        for (int i = 0; i < n; i++) {
            if (sc.hasNextInt()) {
                a[i] = sc.nextInt();
            } else {
                a[i] = 0; // 若輸入不足，預設為 0
            }
        }
        sc.close();

        Arrays.sort(a);

        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(a[i]);
        }
        System.out.println(sb.toString());
    }
}

修改說明：
- 由原先的星星輸出改為實作題目需求：先讀取長度 n，接著讀入 n 個整數，最後對整數陣列排序再輸出。
- 使用 Arrays.sort(a) 進行排序，穩定且效率較高，能正確處理包含負數的情況。
- 輸出使用 StringBuilder，確保輸出不會出現尾端多餘空格，且效能較直接拼接好。
- 加入程式頭註解，說明檔案名稱、功能、作者與日期，提升可維護性與可讀性。
- 變數命名與程式結構調整為符合一般 Java實作慣例，使用 public class Main，與簡潔的流程控制。
