修改後完整程式碼
- 說明：以下程式已符合題意，支援負數、任意長度的整數輸入，並以空格分隔輸出排序結果。

import java.util.Arrays;
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        // 讀取陣列長度
        if (!scanner.hasNextInt()) {
            scanner.close();
            return;
        }
        int n = scanner.nextInt();
        if (n <= 0) {
            scanner.close();
            return;
        }

        // 讀取 n 個整數
        int[] arr = new int[n];
        for (int i = 0; i < n; i++) {
            if (scanner.hasNextInt()) {
                arr[i] = scanner.nextInt();
            } else {
                // 若輸入不足，預設為 0；也可改為直接結束或拋出錯誤
                arr[i] = 0;
            }
        }

        // 升序排序，包含負數
        Arrays.sort(arr);

        // 輸出排序後的陣列，數字以單一空格分隔
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(arr[i]);
        }
        System.out.println(sb.toString());

        scanner.close();
    }
}

修改原因摘要
- 以陣列讀入與排序的方式，正確實作題目需求；使用 Arrays.sort 提供穩定且高效的排序，並以 StringBuilder 組裝輸出以確保格式正確。
