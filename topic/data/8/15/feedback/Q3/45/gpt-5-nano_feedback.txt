修改後完整程式碼
```java
/*
File: SortArrayAsc.java
功能: 從使用者讀取整數長度 n 與 n 個整數，將陣列內容由小到大排序，並以空格分隔輸出
作者: 你的名字
日期: 2025-11-12
*/

import java.util.Scanner;
import java.util.Arrays;

public class SortArrayAsc {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 讀取 n
        if (!sc.hasNextInt()) {
            sc.close();
            return;
        }
        int n = sc.nextInt();
        if (n <= 0) {
            sc.close();
            return;
        }

        // 讀入 n 個整數
        int[] a = new int[n];
        for (int i = 0; i < n; i++) {
            if (sc.hasNextInt()) {
                a[i] = sc.nextInt();
            } else {
                a[i] = 0; // 若輸入不足，補 0
            }
        }

        // 由小到大排序
        Arrays.sort(a);

        // 輸出：以空格分隔，結尾換行
        for (int i = 0; i < n; i++) {
            if (i > 0) System.out.print(' ');
            System.out.print(a[i]);
        }
        System.out.println();

        sc.close();
    }
}
```

- 使用 Arrays.sort 提升效率與穩定性，負數也能正確排序，並讓程式碼更簡潔易讀。
