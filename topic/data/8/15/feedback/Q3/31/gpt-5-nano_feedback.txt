import java.util.*;

public class CSIE_41343203_1031_3a {
    /*
     * 檔案名稱: CSIE_41343203_1031_3a.java
     * 功能說明: 讀取整數 n，接著讀入 n 個整數，將其由小到大排序後輸出成一行，中間以單一空格分隔。
     * 作者: (請自行填寫)
     * 日期: 2025-11-12
     */
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 讀取長度 n
        if (!sc.hasNextInt()) {
            sc.close();
            return;
        }
        int n = sc.nextInt();
        if (n <= 0) {
            sc.close();
            return;
        }

        // 讀入 n 個整數
        int[] arr = new int[n];
        for (int i = 0; i < n; i++) {
            if (sc.hasNextInt()) {
                arr[i] = sc.nextInt();
            } else {
                // 若輸入不足，預設為 0
                arr[i] = 0;
            }
        }
        sc.close();

        // 升冪排序
        Arrays.sort(arr);

        // 輸出，中間以空格分隔，無 trailing 空格
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < n; i++) {
            if (i > 0) sb.append(' ');
            sb.append(arr[i]);
        }
        System.out.print(sb.toString());
    }
}

修改說明：
- 從原本輸出星號圖形的邏輯，改為正確的數值輸入與排序流程，避免與題目需求不符。
- 使用陣列 arr 儲存 n 個整數，並用 Arrays.sort(arr) 執行升冪排序，時間複雜度為 O(n log n)，符合一般排序需求。
- 輸出改為空格分隔，確保沒有多餘的尾端空格，與需求輸出格式一致。
- 增加程式標頭註解與程式內註解，提升可讀性與維護性。
- 移除硬性長度限制（原程式限制 n <= 20），改以實際讀入的 n 作為陣列長度，讓題目可以處理任意長度（受限於資源）。

一句話改進建議/反饋：
