- 下列為一份可直接用於題目的優化後版本，改以一維陣列讀取、排序並輸出結果，支援負數，且輸出格式符合題目範例。

修改後完整程式碼
import java.util.Arrays;
import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 1. 讀取陣列長度
        int n = 0;
        if (sc.hasNextInt()) {
            n = sc.nextInt();
        }

        // 2. 讀取 n 個整數
        int[] arr = new int[Math.max(0, n)];
        for (int i = 0; i < n; i++) {
            if (sc.hasNextInt()) {
                arr[i] = sc.nextInt();
            } else {
                arr[i] = 0; // 若輸入不足，補0以避免例外
            }
        }

        // 關閉 Scanner（結束輸入）
        sc.close();

        // 3. 升序排序
        Arrays.sort(arr);

        // 4. 輸出排序結果，元素以空格分隔，末尾不多餘空格
        StringBuilder sb = new StringBuilder();
        for (int i = 0; i < arr.length; i++) {
            if (i > 0) sb.append(' ');
            sb.append(arr[i]);
        }
        System.out.print(sb.toString());
    }
}

修改原因說明
- 以一維整數陣列取代原本的二維三角矩陣與不相關的星號輸出，直接符合「建立任意長度的一維陣列，對內容進行升序排序，輸入長度與內容由使用者提供，最後印出排序後的內容」的需求。
- 使用 Arrays.sort 進行排序，時間複雜度為 O(n log n)，適合一般規模輸入，且對負數有原生支援。
- 使用 StringBuilder 組裝輸出，避免逐個 print 的效能問題，輸出格式與題目要求一致。
