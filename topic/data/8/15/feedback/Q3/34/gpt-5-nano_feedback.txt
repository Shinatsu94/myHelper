/*
檔案名稱: ArraySort.java
功能: 讀取長度與元素，升冪排序後輸出
作者: 教師批改
日期: 2025-11-12

import java.util.Scanner;
import java.util.Arrays;

public class ArraySort {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();
        int[] a = new int[n];
        for (int i = 0; i < n; i++) {
            a[i] = sc.nextInt();
        }
        Arrays.sort(a);
        for (int i = 0; i < n; i++) {
            if (i > 0) System.out.print(' ');
            System.out.print(a[i]);
        }
        System.out.println();
        sc.close();
    }
}

修改原因與說明
- 使用一維陣列存放輸入的資料，符合題目「建立一個任意長度的一維陣列」的要求，且能後續進行排序與輸出。
- 使用 Arrays.sort(a) 直接實作升冪排序（由小到大），保證正確性與效能，並避免手寫排序的邏輯錯誤。
- 增加程式頭部註解（檔案名稱、功能、日期等），提升可讀性與維護性，同時改善格式規範。
- 輸出改為單行數字，使用空格分隔，符合題目輸出格式。

一句話改進建議
建議在大資料量情況下改用 BufferedReader 與 StringTokenizer 進行輸入，以提升效能，避免 Scanner 的 I/O 效能瓶頸。
